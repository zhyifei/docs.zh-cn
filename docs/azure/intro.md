## <a name="choosing-a-hosting-option"></a><span data-ttu-id="6dd46-119">选择承载选项</span><span class="sxs-lookup"><span data-stu-id="6dd46-119">Choosing a hosting option</span></span>

<span data-ttu-id="6dd46-120">Azure 中的承载可以分为三个类别。</span><span class="sxs-lookup"><span data-stu-id="6dd46-120">Hosting in Azure can be divided into three categories.</span></span>

* <span data-ttu-id="6dd46-121">**基础结构即服务 (IaaS)**：使用 IaaS 可以预配所需的虚拟机以及关联的网络和存储组件。</span><span class="sxs-lookup"><span data-stu-id="6dd46-121">**Infrastructure-as-a-Service (IaaS)**: With IaaS, you provision the virtual machines you need along with associated network and storage components.</span></span> <span data-ttu-id="6dd46-122">然后将需要的任何软件和应用程序部署到这些 VM 上。</span><span class="sxs-lookup"><span data-stu-id="6dd46-122">You then deploy whatever software and applications you want onto those VMs.</span></span> <span data-ttu-id="6dd46-123">除了由 Microsoft 管理基础结构，该模型最接近传统的本地环境。</span><span class="sxs-lookup"><span data-stu-id="6dd46-123">This model is the closest to a traditional on-premises environment except that Microsoft manages the infrastructure.</span></span> <span data-ttu-id="6dd46-124">仍由你管理单独的 VM，包括操作系统、自定义软件和安全更新。</span><span class="sxs-lookup"><span data-stu-id="6dd46-124">You still manage the individual VMs, including the operating system, custom software, and security updates.</span></span>

* <span data-ttu-id="6dd46-128">**功能即服务 (FaaS)**：通常称为无服务器计算。与 PaaS 相比，FaaS 更不需担心承载环境。</span><span class="sxs-lookup"><span data-stu-id="6dd46-128">**Functions-as-a-Service (FaaS)**: Commonly referred to as serverless computing, FaaS goes even further than PaaS in abstracting the concerns of the hosting environment.</span></span> <span data-ttu-id="6dd46-129">只需部署代码，服务便会自动运行它，无需创建计算实例并向其部署代码。</span><span class="sxs-lookup"><span data-stu-id="6dd46-129">Instead of creating compute instances and then deploying code to those instances, you deploy your code and the service automatically runs it.</span></span> <span data-ttu-id="6dd46-130">无需管理计算资源。</span><span class="sxs-lookup"><span data-stu-id="6dd46-130">You don't need to administer the compute resources.</span></span> <span data-ttu-id="6dd46-131">平台会根据处理流量的需要将代码无缝缩放到相应级别，你只需按代理运行情况付费。</span><span class="sxs-lookup"><span data-stu-id="6dd46-131">The platform seamlessly scales your code up or down to whatever level necessary to handle the traffic, and you pay only when your code is running.</span></span> <span data-ttu-id="6dd46-132">Azure Functions 是 FaaS 服务。</span><span class="sxs-lookup"><span data-stu-id="6dd46-132">Azure Functions is a FaaS service.</span></span>
