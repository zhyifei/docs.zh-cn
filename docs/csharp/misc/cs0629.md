# <a name="compiler-error-cs0629"></a><span data-ttu-id="aa156-102">编译器错误 CS0629</span><span class="sxs-lookup"><span data-stu-id="aa156-102">Compiler Error CS0629</span></span>
<span data-ttu-id="aa156-103">条件成员“member”不能在类型“Type Name”中实现接口成员“base class member”</span><span class="sxs-lookup"><span data-stu-id="aa156-103">Conditional member 'member' cannot implement interface member 'base class member' in type 'Type Name'</span></span>  
  
 <span data-ttu-id="aa156-104">[条件](http://msdn.microsoft.com/library/e1c4913b-74d0-421a-8a6d-c14b3f0e68fb)属性不能在接口的实现。</span><span class="sxs-lookup"><span data-stu-id="aa156-104">The [Conditional](http://msdn.microsoft.com/library/e1c4913b-74d0-421a-8a6d-c14b3f0e68fb) attribute cannot be used on the implementation of an interface.</span></span>  
  
 <span data-ttu-id="aa156-105">以下示例生成 CS0629：</span><span class="sxs-lookup"><span data-stu-id="aa156-105">The following sample generates CS0629:</span></span>  
  
```  
// CS0629.cs  
interface MyInterface  
{  
   void MyMethod();  
}  
  
public class MyClass : MyInterface  
{  
   [System.Diagnostics.Conditional("debug")]  
   public void MyMethod()    // CS0629, remove the Conditional attribute  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
