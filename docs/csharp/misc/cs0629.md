# <a name="compiler-error-cs0629"></a><span data-ttu-id="7bb5b-102">编译器错误 CS0629</span><span class="sxs-lookup"><span data-stu-id="7bb5b-102">Compiler Error CS0629</span></span>
<span data-ttu-id="7bb5b-103">条件成员“member”不能在类型“Type Name”中实现接口成员“base class member”</span><span class="sxs-lookup"><span data-stu-id="7bb5b-103">Conditional member 'member' cannot implement interface member 'base class member' in type 'Type Name'</span></span>  
  
 <span data-ttu-id="7bb5b-104">不能在接口的实现上使用 [条件](../../csharp/programming-guide/concepts/attributes/common-attributes.md#Conditional) 特性。</span><span class="sxs-lookup"><span data-stu-id="7bb5b-104">The [Conditional](../../csharp/programming-guide/concepts/attributes/common-attributes.md#Conditional) attribute cannot be used on the implementation of an interface.</span></span>  
  
 <span data-ttu-id="7bb5b-105">以下示例生成 CS0629：</span><span class="sxs-lookup"><span data-stu-id="7bb5b-105">The following sample generates CS0629:</span></span>  
  
```csharp  
// CS0629.cs  
interface MyInterface  
{  
   void MyMethod();  
}  
  
public class MyClass : MyInterface  
{  
   [System.Diagnostics.Conditional("debug")]  
   public void MyMethod()    // CS0629, remove the Conditional attribute  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
