# <a name="compiler-error-cs1537"></a>编译器错误 CS1537
using 别名“alias”以前在此命名空间中出现过  
  
 你曾两次将某个符号定义为了命名空间的别名。 一个符号只能定义一次。  
  
 下面的示例生成 CS1537：  
  
```csharp  
// CS1537.cs  
namespace x  
{  
   using System;  
   using Object = System.Object;  
   using Object = System.Object;   // CS1537, delete this line to resolve  
   using System = System;  
}  
```
