# <a name="compiler-error-cs0525"></a>编译器错误 CS0525
接口不能包含字段  
  
 [接口](../../csharp/language-reference/keywords/interface.md) 可以包含方法和属性但不能包含字段。  
  
 下面的示例生成 CS0525：  
  
```  
// CS0525.cs  
namespace x  
{  
   public interface clx  
   {  
      public int i;   // CS0525  
   }  
}  
```
