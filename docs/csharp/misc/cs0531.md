# <a name="compiler-error-cs0531"></a><span data-ttu-id="695aa-102">编译器错误 CS0531</span><span class="sxs-lookup"><span data-stu-id="695aa-102">Compiler Error CS0531</span></span>
<span data-ttu-id="695aa-103">“member”：接口成员不能有定义</span><span class="sxs-lookup"><span data-stu-id="695aa-103">'member' : interface members cannot have a definition</span></span>  
  
 <span data-ttu-id="695aa-104">在 [接口](../../csharp/language-reference/keywords/interface.md) 中声明的方法必须在从接口继承的类而不是接口本身中实现。</span><span class="sxs-lookup"><span data-stu-id="695aa-104">Methods that are declared in an [interface](../../csharp/language-reference/keywords/interface.md) must be implemented in a class that inherits from it and not in the interface itself.</span></span>  
  
 <span data-ttu-id="695aa-105">下面的示例生成 CS0531：</span><span class="sxs-lookup"><span data-stu-id="695aa-105">The following sample generates CS0531:</span></span>  
  
```csharp  
// CS0531.cs  
namespace x  
{  
   public interface clx  
   {  
      int xclx()   // CS0531, cannot define xclx  
      // Try the following declaration instead:  
      // int xclx();  
      {  
         return 0;  
      }  
   }  
  
   public class cly  
   {  
      public static void Main()  
      {  
      }  
   }  
}  
```
