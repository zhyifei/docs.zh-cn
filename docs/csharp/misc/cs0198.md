 <span data-ttu-id="3dcdd-104">[readonly](../../csharp/language-reference/keywords/readonly.md) 变量必须与构造函数具有相同的 [静态](../../csharp/language-reference/keywords/static.md) 用法，在此构造函数中你要对其进行初始化。</span><span class="sxs-lookup"><span data-stu-id="3dcdd-104">A [readonly](../../csharp/language-reference/keywords/readonly.md) variable must have the same [static](../../csharp/language-reference/keywords/static.md) usage as the constructor in which you want to initialize it.</span></span> <span data-ttu-id="3dcdd-105">有关详细信息，请参阅[静态构造函数](../../csharp/programming-guide/classes-and-structs/static-constructors.md)。</span><span class="sxs-lookup"><span data-stu-id="3dcdd-105">For more information, see [Static Constructors](../../csharp/programming-guide/classes-and-structs/static-constructors.md).</span></span>

```csharp  
// CS0198.cs  
class MyClass  
{  
   public static readonly int TestInt = 6;  
  
   MyClass()  
   {  
      TestInt = 11;   // CS0198, constructor is not static and readonly field is  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
