# <a name="compiler-error-cs0416"></a><span data-ttu-id="50cf3-102">编译器错误 CS0416</span><span class="sxs-lookup"><span data-stu-id="50cf3-102">Compiler Error CS0416</span></span>

 <span data-ttu-id="50cf3-104">类型形参用作了特性实参，这是不被允许的。</span><span class="sxs-lookup"><span data-stu-id="50cf3-104">A type parameter was used as an attribute argument, which is not allowed.</span></span> <span data-ttu-id="50cf3-105">使用非泛型类型</span><span class="sxs-lookup"><span data-stu-id="50cf3-105">Use a non-generic type.</span></span>

```  
// CS0416.cs

{  
   public MyAttribute(System.Type t)  
   {  
   }  
}

{  
  
   [MyAttribute(typeof(G<T>))]  // CS0416  
   public void F()  
   {  
   }  
  
}  
```
