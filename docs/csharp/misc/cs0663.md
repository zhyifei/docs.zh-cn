# <a name="compiler-error-cs0663"></a><span data-ttu-id="cdff5-102">编译器错误 CS0663</span><span class="sxs-lookup"><span data-stu-id="cdff5-102">Compiler Error CS0663</span></span>
<span data-ttu-id="cdff5-103">无法定义仅在 ref 和 out 上存在不同的重载方法。</span><span class="sxs-lookup"><span data-stu-id="cdff5-103">Cannot define overloaded methods that differ only on ref and out.</span></span>  
  
 <span data-ttu-id="cdff5-104">仅在他们使用存在不同的方法[中](../../csharp/language-reference/keywords/in-parameter-modifier.md)， [ref](../../csharp/language-reference/keywords/ref.md)和[出](../../csharp/language-reference/keywords/out-parameter-modifier.md)不允许参数上。</span><span class="sxs-lookup"><span data-stu-id="cdff5-104">Methods that differ only on their use of [in](../../csharp/language-reference/keywords/in-parameter-modifier.md), [ref](../../csharp/language-reference/keywords/ref.md) and [out](../../csharp/language-reference/keywords/out-parameter-modifier.md) on a parameter are not allowed.</span></span>  
  
 <span data-ttu-id="cdff5-105">以下示例生成 CS0663：</span><span class="sxs-lookup"><span data-stu-id="cdff5-105">The following sample generates CS0663:</span></span>  
  
```csharp  
// CS0663.cs  
class TestClass  
{  
   public static void Main()  
   {  
   }  
  
   public void Test(ref int i)  
   {  
   }  
  
   public void Test(out int i)   // CS0663  
   {  
   }  
}  
```
