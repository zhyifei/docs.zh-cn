## <a name="example"></a><span data-ttu-id="92d60-111">示例</span><span class="sxs-lookup"><span data-stu-id="92d60-111">Example</span></span>  
 <span data-ttu-id="92d60-112">以下代码生成 CS0312：</span><span class="sxs-lookup"><span data-stu-id="92d60-112">The following code generates CS0312:</span></span>  
  
```csharp

    static void MTyVar<T, U>() where T : U { }  
  
    static int Main()  
    {  
        MTyVar<int?, int>(); // CS0312  
        return 1;  
    }  
}  
```
