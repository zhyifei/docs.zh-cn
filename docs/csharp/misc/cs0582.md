# <a name="compiler-error-cs0582"></a>编译器错误 CS0582
Conditional 在接口成员上无效  
  
 **ConditionalAttribute** 在接口成员上无效。  
  
 下面的示例生成 CS0582：  
  
```csharp  
// CS0582.cs  
// compile with: /target:library  
using System.Diagnostics;  
interface MyIFace  
{  
   [ConditionalAttribute("DEBUG")]   // CS0582  
   void zz();  
}  
```
