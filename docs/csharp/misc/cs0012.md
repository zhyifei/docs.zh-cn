# <a name="compiler-error-cs0012"></a><span data-ttu-id="c8a1c-102">编译器错误 CS0012</span><span class="sxs-lookup"><span data-stu-id="c8a1c-102">Compiler Error CS0012</span></span>
<span data-ttu-id="c8a1c-103">在未被引用的程序集中定义了类型“type”。</span><span class="sxs-lookup"><span data-stu-id="c8a1c-103">The type 'type' is defined in an assembly that is not referenced.</span></span> <span data-ttu-id="c8a1c-104">必须添加对程序集“assembly”的引用。</span><span class="sxs-lookup"><span data-stu-id="c8a1c-104">You must add a reference to assembly 'assembly'.</span></span>  
  
 <span data-ttu-id="c8a1c-105">找不到引用类型的定义。</span><span class="sxs-lookup"><span data-stu-id="c8a1c-105">The definition for a referenced type was not found.</span></span> <span data-ttu-id="c8a1c-106">如果编译中不包含所需 .DLL 文件，则可能出现这种情况。</span><span class="sxs-lookup"><span data-stu-id="c8a1c-106">This could occur if a required .DLL file is not included in the compilation.</span></span> <span data-ttu-id="c8a1c-107">有关详细信息，请参阅[Add Reference Dialog Box](/visualstudio/ide/how-to-add-or-remove-references-by-using-the-reference-manager)并[/reference (C#编译器选项)](../../csharp/language-reference/compiler-options/reference-compiler-option.md)。</span><span class="sxs-lookup"><span data-stu-id="c8a1c-107">For more information, see [Add Reference Dialog Box](/visualstudio/ide/how-to-add-or-remove-references-by-using-the-reference-manager) and [/reference (C# Compiler Options)](../../csharp/language-reference/compiler-options/reference-compiler-option.md).</span></span>  
  
 <span data-ttu-id="c8a1c-108">下面的编译序列将导致 CS0012：</span><span class="sxs-lookup"><span data-stu-id="c8a1c-108">The following sequence of compilations will result in CS0012:</span></span>  
  
```csharp  
// cs0012a.cs  
// compile with: /target:library  
public class A {}  
```  
  
 <span data-ttu-id="c8a1c-109">然后：</span><span class="sxs-lookup"><span data-stu-id="c8a1c-109">Then:</span></span>  
  
```csharp  
// cs0012b.cs  
// compile with: /target:library /reference:cs0012a.dll  
public class B  
{  
   public static A f()  
   {  
      return new A();  
   }  
}  
```  
  
 <span data-ttu-id="c8a1c-110">然后：</span><span class="sxs-lookup"><span data-stu-id="c8a1c-110">Then:</span></span>  
  
```csharp  
// cs0012c.cs  
// compile with: /reference:cs0012b.dll  
class C  
{  
   public static void Main()  
   {  
      object o = B.f();   // CS0012  
   }  
}  
```  
  
 <span data-ttu-id="c8a1c-111">可通过使用 `/reference:cs0012b.dll;cs0012a.dll`进行编译解决此 CS0012，或在 Visual Studio 中通过使用 [Add Reference Dialog Box](/visualstudio/ide/how-to-add-or-remove-references-by-using-the-reference-manager) 添加对 cs0012a.dll 以及 cs0012b.dll 的引用。</span><span class="sxs-lookup"><span data-stu-id="c8a1c-111">You could resolve this CS0012 by compiling with `/reference:cs0012b.dll;cs0012a.dll`, or in Visual Studio by using the [Add Reference Dialog Box](/visualstudio/ide/how-to-add-or-remove-references-by-using-the-reference-manager) to add a reference to cs0012a.dll in addition to cs0012b.dll.</span></span>
