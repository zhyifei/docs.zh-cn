# <a name="compiler-error-cs1665"></a>编译器错误 CS1665
固定大小缓冲区的长度必须大于零  
  
 如果将固定大小的缓冲区声明为零或负数，则会发生此错误。 固定大小的缓冲区的长度必须是正整数。  
  
## <a name="example"></a>示例  
 下面的示例生成 CS1665。  
  
```csharp  
// CS1665.cs  
// compile with: /unsafe /target:library  
struct S  
{   
   public unsafe fixed int A[0];   // CS1665  
}  
```
