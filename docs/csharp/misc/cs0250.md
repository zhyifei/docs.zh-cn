# <a name="compiler-error-cs0250"></a>编译器错误 CS0250
请勿直接调用基类 Finalize 方法。 将从析构函数自动调用它。  
  
 程序不能尝试强制清理基类资源。  
  
 有关详细信息，请参阅 [Finalize 方法和析构函数](http://msdn.microsoft.com/en-us/fd376774-1643-499b-869e-9546a3aeea70)  
  
 下面的示例生成 CS0250  
  
```  
// CS0250.cs  
class B  
{  
}  
  
class C : B  
{  
   ~C()  
   {  
      base.Finalize();   // CS0250  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
