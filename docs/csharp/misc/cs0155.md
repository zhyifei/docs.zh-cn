# <a name="compiler-error-cs0155"></a><span data-ttu-id="6a708-102">编译器错误 CS0155</span><span class="sxs-lookup"><span data-stu-id="6a708-102">Compiler Error CS0155</span></span>
<span data-ttu-id="6a708-103">捕获或抛弃的类型必须从 System.Exception 派生</span><span class="sxs-lookup"><span data-stu-id="6a708-103">The type caught or thrown must be derived from System.Exception</span></span>  
  
 <span data-ttu-id="6a708-104">尝试将不是派生自 **System.Exception** 的数据类型传递到 [catch](../../csharp/language-reference/keywords/try-catch.md) 块中。</span><span class="sxs-lookup"><span data-stu-id="6a708-104">An attempt was made to pass a data type that does not derive from **System.Exception** into a [catch](../../csharp/language-reference/keywords/try-catch.md) block.</span></span> <span data-ttu-id="6a708-105">只有派生自 **System.Exception** 的数据类型才能传递到 **catch** 块中。</span><span class="sxs-lookup"><span data-stu-id="6a708-105">Only data types that derive from **System.Exception** can be passed into a **catch** block.</span></span> <span data-ttu-id="6a708-106">有关详细信息，请参阅[异常处理语句](../../csharp/language-reference/keywords/exception-handling-statements.md)和[异常和异常处理](../../csharp/programming-guide/exceptions/index.md)。</span><span class="sxs-lookup"><span data-stu-id="6a708-106">For more information, see [Exception Handling Statements](../../csharp/language-reference/keywords/exception-handling-statements.md) and [Exceptions and Exception Handling](../../csharp/programming-guide/exceptions/index.md).</span></span>  
  
 <span data-ttu-id="6a708-107">下面的示例生成 CS0155：</span><span class="sxs-lookup"><span data-stu-id="6a708-107">The following sample generates CS0155:</span></span>  
  
```  
// CS0155.cs  
using System;  
  
namespace MyNamespace  
{  
    public class MyClass2  
    // try the following line instead  
    // public class MyClass2 : Exception  
    {  
    }  
    public class MyClass  
    {  
        public static void Main()  
        {  
            try  
            {  
            }  
            catch (MyClass2)   // CS0155, resolves if you derive MyClass2 from Exception  
            {  
            }  
        }  
    }  
}  
```
