# <a name="compiler-error-cs0601"></a><span data-ttu-id="57b63-102">编译器错误 CS0601</span><span class="sxs-lookup"><span data-stu-id="57b63-102">Compiler Error CS0601</span></span>
<span data-ttu-id="57b63-103">必须在标记为 "static" 和 "extern" 的方法上指定 DllImport 特性</span><span class="sxs-lookup"><span data-stu-id="57b63-103">The DllImport attribute must be specified on a method marked 'static' and 'extern'</span></span>  
  
 <span data-ttu-id="57b63-104">在没有正确的访问关键字的方法上使用 `DllImport` 特性。</span><span class="sxs-lookup"><span data-stu-id="57b63-104">The `DllImport` attribute was used on a method that did not have the correct access keywords.</span></span>  
  
 <span data-ttu-id="57b63-105">以下示例生成 CS0601：</span><span class="sxs-lookup"><span data-stu-id="57b63-105">The following sample generates CS0601:</span></span>  
  
```  
// CS0601.cs  
using System.Runtime.InteropServices;  
using System.Text;  
  
public class C  
{  
   [DllImport("KERNEL32.DLL")]  
   extern int GetCurDirectory(int bufSize, StringBuilder buf);   // CS0601  
   // Try the following line instead:  
   // static extern int GetCurDirectory(int bufSize, StringBuilder buf);  
}  
  
public class MainClass  
{  
   public static void Main ()  
   {  
   }  
}  
```
