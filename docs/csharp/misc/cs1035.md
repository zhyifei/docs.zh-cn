# <a name="compiler-error-cs1035"></a>编译器错误 CS1035
发现文件尾，应输入 "*/"  
  
 注释起始分隔符与结尾分隔符不匹配。  
  
 下面的示例生成 CS1035：  
  
```csharp  
// CS1035.cs  
public class a  
{  
   public static void Main()  
   {  
   }  
}  
/*   // CS1035, needs closing comment  
```
