# <a name="compiler-error-cs2024"></a>编译器错误 CS2024
文件节的对齐编号“#”无效  
  
 传递给[-filealign](../language-reference/compiler-options/filealign-compiler-option.md)编译器选项的值无效。  
  
 下面的示例生成 CS2024：  
  
```csharp  
// CS2024.cs  
// compile with: /filealign:ex  
// CS2024 expected  
class MyClass  
{  
   public static void Main()  
   {  
   }  
}  
```
