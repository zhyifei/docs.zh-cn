# <a name="compiler-error-cs2024"></a>编译器错误 CS2024
文件节的对齐编号“#”无效  
  
 向 [/filealign](../../csharp/language-reference/compiler-options/filealign-compiler-option.md) 编译器选项传递的值无效。  
  
 下面的示例生成 CS2024：  
  
```  
// CS2024.cs  
// compile with: /filealign:ex  
// CS2024 expected  
class MyClass  
{  
   public static void Main()  
   {  
   }  
}  
```
