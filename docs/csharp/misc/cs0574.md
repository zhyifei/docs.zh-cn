# <a name="compiler-error-cs0574"></a>编译器错误 CS0574
析构函数的名称必须与类的名称匹配  
  
 析构函数的名称必须为前面是波形符 (~) 的类名称。  
  
 下面的示例生成 CS0574：  
  
```csharp  
// CS0574.cs  
namespace x  
{  
   public class iii  
   {  
      ~iiii()   // CS0574  
      {  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
