# <a name="compiler-error-cs0621"></a>编译器错误 CS0621
“member”：虚拟成员或抽象成员不能是私有的  
  
 不允许私有的 **虚拟** 或 **抽象** 成员。  
  
 下面的示例生成 CS0621:  
  
```csharp  
// CS0621.cs  
abstract class MyClass  
{  
   private virtual void DoNothing1()   // CS0621  
   {  
   }  
  
   private abstract void DoNothing2();   // CS0621  
  
   public static void Main()  
   {  
   }  
}  
```
