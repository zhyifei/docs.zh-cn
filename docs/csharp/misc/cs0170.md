 <span data-ttu-id="95723-104">结构中的字段在使用前未初始化。</span><span class="sxs-lookup"><span data-stu-id="95723-104">A field in a structure was used without first being initialized.</span></span> <span data-ttu-id="95723-105">若要解决此问题，首先确定哪个字段尚未初始化，然后将其初始化，再尝试访问该字段。</span><span class="sxs-lookup"><span data-stu-id="95723-105">To solve this problem, first determine which field was uninitialized and then initialize it before you try to access it.</span></span> <span data-ttu-id="95723-106">有关初始化结构的详细信息，请参阅[结构](../../csharp/programming-guide/classes-and-structs/structs.md)和[使用结构](../../csharp/programming-guide/classes-and-structs/using-structs.md)。</span><span class="sxs-lookup"><span data-stu-id="95723-106">For more information about initializing structs, see [Structs](../../csharp/programming-guide/classes-and-structs/structs.md) and [Using Structs](../../csharp/programming-guide/classes-and-structs/using-structs.md).</span></span>  
  
 <span data-ttu-id="95723-107">下面的示例生成 CS0170：</span><span class="sxs-lookup"><span data-stu-id="95723-107">The following sample generates CS0170:</span></span>  
  
```  
// CS0170.cs  
public struct error  
{  
   public int i;  
}  
  
public class MyClass  
{  
   public static void Main()  
   {  
      error e;  
      // uncomment the next line to resolve this error  
      // e.i = 0;  
      System.Console.WriteLine( e.i );   // CS0170 because   
                                         //e.i was never assigned  
   }  
}  
```
