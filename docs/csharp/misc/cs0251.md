# <a name="compiler-warning-level-2-cs0251"></a><span data-ttu-id="b850b-102">编译器警告（等级 2）CS0251</span><span class="sxs-lookup"><span data-stu-id="b850b-102">Compiler Warning (level 2) CS0251</span></span>
<span data-ttu-id="b850b-103">用负索引对数组进行索引(数组索引总是从零开始)</span><span class="sxs-lookup"><span data-stu-id="b850b-103">Indexing an array with a negative index (array indices always start at zero)</span></span>  
  
 <span data-ttu-id="b850b-104">对于指向数组内的索引，请勿使用负数。</span><span class="sxs-lookup"><span data-stu-id="b850b-104">Do not use a negative number to index into an array.</span></span>  
  
 <span data-ttu-id="b850b-105">下面的示例生成 CS0251：</span><span class="sxs-lookup"><span data-stu-id="b850b-105">The following sample generates CS0251:</span></span>  
  
```csharp  
// CS0251.cs  
// compile with: /W:2  
class MyClass  
{  
   public static void Main()  
   {  
      int[] myarray = new int[] {1,2,3};     
      try  
      {  
         myarray[-1]++;   // CS0251  
         // try the following line instead  
         // myarray[1]++;  
      }  
      catch (System.IndexOutOfRangeException e)  
      {  
         System.Console.WriteLine("{0}", e);  
      }  
   }  
}  
```
