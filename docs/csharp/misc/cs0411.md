## <a name="example"></a><span data-ttu-id="bce1e-107">示例</span><span class="sxs-lookup"><span data-stu-id="bce1e-107">Example</span></span>  
 <span data-ttu-id="bce1e-108">以下示例生成 CS0411：</span><span class="sxs-lookup"><span data-stu-id="bce1e-108">The following sample generates CS0411:</span></span>  
  
```  
// CS0411.cs  
class C  
{  
    void G<T>()  
    {  
    }  
  
    public static void Main()  
    {  
        G();  // CS0411  
        // Try this instead:  
        // G<int>();  
    }  
}  
```
