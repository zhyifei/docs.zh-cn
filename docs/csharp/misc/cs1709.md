# <a name="compiler-warning-level-1-cs1709"></a>编译器警告（等级 1）CS1709
为预处理器指令指定的文件名为空  
  
 你指定了包含文件名的预处理器指令，但该文件为空。 若要解决此警告，请将所需内容放入该文件。  
  
## <a name="example"></a>示例  
 下面的示例生成 CS1709。  
  
```csharp  
// CS1709.cs  
class Test  
{  
    static void Main()  
    {  
        #pragma checksum "" "{406EA660-64CF-4C82-B6F0-42D48172A799}" ""  // CS1709  
    }  
}  
```
