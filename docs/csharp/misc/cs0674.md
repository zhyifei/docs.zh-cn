# <a name="compiler-error-cs0674"></a>编译器错误 CS0674
不要使用“System.ParamArrayAttribute”。 请改用“params”关键字。  
  
 C# 编译器不允许使用<xref:System.ParamArrayAttribute?displayProperty=nameWithType>; 使用[params](../../csharp/language-reference/keywords/params.md)相反。  
  
 以下示例生成 CS0674：  
  
```csharp  
// CS0674.cs  
using System;  
public class MyClass   
{  
  
   public static void UseParams([ParamArray] int[] list)   // CS0674  
   // try the following line instead  
   // public static void UseParams(params int[] list)   
   {  
      for ( int i = 0 ; i < list.Length ; i++ )  
         Console.WriteLine(list[i]);  
      Console.WriteLine();  
   }  
  
   public static void Main()   
   {  
      UseParams(1, 2, 3);  
   }  
}  
```
