# <a name="compiler-error-cs2007"></a>编译器错误 CS2007
无法识别的命令行选项:“option”  
  
 向编译器传递了不是 [编译器选项](../../csharp/language-reference/compiler-options/index.md)的字符串，即使它以正斜杠 (/) 开头。  
  
 下面的示例生成 CS2007：  
  
```csharp  
// CS2007.cs  
// compile with: /recur  
// CS2007 expected  
class x  
{  
   public static void Main() {}  
}  
```
