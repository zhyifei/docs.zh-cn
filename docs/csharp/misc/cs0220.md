# <a name="compiler-error-cs0220"></a><span data-ttu-id="8c81f-102">编译器错误 CS0220</span><span class="sxs-lookup"><span data-stu-id="8c81f-102">Compiler Error CS0220</span></span>
<span data-ttu-id="8c81f-103">在 checked 模式下，运算在编译时溢出</span><span class="sxs-lookup"><span data-stu-id="8c81f-103">The operation overflows at compile time in checked mode</span></span>  
  
 <span data-ttu-id="8c81f-104">[已检查](../../csharp/language-reference/keywords/checked.md)（这是默认操作）检测到一个操作，结果导致数据丢失。</span><span class="sxs-lookup"><span data-stu-id="8c81f-104">An operation was detected by [checked](../../csharp/language-reference/keywords/checked.md), which is the default, and a data loss resulted.</span></span> <span data-ttu-id="8c81f-105">更正赋值的输入或使用 [未检查](../../csharp/language-reference/keywords/unchecked.md) 来解决此错误。</span><span class="sxs-lookup"><span data-stu-id="8c81f-105">Either correct the inputs to the assignment or use [unchecked](../../csharp/language-reference/keywords/unchecked.md) to resolve this error.</span></span> <span data-ttu-id="8c81f-106">有关详细信息，请参阅[已选中和未选中](../../csharp/language-reference/keywords/checked-and-unchecked.md)。</span><span class="sxs-lookup"><span data-stu-id="8c81f-106">For more information, see [Checked and Unchecked](../../csharp/language-reference/keywords/checked-and-unchecked.md).</span></span>  
  
 <span data-ttu-id="8c81f-107">以下示例生成 CS0220：</span><span class="sxs-lookup"><span data-stu-id="8c81f-107">The following sample generates CS0220:</span></span>  
  
```csharp  
// CS0220.cs  
using System;  
  
class TestClass  
{  
   const int x = 1000000;  
   const int y = 1000000;  
  
   public int MethodCh()  
   {  
      int z = (x * y);   // CS0220  
      return z;  
   }  
  
   public int MethodUnCh()  
   {  
      unchecked  
      {  
         int z = (x * y);  
         return z;  
      }  
   }  
  
   public static void Main()  
   {  
      TestClass myObject = new TestClass();  
      Console.WriteLine("Checked  : {0}", myObject.MethodCh());  
      Console.WriteLine("Unchecked: {0}", myObject.MethodUnCh());  
   }  
}  
```
