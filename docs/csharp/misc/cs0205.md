 <span data-ttu-id="2ba73-106">下面的示例生成 CS0205：</span><span class="sxs-lookup"><span data-stu-id="2ba73-106">The following sample generates CS0205:</span></span>  
  
```csharp  
// CS0205.cs

   abstract public void M();  
}

{  
   public override void M()  
   {  
      base.M();   // CS0205, delete this line  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
