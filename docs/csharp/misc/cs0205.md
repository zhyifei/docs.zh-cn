 <span data-ttu-id="5c140-106">下面的示例生成 CS0205：</span><span class="sxs-lookup"><span data-stu-id="5c140-106">The following sample generates CS0205:</span></span>

```csharp  
   abstract public void M();  
}

      base.M();   // CS0205, delete this line

   {  
   }  
}  
```
