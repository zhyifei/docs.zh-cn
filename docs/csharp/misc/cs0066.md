# <a name="compiler-error-cs0066"></a><span data-ttu-id="37d40-102">编译器错误 CS0066</span><span class="sxs-lookup"><span data-stu-id="37d40-102">Compiler Error CS0066</span></span>
<span data-ttu-id="37d40-103">“event”: 事件必须是委托类型</span><span class="sxs-lookup"><span data-stu-id="37d40-103">'event': event must be of a delegate type</span></span>  
  
 <span data-ttu-id="37d40-104">事件关键字需要 [委托](../language-reference/builtin-types/reference-types.md) 类型。</span><span class="sxs-lookup"><span data-stu-id="37d40-104">The event keyword requires a [delegate](../language-reference/builtin-types/reference-types.md) type.</span></span> <span data-ttu-id="37d40-105">有关详细信息，请参阅[事件](../programming-guide/events/index.md)和[委托](../programming-guide/delegates/index.md)。</span><span class="sxs-lookup"><span data-stu-id="37d40-105">For more information, see [Events](../programming-guide/events/index.md) and [Delegates](../programming-guide/delegates/index.md).</span></span>  
  
 <span data-ttu-id="37d40-106">下面的示例生成 CS0066：</span><span class="sxs-lookup"><span data-stu-id="37d40-106">The following sample generates CS0066:</span></span>  
  
```csharp  
// CS0066.cs  
using System;  
  
public class EventHandler  
{  
}  
  
// to fix the error, remove the event declaration and the  
// EventHandler class declaration, and uncomment the following line  
// public delegate void EventHandler();  
  
public class a  
{  
   public event EventHandler Click;   // CS0066  
  
   private void TestMethod()  
   {  
      if (Click != null)  
         Click();  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
