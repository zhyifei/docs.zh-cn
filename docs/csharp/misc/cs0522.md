# <a name="compiler-error-cs0522"></a><span data-ttu-id="8db85-102">编译器错误 CS0522</span><span class="sxs-lookup"><span data-stu-id="8db85-102">Compiler Error CS0522</span></span>
<span data-ttu-id="8db85-103">“constructor”：结构无法调用基类构造函数</span><span class="sxs-lookup"><span data-stu-id="8db85-103">'constructor' : structs cannot call base class constructors</span></span>  
  
 <span data-ttu-id="8db85-104">[结构](../../csharp/language-reference/keywords/struct.md) 不能调用基类构造函数；删除对基类构造函数的调用。</span><span class="sxs-lookup"><span data-stu-id="8db85-104">A [struct](../../csharp/language-reference/keywords/struct.md) cannot call a base class constructor; remove the call to the base class constructor.</span></span>  
  
 <span data-ttu-id="8db85-105">以下示例生成 CS0522：</span><span class="sxs-lookup"><span data-stu-id="8db85-105">The following sample generates CS0522:</span></span>  
  
```csharp  
// CS0522.cs  
public class clx  
{  
   public clx(int i)  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
  
public struct cly  
{  
   public cly(int i):base(0)   // CS0522  
   // try the following line instead  
   // public cly(int i)  
   {  
   }  
}  
```
