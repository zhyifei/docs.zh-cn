<span data-ttu-id="5b6c2-103">运算符“operator”对于“type”类型的操作数具有二义性</span><span class="sxs-lookup"><span data-stu-id="5b6c2-103">Operator 'operator' is ambiguous on an operand of type 'type'</span></span>  
  
 <span data-ttu-id="5b6c2-104">编译器有多个可用的转换，但在应用运算符之前不知道选择哪一个。</span><span class="sxs-lookup"><span data-stu-id="5b6c2-104">The compiler has more than one available conversion and does not know which to choose before applying the operator.</span></span> <span data-ttu-id="5b6c2-105">有关详细信息，请参阅[模板化用户定义的转换](/cpp/misc/templated-user-defined-conversions)和[转换运算符](../../csharp/programming-guide/statements-expressions-operators/conversion-operators.md)。</span><span class="sxs-lookup"><span data-stu-id="5b6c2-105">For more information, see [Templated User Defined Conversions](/cpp/misc/templated-user-defined-conversions) and [Conversion Operators](../../csharp/programming-guide/statements-expressions-operators/conversion-operators.md).</span></span>  
  
 <span data-ttu-id="5b6c2-106">下面的示例生成 CS0035：</span><span class="sxs-lookup"><span data-stu-id="5b6c2-106">The following sample generates CS0035:</span></span>  
  
```csharp  
// CS0035.cs  
class MyClass  
{  
   private int i;  
  
   public MyClass(int i)  
   {  
      this.i = i;  
   }  
  
   public static implicit operator double(MyClass x)  
   {  
      return (double) x.i;  
   }  
  
   public static implicit operator decimal(MyClass x)  
   {  
      return (decimal) x.i;  
   }  
}  
  
class MyClass2  
{  
   static void Main()  
   {  
      MyClass x = new MyClass(7);  
      object o = - x;   // CS0035  
      // try a cast:  
      // object o = - (double)x;  
   }  
}  
```
