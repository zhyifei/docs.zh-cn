# <a name="compiler-error-cs0035"></a><span data-ttu-id="3da10-102">编译器错误 CS0035</span><span class="sxs-lookup"><span data-stu-id="3da10-102">Compiler Error CS0035</span></span>
<span data-ttu-id="3da10-103">运算符“operator”对于“type”类型的操作数具有二义性</span><span class="sxs-lookup"><span data-stu-id="3da10-103">Operator 'operator' is ambiguous on an operand of type 'type'</span></span>  
  
<span data-ttu-id="3da10-104">编译器有多个可用的转换，但在应用运算符之前不知道选择哪一个。</span><span class="sxs-lookup"><span data-stu-id="3da10-104">The compiler has more than one available conversion and does not know which to choose before applying the operator.</span></span>
  
<span data-ttu-id="3da10-105">下面的示例生成 CS0035：</span><span class="sxs-lookup"><span data-stu-id="3da10-105">The following sample generates CS0035:</span></span>  
  
```csharp  
// CS0035.cs  
class MyClass  
{  
   private int i;  
  
   public MyClass(int i)  
   {  
      this.i = i;  
   }  
  
   public static implicit operator double(MyClass x)  
   {  
      return (double) x.i;  
   }  
  
   public static implicit operator decimal(MyClass x)  
   {  
      return (decimal) x.i;  
   }  
}  
  
class MyClass2  
{  
   static void Main()  
   {  
      MyClass x = new MyClass(7);  
      object o = - x;   // CS0035  
      // try a cast:  
      // object o = - (double)x;  
   }  
}  
```
