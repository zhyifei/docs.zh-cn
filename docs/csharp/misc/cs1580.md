# <a name="compiler-warning-level-1-cs1580"></a><span data-ttu-id="c6dbb-102">编译器警告（等级 1）CS1580</span><span class="sxs-lookup"><span data-stu-id="c6dbb-102">Compiler Warning (level 1) CS1580</span></span>
<span data-ttu-id="c6dbb-103">XML 注释 cref 特性中参数“parameter number”的类型无效</span><span class="sxs-lookup"><span data-stu-id="c6dbb-103">Invalid type for parameter 'parameter number' in XML comment cref attribute</span></span>  
  
 <span data-ttu-id="c6dbb-104">当试图引用某个方法的重载格式时，编译器检测到了语法错误。</span><span class="sxs-lookup"><span data-stu-id="c6dbb-104">When attempting to reference an overload form of a method, the compiler detected a syntax error.</span></span> <span data-ttu-id="c6dbb-105">通常情况下，这指示指定了参数名称，而不是类型。</span><span class="sxs-lookup"><span data-stu-id="c6dbb-105">Typically, this indicates that the parameter name, and not the type, was specified.</span></span> <span data-ttu-id="c6dbb-106">在生成的 XML 文件中将显示格式不正确的行。</span><span class="sxs-lookup"><span data-stu-id="c6dbb-106">A malformed line will appear in the generated XML file.</span></span>  
  
 <span data-ttu-id="c6dbb-107">下面的示例生成 CS1580：</span><span class="sxs-lookup"><span data-stu-id="c6dbb-107">The following sample generates CS1580:</span></span>  
  
```csharp

/// <seealso cref="Test(i)"/>   // CS1580  
// try the following line instead  
// /// <seealso cref="Test(int)"/>

{  
   /// <summary>help text</summary>  
   public static void Main()  
   {  
   }  
  
   /// <summary>help text</summary>  
   public void Test(int i)  
   {  
   }  
  
   /// <summary>help text</summary>  
   public void Test(char i)  
   {  
   }  
}  
```
