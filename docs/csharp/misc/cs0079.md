# <a name="compiler-error-cs0079"></a><span data-ttu-id="7578d-102">编译器错误 CS0079</span><span class="sxs-lookup"><span data-stu-id="7578d-102">Compiler Error CS0079</span></span>
<span data-ttu-id="7578d-103">事件“event”只能出现在 += 或 -= 的左边</span><span class="sxs-lookup"><span data-stu-id="7578d-103">The event 'event' can only appear on the left hand side of += or -=</span></span>  
  
 <span data-ttu-id="7578d-104">未正确调用 [event](../../csharp/language-reference/keywords/event.md) 。</span><span class="sxs-lookup"><span data-stu-id="7578d-104">An [event](../../csharp/language-reference/keywords/event.md) was called incorrectly.</span></span> <span data-ttu-id="7578d-105">有关详细信息，请参阅[事件](../../csharp/programming-guide/events/index.md)并[委托](../../csharp/programming-guide/delegates/index.md)。</span><span class="sxs-lookup"><span data-stu-id="7578d-105">For more information, see [Events](../../csharp/programming-guide/events/index.md) and [Delegates](../../csharp/programming-guide/delegates/index.md).</span></span>  
  
 <span data-ttu-id="7578d-106">下面的示例生成 CS0079：</span><span class="sxs-lookup"><span data-stu-id="7578d-106">The following sample generates CS0079:</span></span>  
  
```csharp  
// CS0079.cs  
using System;  
  
public delegate void MyEventHandler();  
  
public class Class1  
{  
   private MyEventHandler _e;  
  
   public event MyEventHandler Pow  
   {  
      add  
      {  
         _e += value;  
         Console.WriteLine("in add accessor");  
      }  
      remove  
      {  
         _e -= value;  
         Console.WriteLine("in remove accessor");  
      }  
   }  
  
   public void Handler()  
   {  
   }  
  
   public void Fire()  
   {  
      if (_e != null)  
      {  
         Pow();   // CS0079  
         // try the following line instead  
         // _e();  
      }  
   }  
  
   public static void Main()  
   {  
      Class1 p = new Class1();  
      p.Pow += new MyEventHandler(p.Handler);  
      p._e();  
      p.Pow += new MyEventHandler(p.Handler);  
      p._e();  
      p._e -= new MyEventHandler(p.Handler);  
      if (p._e != null)  
      {  
         p._e();  
      }  
      p.Pow -= new MyEventHandler(p.Handler);  
      if (p._e != null)  
      {  
         p._e();  
      }  
   }  
}  
```
