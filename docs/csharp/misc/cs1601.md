# <a name="compiler-error-cs1601"></a>编译器错误 CS1601
方法或委托参数不能是“type”类型  
  
 .NET Framework 类库中，例如，某些类型<xref:System.TypedReference>，<xref:System.RuntimeArgumentHandle>并<xref:System.ArgIterator>不能用作[中](../../csharp/language-reference/keywords/in-parameter-modifier.md)， [ref](../../csharp/language-reference/keywords/ref.md)或[出](../../csharp/language-reference/keywords/out-parameter-modifier.md)参数由于它们可能无法用来执行不安全的操作。  
  
 下面的示例生成 CS1601：  
  
```csharp  
// CS1601.cs  
using System;

class MyClass
{
    public void Test1(in TypedReference t)   // CS1601  
    {
    }

    public void Test2(ref TypedReference t)   // CS1601  
    {
    }

    public void Test3(out ArgIterator t)   // CS1601  
    {
        t = default;
    }
}
```
