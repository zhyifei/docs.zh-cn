# <a name="compiler-error-cs0113"></a><span data-ttu-id="e6d8d-102">编译器错误 CS0113</span><span class="sxs-lookup"><span data-stu-id="e6d8d-102">Compiler Error CS0113</span></span>
<span data-ttu-id="e6d8d-103">标记为 override 的成员“function”不能标记为 new 或 virtual</span><span class="sxs-lookup"><span data-stu-id="e6d8d-103">A member 'function' marked as override cannot be marked as new or virtual</span></span>  
  
 <span data-ttu-id="e6d8d-104">使用 [new](../../csharp/language-reference/keywords/new.md) 和 [override](../../csharp/language-reference/keywords/override.md) 关键字标记一种方法是互相排斥的。</span><span class="sxs-lookup"><span data-stu-id="e6d8d-104">It is mutually exclusive to mark a method with the [new](../../csharp/language-reference/keywords/new.md) and [override](../../csharp/language-reference/keywords/override.md) keywords.</span></span>  
  
 <span data-ttu-id="e6d8d-105">下面的示例生成 CS0113：</span><span class="sxs-lookup"><span data-stu-id="e6d8d-105">The following sample generates CS0113:</span></span>  
  
```csharp  
// CS0113.cs  
namespace MyNamespace  
{  
   abstract public class MyClass  
   {  
      public abstract void Foo();  
   }  
  
   public class MyClass2 : MyClass  
   {  
      override new public void Foo()   // CS0113, remove new keyword  
      {  
      }  
  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
