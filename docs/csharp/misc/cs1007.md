 <span data-ttu-id="4a28c-104">声明 [属性](../../csharp/programming-guide/classes-and-structs/using-properties.md)时，还必须声明其访问器方法。</span><span class="sxs-lookup"><span data-stu-id="4a28c-104">When you declare a [property](../../csharp/programming-guide/classes-and-structs/using-properties.md), you must also declare its accessor methods.</span></span> <span data-ttu-id="4a28c-105">但是，属性不能拥有多个 `get` 或 `set` 访问器方法。</span><span class="sxs-lookup"><span data-stu-id="4a28c-105">However, a property cannot have more than one `get` accessor method or more than one `set` accessor method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="4a28c-106">示例</span><span class="sxs-lookup"><span data-stu-id="4a28c-106">Example</span></span>

```

{  
    public int MyProperty  
    {  
        get  
        {  
            return 0;  
        }  
        get   // CS1007, this is the second get method

            return 0;  
        }  
    }  
  
    public static void Main() {}  
}  
```
