# <a name="compiler-error-cs1626"></a>编译器错误 CS1626
无法在包含 catch 子句的 Try 块体中生成值  
  
 如果有一个 catch 子句与 try 块关联，该 try 块内不允许有 yield 语句。 为了避免此错误，请将屈服语句移出 try/catch/finally 块，或删除 catch 块。
  
 下面的示例生成 CS1626：  
  
```csharp  
// CS1626.cs  
using System.Collections;  
  
class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()  
   {  
      try  
      {  
         yield return this;  // CS1626  
      }  
      catch  
      {  
  
      }  
      finally
      {

      }
   }  
}  
  
public class CMain  
{  
   public static void Main() { }  
}  
```
