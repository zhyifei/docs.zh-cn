# <a name="compiler-error-cs1552"></a>编译器错误 CS1552
数组类型说明符 [] 必须出现在参数名之前  
  
 在数组声明中，数组类型说明符在变量名之后。  
  
## <a name="example"></a>示例  
 下面的示例生成 CS1552：  
  
```  
// CS1552.cs  
public class C  
{  
    public static void Main(string args[])   // CS1552  
    // try the following line instead  
    // public static void Main(string [] args)  
    {  
    }  
}  
```
