# <a name="compiler-error-cs0070"></a><span data-ttu-id="7597a-102">编译器错误 CS0070</span><span class="sxs-lookup"><span data-stu-id="7597a-102">Compiler Error CS0070</span></span>
<span data-ttu-id="7597a-103">事件“event”只能出现在 += 或 -= 的左边（从类型“type”中使用时除外）</span><span class="sxs-lookup"><span data-stu-id="7597a-103">The event 'event' can only appear on the left hand side of += or -= (except when used from within the type 'type')</span></span>  
  
 <span data-ttu-id="7597a-104">在定义事件的类的外部， [event](../../csharp/language-reference/keywords/event.md) 只能加上或减去引用。</span><span class="sxs-lookup"><span data-stu-id="7597a-104">Outside of the class it is defined in, an [event](../../csharp/language-reference/keywords/event.md) can only add or subtract references.</span></span> <span data-ttu-id="7597a-105">有关详细信息，请参阅[事件](../../csharp/programming-guide/events/index.md)。</span><span class="sxs-lookup"><span data-stu-id="7597a-105">For more information, see [Events](../../csharp/programming-guide/events/index.md).</span></span>  
  
 <span data-ttu-id="7597a-106">下面的示例生成 CS0070：</span><span class="sxs-lookup"><span data-stu-id="7597a-106">The following sample generates CS0070:</span></span>  
  
```  
// CS0070.cs  
using System;  
public delegate void EventHandler();  
  
public class A  
{  
   public event EventHandler Click;  
  
   public static void OnClick()  
   {  
      EventHandler eh;  
      A a = new A();  
      eh = a.Click;  
   }  
  
   public static void Main()  
   {  
   }  
}  
  
public class B  
{  
   public int Foo ()  
   {  
      EventHandler eh = new EventHandler(A.OnClick);  
      A a = new A();  
      eh = a.Click;   // CS0070  
      // try the following line instead  
      // a.Click += eh;  
      return 1;  
   }  
}  
```
