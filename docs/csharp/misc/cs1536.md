# <a name="compiler-error-cs1536"></a>编译器错误 CS1536
参数类型 void 无效  
  
 不需要指定除 void 指针以外的 [void](../language-reference/keywords/void.md) 参数或指定无效。  
  
 下面的示例生成 CS1536：  
  
```csharp  
// CS1536.cs  
class a  
{  
   public static int x( void )   // CS1536  
   // try the following line instead  
   // public static int x()  
   {  
      return 0;  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
