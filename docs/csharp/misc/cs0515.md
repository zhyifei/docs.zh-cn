# <a name="compiler-error-cs0515"></a>编译器错误 CS0515
“function”：静态构造函数中不允许出现访问修饰符  
  
 静态构造函数不能包含 [访问修饰符](/dotnet/csharp/language-reference/keywords)。  
  
## <a name="example"></a>示例  
 下面的示例生成 CS0515：  
  
```csharp  
// CS0515.cs  
public class Clx  
{  
    public static void Main()  
    {  
    }  
}  
  
public class Clz  
{  
    public static Clz()   // CS0515, remove public keyword  
    {  
    }  
}  
```
