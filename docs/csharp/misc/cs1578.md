# <a name="compiler-error-cs1578"></a>编译器错误 CS1578
应输入文件名、单行注释或行尾  
  
 [#line](../../csharp/language-reference/preprocessor-directives/preprocessor-line.md) 指令后面只能出现文件名（双引号括起）或单行注释。  
  
 以下示例生成 CS1578：  
  
```csharp  
// CS1578.cs  
class MyClass  
{  
   static void Main()  
   {  
      #line 101 abc.cs   // CS1578  
      // try the following line instead  
      //#line 101 "abc.cs"  
      intt i;          // error will be reported on line 101  
   }  
}  
```
