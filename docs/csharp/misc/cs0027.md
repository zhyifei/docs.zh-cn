# <a name="compiler-error-cs0027"></a><span data-ttu-id="a08d6-102">编译器错误 CS0027</span><span class="sxs-lookup"><span data-stu-id="a08d6-102">Compiler Error CS0027</span></span>
<span data-ttu-id="a08d6-103">关键字 "this" 在当前上下文中不可用</span><span class="sxs-lookup"><span data-stu-id="a08d6-103">Keyword 'this' is not available in the current context</span></span>  
  
 <span data-ttu-id="a08d6-104">在属性、方法或构造函数的外部发现了 [this](../../csharp/language-reference/keywords/this.md) 关键字。</span><span class="sxs-lookup"><span data-stu-id="a08d6-104">The [this](../../csharp/language-reference/keywords/this.md) keyword was found outside of a property, method, or constructor.</span></span>  
  
 <span data-ttu-id="a08d6-105">若要修复此错误，请修改语句以消除 `this` 关键字，关键字的使用，并且/或者将语句的一部分或全部移到属性、方法或构造函数的内部。</span><span class="sxs-lookup"><span data-stu-id="a08d6-105">To fix this error, either modify the statement to eliminate use of the `this` keyword, and/or move part or all of the statement inside a property, method, or constructor.</span></span>  
  
 <span data-ttu-id="a08d6-106">下面的示例生成 CS0027：</span><span class="sxs-lookup"><span data-stu-id="a08d6-106">The following example generates CS0027:</span></span>  
  
```csharp  
using System;  
using System.Collections.Generic;  
using System.Text;  
  
namespace ConsoleApplication3  
{  
    class MyClass  
    {  
  
        int err1 = this.Fun() + 1;  // CS0027   
  
        public int Fun()  
        {  
            return 10;  
        }  
  
        public void Test()  
        {  
            // valid use of this  
            int err = this.Fun() + 1;  
            Console.WriteLine(err);  
        }  
  
        public static void Main()  
        {  
            MyClass c = new MyClass();  
            c.Test();  
        }  
    }  
}  
```
