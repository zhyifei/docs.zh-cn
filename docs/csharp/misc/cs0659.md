# <a name="compiler-warning-level-3-cs0659"></a><span data-ttu-id="75d6a-102">编译器警告（等级 3）CS0659</span><span class="sxs-lookup"><span data-stu-id="75d6a-102">Compiler Warning (level 3) CS0659</span></span>

<span data-ttu-id="75d6a-103">“class”重写 Object.Equals(object o) 但不重写 Object.GetHashCode()</span><span class="sxs-lookup"><span data-stu-id="75d6a-103">'class' overrides Object.Equals(object o) but does not override Object.GetHashCode()</span></span>  

<span data-ttu-id="75d6a-104">编译器检测到的重写<xref:System.Object.Equals%2A?displayProperty=nameWithType>方法，但不是重写的<xref:System.Object.GetHashCode%2A?displayProperty=nameWithType>方法。</span><span class="sxs-lookup"><span data-stu-id="75d6a-104">The compiler detected an override of the <xref:System.Object.Equals%2A?displayProperty=nameWithType> method but no override of the <xref:System.Object.GetHashCode%2A?displayProperty=nameWithType> method.</span></span> <span data-ttu-id="75d6a-105">重写<xref:System.Object.Equals%2A>意味着你也想要重写<xref:System.Object.GetHashCode%2A>。</span><span class="sxs-lookup"><span data-stu-id="75d6a-105">An override of <xref:System.Object.Equals%2A> implies that you also want to override <xref:System.Object.GetHashCode%2A>.</span></span>  

<span data-ttu-id="75d6a-106">下面的代码生成 CS0659:</span><span class="sxs-lookup"><span data-stu-id="75d6a-106">The following code generates CS0659:</span></span>  

```csharp
// CS0659.cs  
// compile with: /W:3 /target:library  
class Test
{  
   public override bool Equals(object o) { return true; }   // CS0659  
}  
  
// OK  
class Test2  
{  
   public override bool Equals(object o) { return true; }  
   public override int GetHashCode() { return 0; }  
}  
```
