# <a name="compiler-error-cs0744"></a><span data-ttu-id="fd888-102">编译器错误 CS0744</span><span class="sxs-lookup"><span data-stu-id="fd888-102">Compiler Error CS0744</span></span>
<span data-ttu-id="fd888-103">应为上下文关键字 "equals"</span><span class="sxs-lookup"><span data-stu-id="fd888-103">Expected contextual keyword 'equals'</span></span>  
  
 <span data-ttu-id="fd888-104">`join` 子句的模式是 `join`...`in`...`on`...`equals`，如本示例所示：</span><span class="sxs-lookup"><span data-stu-id="fd888-104">The pattern for a `join` clause is `join`...`in`...`on`...`equals`, as shown in this example:</span></span>  
  
```csharp  
var query = from x in array1  
            join y in array2 on x equals y  
            select x;  
```  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="fd888-105">更正此错误</span><span class="sxs-lookup"><span data-stu-id="fd888-105">To correct this error</span></span>  
  
1.  <span data-ttu-id="fd888-106">将 `equals` 关键字添加到 `join` 子句中。</span><span class="sxs-lookup"><span data-stu-id="fd888-106">Add the `equals` keyword to the `join` clause.</span></span>  
  
## <a name="example"></a><span data-ttu-id="fd888-107">示例</span><span class="sxs-lookup"><span data-stu-id="fd888-107">Example</span></span>  
 <span data-ttu-id="fd888-108">下面的代码生成 CS0744：</span><span class="sxs-lookup"><span data-stu-id="fd888-108">The following code generates CS0744:</span></span>  
  
```csharp  
// cs0744.cs  
using System;  
using System.Linq;  
  
public class C  
{  
    public static int Main()  
    {  
        int[] array1 = { 1, 2, 3 ,4, 5, 6,};  
        int[] array2 = { 5, 6, 7, 8, 9 };  
        var c = from x in array1  
                join y in array2 on x y // CS0744  
                select x;  
        return 1;  
    }  
}  
```
