## <a name="example"></a><span data-ttu-id="66854-105">示例</span><span class="sxs-lookup"><span data-stu-id="66854-105">Example</span></span>  
 <span data-ttu-id="66854-106">以下示例生成 CS0730。</span><span class="sxs-lookup"><span data-stu-id="66854-106">The following sample generates CS0730.</span></span> <span data-ttu-id="66854-107">它包括两个源文件。</span><span class="sxs-lookup"><span data-stu-id="66854-107">It consists of two source files.</span></span> <span data-ttu-id="66854-108">首先，编译库文件 `CS0730a.cs`，并编译引用库文件的文件 `CS0730.cs` 。</span><span class="sxs-lookup"><span data-stu-id="66854-108">First, compile the library file `CS0730a.cs`, and the compile the file `CS0730.cs` referencing the library file.</span></span>  
  
```csharp  
// CS0730a.cs  
// compile with: /t:library  
public class Outer  
{  
   public class Nested {}  
}  
```  
  
```csharp  
// CS0730.cs  
// compile with: /t:library /r:CS0730a.dll  
using System.Runtime.CompilerServices;  
  
[assembly:TypeForwardedToAttribute(typeof(Outer.Nested))]   // CS0730  
  
[assembly:TypeForwardedToAttribute(typeof(Outer))]   // OK  
```
