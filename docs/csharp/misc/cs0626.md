<span data-ttu-id="43e7b-103">方法、运算符或访问器“method”被标记为 external，且其上不具有特性。</span><span class="sxs-lookup"><span data-stu-id="43e7b-103">Method, operator, or accessor 'method' is marked external and has no attributes on it.</span></span> <span data-ttu-id="43e7b-104">请考虑添加 DllImport 特性来指定外部实现。</span><span class="sxs-lookup"><span data-stu-id="43e7b-104">Consider adding a DllImport attribute to specify the external implementation.</span></span>
  
 <span data-ttu-id="43e7b-105">标记为 `extern` 的方法也应使用特性（例如 [DllImport](xref:System.Runtime.InteropServices.DllImportAttribute) 特性）进行标记。</span><span class="sxs-lookup"><span data-stu-id="43e7b-105">A method marked `extern` should also be marked with an attribute, for example, the [DllImport](xref:System.Runtime.InteropServices.DllImportAttribute) attribute.</span></span>
  
 <span data-ttu-id="43e7b-106">该特性用于指定实现方法的位置。</span><span class="sxs-lookup"><span data-stu-id="43e7b-106">The attribute specifies where the method is implemented.</span></span> <span data-ttu-id="43e7b-107">在运行时，程序将需要此信息。</span><span class="sxs-lookup"><span data-stu-id="43e7b-107">At run time, the program will need this information.</span></span>

```csharp
   }  
}  
```
