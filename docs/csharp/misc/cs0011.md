## <a name="example"></a><span data-ttu-id="f380d-108">示例</span><span class="sxs-lookup"><span data-stu-id="f380d-108">Example</span></span>

// compile with: /target:library /reference:CS0011_1.dll  
// post-build command: del /f CS0011_1.dll

## <a name="example"></a><span data-ttu-id="f380d-110">示例</span><span class="sxs-lookup"><span data-stu-id="f380d-110">Example</span></span>
