# <a name="compiler-error-cs0501"></a>编译器错误 CS0501
“member function”必须声明主体，因为它未标记为 abstract、extern 或 partial  
  
 非抽象方法必须具有实现。  
  
 下面的示例生成 CS0501：  
  
```csharp  
// CS0501.cs  
// compile with: /target:library  
public class clx  
{  
   public void f();   // CS0501 declared but not defined  
   public void g() {}   // OK  
}  
```
