# <a name="compiler-error-cs0758"></a><span data-ttu-id="ae038-102">编译器错误 CS0758</span><span class="sxs-lookup"><span data-stu-id="ae038-102">Compiler Error CS0758</span></span>

 <span data-ttu-id="ae038-104">如果分部方法的一部分指定了 `params` 参数，则另一部分也必须指定该参数。</span><span class="sxs-lookup"><span data-stu-id="ae038-104">If one part of a partial method specifies a `params` parameter, the other part must specify one also.</span></span>

## <a name="example"></a><span data-ttu-id="ae038-107">示例</span><span class="sxs-lookup"><span data-stu-id="ae038-107">Example</span></span>

```csharp  
    public partial class C

        partial void Part(int i, params char[] array);  
        partial void Part(int i, char[] array) // CS0758

            return 1;  
        }  
  
    }  
```
