# <a name="compiler-error-cs1608"></a>编译器错误 CS1608
C# 类型上不允许有 Required 特性  
  
 在 C# 代码中定义的类型上不允许<xref:System.Runtime.CompilerServices.RequiredAttributeAttribute> 。
