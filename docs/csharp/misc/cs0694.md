# <a name="compiler-error-cs0694"></a><span data-ttu-id="3be3c-102">编译器错误 CS0694</span><span class="sxs-lookup"><span data-stu-id="3be3c-102">Compiler Error CS0694</span></span>

## <a name="example"></a><span data-ttu-id="3be3c-105">示例</span><span class="sxs-lookup"><span data-stu-id="3be3c-105">Example</span></span>

