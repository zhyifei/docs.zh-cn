# <a name="compiler-error-cs0466"></a><span data-ttu-id="e83df-102">编译器错误 CS0466</span><span class="sxs-lookup"><span data-stu-id="e83df-102">Compiler Error CS0466</span></span>
<span data-ttu-id="e83df-103">由于“method2”没有 params 参数，因此“method1”也不应当有该参数</span><span class="sxs-lookup"><span data-stu-id="e83df-103">'method1' should not have a params parameter since 'method2' does not</span></span>  
  
 <span data-ttu-id="e83df-104">如果实现的接口不使用，则不能对类成员使用 `params` 参数。</span><span class="sxs-lookup"><span data-stu-id="e83df-104">You cannot use `params` parameter on a class member if the implemented interface doesn't use it.</span></span>  
  
## <a name="example"></a><span data-ttu-id="e83df-105">示例</span><span class="sxs-lookup"><span data-stu-id="e83df-105">Example</span></span>  
 <span data-ttu-id="e83df-106">下面的示例生成 CS0466。</span><span class="sxs-lookup"><span data-stu-id="e83df-106">The following sample generates CS0466.</span></span>  
  
```csharp  
// CS0466.cs  
interface I  
{  
   void F1(params int[] a);  
   void F2(int[] a);  
}  
  
class C : I  
{  
   void I.F1(params int[] a) {}  
   void I.F2(params int[] a) {}   // CS0466  
   void I.F2(int[] a) {}   // OK  
  
   public static void Main()  
   {  
      I i = (I) new C();  
  
      i.F1(new int[] {1, 2} );  
      i.F2(new int[] {1, 2} );  
   }  
}  
```
