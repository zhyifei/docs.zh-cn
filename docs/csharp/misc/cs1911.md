# <a name="compiler-warning-level-1-cs1911"></a><span data-ttu-id="afe5b-102">编译器警告（等级 1）CS1911</span><span class="sxs-lookup"><span data-stu-id="afe5b-102">Compiler Warning (level 1) CS1911</span></span>
<span data-ttu-id="afe5b-103">通过“base”关键字从匿名方法、lambda 表达式、查询表达式或迭代器访问成员“name”会导致无法验证的代码。</span><span class="sxs-lookup"><span data-stu-id="afe5b-103">Access to member 'name' through a 'base' keyword from an anonymous method, lambda expression, query expression, or iterator results in unverifiable code.</span></span> <span data-ttu-id="afe5b-104">请考虑在包含类型上移入一个帮助程序方法的访问权限。</span><span class="sxs-lookup"><span data-stu-id="afe5b-104">Consider moving the access into a helper method on the containing type.</span></span>  
  
 <span data-ttu-id="afe5b-105">在迭代器或匿名方法的方法主体内使用 `base` 关键字调用虚函数将导致无法验证的代码。</span><span class="sxs-lookup"><span data-stu-id="afe5b-105">Calling virtual functions with the `base` keyword inside the method body of an iterator or anonymous methods will result in unverifiable code.</span></span> <span data-ttu-id="afe5b-106">无法验证的代码将无法在部分信任环境中运行。</span><span class="sxs-lookup"><span data-stu-id="afe5b-106">Unverifiable code will fail to run in a partial trust environment.</span></span>  
  
 <span data-ttu-id="afe5b-107">CS1911 的一个解决方法是将虚函数调用移到帮助程序函数中。</span><span class="sxs-lookup"><span data-stu-id="afe5b-107">One resolution for CS1911 is to move the virtual function call to a helper function.</span></span>  
  
## <a name="example"></a><span data-ttu-id="afe5b-108">示例</span><span class="sxs-lookup"><span data-stu-id="afe5b-108">Example</span></span>  
 <span data-ttu-id="afe5b-109">下面的示例生成 CS1911。</span><span class="sxs-lookup"><span data-stu-id="afe5b-109">The following sample generates CS1911.</span></span>  
  
```  
// CS1911.cs  
// compile with: /W:1  
using System;  
  
delegate void D();  
delegate D RetD();  
  
class B {  
   protected virtual void M() {  
      Console.WriteLine("B.M");  
   }  
}  
  
class Der : B {  
   protected override void M() {  
      Console.WriteLine("D.M");  
   }  
  
   void Test() { base.M(); }  
   D Test2() { return new D(base.M); }  
  
   public D CallBaseM() {  
      return delegate () { base.M(); };   // CS1911  
  
      // try the following line instead  
      // return delegate () { Test(); };  
   }  
  
   public RetD DelToBaseM() {  
      return delegate () { return new D(base.M); };   // CS1911  
  
      // try the following line instead  
      // return delegate () { return Test2(); };  
   }  
}  
  
class Program {  
   public static void Main() {  
      Der der = new Der();  
      D d = der.CallBaseM();  
      d();  
      RetD rd = der.DelToBaseM();  
      rd()();  
   }  
}  
```
