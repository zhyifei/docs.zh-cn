# <a name="compiler-error-cs0711"></a>编译器错误 CS0711
静态类不能包含析构函数  
  
 不能实例化静态类，因此它不需要构造函数或析构函数。 若要避免此错误，请从静态类中删除任何析构函数；如果你确实希望构造并销毁实例，请使用非静态类。  
  
 下面的示例生成 CS0711：  
  
```csharp  
// CS0711.cs  
public static class C  
{  
   ~C()  // CS0711  
   {  
   }  
  
   public static void Main()  
   {  
   }  
}  
```
