 <span data-ttu-id="29378-104">如果结构声明中包含标有 **FieldOffset** 特性的任何成员，则必须在该结构声明中使用 **StructLayout(LayoutKind.Explicit)** 特性。</span><span class="sxs-lookup"><span data-stu-id="29378-104">You must use the **StructLayout(LayoutKind.Explicit)** attribute on the struct declaration, if it contains any members marked with the **FieldOffset** attribute.</span></span> <span data-ttu-id="29378-105">有关详细信息，请参阅 [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute).</span><span class="sxs-lookup"><span data-stu-id="29378-105">For more information, see [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute).</span></span>

```csharp  
// CS0636.cs

using System.Runtime.InteropServices;  
  
// To resolve the error, uncomment the following line:  
// [StructLayout(LayoutKind.Explicit)]

{  
   [FieldOffset(4)]public int i;   // CS0636   
}  
  
public class MainClass  
{  
   public static void Main ()  
   {  
   }  
}  
```
