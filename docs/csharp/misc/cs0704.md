## <a name="example"></a><span data-ttu-id="83033-106">示例</span><span class="sxs-lookup"><span data-stu-id="83033-106">Example</span></span>

```

{  
    public class I { }  
}

{  
    T.I f;  // CS0704 – member lookup on type parameter  
    // Try this instead:

{  
    public static void Main() {}  
}
```
