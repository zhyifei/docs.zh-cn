# <a name="compiler-error-cs0704"></a>编译器错误 CS0704
“type”是类型参数，无法在其中执行成员查找  
  
 不能通过类型参数指定内部类型。 请尝试显式使用所需的类型。  
  
## <a name="example"></a>示例  
 以下示例生成 CS0704：  
  
```csharp  
// CS0704.cs  
class B  
{  
    public class I { }  
}  
  
class C<T> where T : B  
{  
    T.I f;  // CS0704 – member lookup on type parameter  
    // Try this instead:  
    // B.I f;  
}  
  
class CMain  
{  
    public static void Main() {}  
}  
```
