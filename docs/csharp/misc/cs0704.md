## <a name="example"></a><span data-ttu-id="c5c9b-106">示例</span><span class="sxs-lookup"><span data-stu-id="c5c9b-106">Example</span></span>  
 <span data-ttu-id="c5c9b-107">以下示例生成 CS0704：</span><span class="sxs-lookup"><span data-stu-id="c5c9b-107">The following sample generates CS0704:</span></span>  
  
```csharp

    public class I { }  
}

{  
    T.I f;  // CS0704 – member lookup on type parameter  
    // Try this instead:

}

{

}
```
