# <a name="compiler-error-cs2017"></a>编译器错误 CS2017
如果生成模块或库，则无法指定 /main  
  
 生成[目标：库](../language-reference/compiler-options/target-library-compiler-option.md)时，不能指定主入口点。  
  
 以下示例生成 CS2017：  
  
```csharp  
// CS2017.cs  
// compile with: /main:MyClass /target:library  
// CS2017 expected  
class MyClass  
{  
   public static void Main()  
   {  
   }  
}  
```
