# <a name="compiler-error-cs0572"></a><span data-ttu-id="3707c-102">编译器错误 CS0572</span><span class="sxs-lookup"><span data-stu-id="3707c-102">Compiler Error CS0572</span></span>
<span data-ttu-id="3707c-103">“type”：无法通过表达式引用类型；请另外尝试“path_to_type”</span><span class="sxs-lookup"><span data-stu-id="3707c-103">'type' : cannot reference a type through an expression; try 'path_to_type' instead</span></span>  
  
 <span data-ttu-id="3707c-104">尝试通过标识符访问类的成员，但这在此情况下不被允许。</span><span class="sxs-lookup"><span data-stu-id="3707c-104">An attempt was made to access a member of a class through an identifier, which is not permitted in this situation.</span></span>  
  
 <span data-ttu-id="3707c-105">以下示例生成 CS0572：</span><span class="sxs-lookup"><span data-stu-id="3707c-105">The following sample generates CS0572:</span></span>  
  
```csharp  
// CS0572.cs  
using System;  
class C  
{  
   public class Inner  
   {  
      public static int v = 9;  
   }  
}  
  
class D : C  
{  
   public static void Main()  
   {  
      C cValue = new C();  
      Console.WriteLine(cValue.Inner.v);   // CS0572  
      // try the following line instead  
      // Console.WriteLine(C.Inner.v);  
   }  
}  
```
