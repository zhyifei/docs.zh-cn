# <a name="compiler-error-cs1010"></a>编译器错误 CS1010
常量中有换行符  
  
 未正确分隔 [字符串](../../csharp/language-reference/keywords/string.md) 。  
  
 下面的示例生成 CS1010：  
  
```csharp  
// CS1010.cs  
class Sample  
{  
   static void Main()  
   {  
      string a = "Hello World;   // CS1010  
      // Use the following line, with end quote before semicolon:  
      string a = "Hello World"; //  
   }  
}  
```
