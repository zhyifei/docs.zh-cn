## <a name="example"></a><span data-ttu-id="88fb7-108">示例</span><span class="sxs-lookup"><span data-stu-id="88fb7-108">Example</span></span>

<span data-ttu-id="88fb7-109">下面的代码生成 CS1955，因为该代码尝试通过使用调用字段和属性[调用运算符`()` ](../language-reference/operators/member-access-operators.md#invocation-operator-)。</span><span class="sxs-lookup"><span data-stu-id="88fb7-109">The following code generates CS1955 because the code is trying to invoke a field and a property by using the [invocation operator `()`](../language-reference/operators/member-access-operators.md#invocation-operator-).</span></span> <span data-ttu-id="88fb7-110">不能调用字段或属性。</span><span class="sxs-lookup"><span data-stu-id="88fb7-110">You cannot call a field or a property.</span></span> <span data-ttu-id="88fb7-111">使用[成员访问运算符`.`](../language-reference/operators/member-access-operators.md#member-access-operator-)来访问它存储的值。</span><span class="sxs-lookup"><span data-stu-id="88fb7-111">Use the [member access operator `.`](../language-reference/operators/member-access-operators.md#member-access-operator-) to access the value it stores.</span></span>
  
```csharp  
// cs1955.cs  
class A  
{  
    public int x = 0;  
    public int X  
    {  
        get { return x; }  
        set { x = value; }  
    }  
}  
  
class Test  
{  
    static int Main()  
    {  
        A a = new A();  
        a.x(); // CS1955  
        a.X(); // CS1955  
        // Try this line instead:  
        // int num = a.x;  
    }  
}  
```
