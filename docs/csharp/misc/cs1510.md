## <a name="example"></a><span data-ttu-id="c25f3-106">示例</span><span class="sxs-lookup"><span data-stu-id="c25f3-106">Example</span></span>  
 <span data-ttu-id="c25f3-107">下面的示例生成 CS1510：</span><span class="sxs-lookup"><span data-stu-id="c25f3-107">The following sample generates CS1510:</span></span>  
  
```csharp  
// CS1510.cs  
public class C  
{  
   public static int j = 0;  
  
   public static void M(ref int j)  
   {  
      j++;  
   }  
  
   public static void Main ()  
   {  
      M (ref 2);   // CS1510, can't pass a number as a ref parameter  
      // try the following to resolve the error  
      // M (ref j);  
   }  
}  
```
