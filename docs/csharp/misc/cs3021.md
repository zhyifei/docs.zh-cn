 <span data-ttu-id="13e6e-104">如果出现此警告`[CLSCompliant(false)]`将出现在类中的程序集不包含程序集级 CLSCompliant 特性设置为 true (也就是说，行`[assembly: CLSCompliant(true)]`)。</span><span class="sxs-lookup"><span data-stu-id="13e6e-104">This warning occurs if `[CLSCompliant(false)]` appears on a class in an assembly which does not have an assembly-level CLSCompliant attribute set to true (that is, the line `[assembly: CLSCompliant(true)]`).</span></span> <span data-ttu-id="13e6e-105">由于程序集为将其自身声明为符合 CLS，所以该程序集内的任何项都不必将其自身声明为不符合，因为已将其假定为不符合。</span><span class="sxs-lookup"><span data-stu-id="13e6e-105">Since the assembly is not declaring itself CLS compliant, there is no need for anything within the assembly to declare itself non-compliant, since it is assumed to be non-compliant.</span></span> <span data-ttu-id="13e6e-106">有关 CLS 遵从性的详细信息，请参阅[语言独立性和与语言无关的组件](../../standard/language-independence.md)。</span><span class="sxs-lookup"><span data-stu-id="13e6e-106">For more information on CLS compliance, see [Language independence and language-independent components](../../standard/language-independence.md).</span></span>

## <a name="example"></a><span data-ttu-id="13e6e-108">示例</span><span class="sxs-lookup"><span data-stu-id="13e6e-108">Example</span></span>  
 <span data-ttu-id="13e6e-109">以下示例生成 CS3021：</span><span class="sxs-lookup"><span data-stu-id="13e6e-109">The following example generates CS3021:</span></span>  
  
```csharp  
// CS3021.cs  
using System;  
// Uncomment the following line to declare the assembly CLS Compliant,  
// and avoid the warning without removing the attribute on the class.  
//[assembly: CLSCompliant(true)]  
  
// Remove the next line to avoid the warning.  
[CLSCompliant(false)]               // CS3021  
public class C  
{  
    public static void Main()  
    {  
    }  
}  
```
