# <a name="compiler-error-cs0127"></a><span data-ttu-id="ea78f-102">编译器错误 CS0127</span><span class="sxs-lookup"><span data-stu-id="ea78f-102">Compiler Error CS0127</span></span>
<span data-ttu-id="ea78f-103">由于“function”返回 void，返回关键字后面不得有对象表达式</span><span class="sxs-lookup"><span data-stu-id="ea78f-103">Since 'function' returns void, a return keyword must not be followed by an object expression</span></span>  
  
 <span data-ttu-id="ea78f-104">具有 [void](../../csharp/language-reference/keywords/void.md) 返回类型的方法不能返回值。</span><span class="sxs-lookup"><span data-stu-id="ea78f-104">A method with a [void](../../csharp/language-reference/keywords/void.md) return type cannot return a value.</span></span> <span data-ttu-id="ea78f-105">有关详细信息，请参阅[方法](../../csharp/programming-guide/classes-and-structs/methods.md)。</span><span class="sxs-lookup"><span data-stu-id="ea78f-105">For more information, see [Methods](../../csharp/programming-guide/classes-and-structs/methods.md).</span></span>  
  
 <span data-ttu-id="ea78f-106">下面的示例生成 CS0127：</span><span class="sxs-lookup"><span data-stu-id="ea78f-106">The following sample generates CS0127:</span></span>  
  
```csharp  
// CS0127.cs  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public int hiddenMember2  
      {  
         get  
         {  
            return 0;  
         }  
         set   // CS0127, set has an implicit void return type  
         {  
            return 0;   // remove return statement to resolve this CS0127  
         }  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
