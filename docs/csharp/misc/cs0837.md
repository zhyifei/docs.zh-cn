# <a name="compiler-error-cs0837"></a><span data-ttu-id="3870f-102">编译器错误 CS0837</span><span class="sxs-lookup"><span data-stu-id="3870f-102">Compiler Error CS0837</span></span>
<span data-ttu-id="3870f-103">“is”或“as”运算符的第一个操作数不能是 lambda 表达式或匿名方法。</span><span class="sxs-lookup"><span data-stu-id="3870f-103">The first operand of an "is" or "as" operator may not be a lambda expression or anonymous method.</span></span>  
  
 <span data-ttu-id="3870f-104">Lambda 表达式和匿名方法不能在 [is](../../csharp/language-reference/keywords/is.md) 或 [as](../../csharp/language-reference/keywords/as.md)的左侧使用。</span><span class="sxs-lookup"><span data-stu-id="3870f-104">Lambda expressions and anonymous methods may not be used on the left side of [is](../../csharp/language-reference/keywords/is.md) or [as](../../csharp/language-reference/keywords/as.md).</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="3870f-105">更正此错误</span><span class="sxs-lookup"><span data-stu-id="3870f-105">To correct this error</span></span>  
  
-   <span data-ttu-id="3870f-106">如果错误涉及 `is` 运算符，请记住， `is` 会采用一个值和一种类型，并会告诉你是否可以通过引用、装箱或取消装箱转换将该值转换为该类型。</span><span class="sxs-lookup"><span data-stu-id="3870f-106">If the error involves the `is` operator, remember that `is` takes a value and a type and tells you whether the value can be made into that type by a reference, boxing, or unboxing conversion.</span></span> <span data-ttu-id="3870f-107">由于 lambda 不是值并且没有引用、装箱或取消装箱转换，因此 lambda 不适用于 `is`。</span><span class="sxs-lookup"><span data-stu-id="3870f-107">Because lambdas are not values and have no reference, boxing, or unboxing conversions, lambdas are not candidates for `is`.</span></span>  
  
-   <span data-ttu-id="3870f-108">如果代码误用了 `as`，则更正方法可能是将它更改为强制转换。</span><span class="sxs-lookup"><span data-stu-id="3870f-108">If the code misuses `as`, the correction is probably to change it to a cast.</span></span>  
  
## <a name="example"></a><span data-ttu-id="3870f-109">示例</span><span class="sxs-lookup"><span data-stu-id="3870f-109">Example</span></span>  
 <span data-ttu-id="3870f-110">下面的示例生成 CS0837：</span><span class="sxs-lookup"><span data-stu-id="3870f-110">The following example generates CS0837:</span></span>  
  
```csharp  
// cs0837.cs  
namespace TestNamespace  
{  
    public delegate void Del();  
  
    class Test  
    {  
        static int Main()  
        {  
            bool b1 = (() => { }) is Del;   // CS0837  
            bool b2 = delegate() { } is Del;// CS0837  
            Del d1 = () => { } as Del;      // CS0837  
            Del d2 = delegate() { } as Del; // CS0837  
            return 1;  
        }  
    }  
}  
```
