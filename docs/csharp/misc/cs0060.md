# <a name="compiler-error-cs0060"></a>编译器错误 CS0060
可访问性不一致: 基类“class1”的可访问性低于类“class2”  
  
 基类和继承类之间的类可访问性应保持一致。  
  
 下面的示例生成 CS0060：  
  
```csharp  
// CS0060.cs  
class MyClass  
// try the following line instead  
// public class MyClass  
{  
}  
  
public class MyClass2 : MyClass   // CS0060  
{  
   public static void Main()  
   {  
   }  
}  
```
