# <a name="compiler-error-cs0575"></a>编译器错误 CS0575
只有类类型才能包含析构函数  
  
 一个 [结构](../language-reference/builtin-types/struct.md) 不能包含析构函数。  
  
 以下示例生成 CS0575：  
  
```csharp  
// CS0575.cs  
namespace x  
{  
   public struct iii  
   {  
      ~iii()   // CS0575  
      {  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
