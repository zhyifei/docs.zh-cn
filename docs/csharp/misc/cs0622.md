# <a name="compiler-error-cs0622"></a>编译器错误 CS0622
只能使用数组初始值设定项表达式为数组类型赋值。 请尝试改用 new 表达式。  
  
 非数组声明中使用的是适用于初始化数组的语法。  
  
## <a name="example"></a>示例  
 下面的示例生成 CS0622：  
  
```  
// CS0622.cs  
using System;  
  
public class Test  
{  
    public static void Main ()  
    {  
        Test t = { new Test() };   // CS0622  
        // Try the following instead:  
        // Test[] t = { new Test() };  
    }  
}  
```
