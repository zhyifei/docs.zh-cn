## <a name="example"></a><span data-ttu-id="e628f-109">示例</span><span class="sxs-lookup"><span data-stu-id="e628f-109">Example</span></span>  
 <span data-ttu-id="e628f-110">下面的示例生成 CS0755：</span><span class="sxs-lookup"><span data-stu-id="e628f-110">The following example generates CS0755:</span></span>  
  
```csharp  
// cs0755.cs  
    public static partial class Ext  
    {  
        static partial void Part(this C c); //Extension method  
  
        // Typically the implementing declaration is in a separate file.  
        static partial void Part(C c) //CS0755  
        {  
        }  
    }

    {  
        public static int Main()  
        {  
            return 1;  
        }  
    }  
```
