# <a name="compiler-error-cs0619"></a><span data-ttu-id="6656d-102">编译器错误 CS0619</span><span class="sxs-lookup"><span data-stu-id="6656d-102">Compiler Error CS0619</span></span>
<span data-ttu-id="6656d-103">“member”已过时：“text”</span><span class="sxs-lookup"><span data-stu-id="6656d-103">'member' is obsolete: 'text'</span></span>  
  
 <span data-ttu-id="6656d-104">类成员标记有[过时](http://msdn.microsoft.com/library/05e99cd0-bda6-4f79-a890-1ca093b4b488)特性，这样，当引用类成员时，将发出错误。</span><span class="sxs-lookup"><span data-stu-id="6656d-104">A class member was marked with the [Obsolete](http://msdn.microsoft.com/library/05e99cd0-bda6-4f79-a890-1ca093b4b488) attribute, such that an error will be issued when the class member is referenced.</span></span>  
  
 <span data-ttu-id="6656d-105">以下示例生成 CS0619：</span><span class="sxs-lookup"><span data-stu-id="6656d-105">The following sample generates CS0619:</span></span>  
  
```  
// CS0619.cs  
using System;  
  
public class C  
{  
   [Obsolete("Use newMethod instead", true)]   // generates an error on use  
   public static void m()  
   {  
   }  
  
   // this is the method you should be using  
   public static void newMethod()  
   {  
   }  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
      C.m();   // CS0619  
   }  
}  
```
