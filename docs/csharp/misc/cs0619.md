# <a name="compiler-error-cs0619"></a><span data-ttu-id="8fb1f-102">编译器错误 CS0619</span><span class="sxs-lookup"><span data-stu-id="8fb1f-102">Compiler Error CS0619</span></span>
<span data-ttu-id="8fb1f-103">“member”已过时：“text”</span><span class="sxs-lookup"><span data-stu-id="8fb1f-103">'member' is obsolete: 'text'</span></span>  
  
 <span data-ttu-id="8fb1f-104">类成员标记与 [已过时] ((.../../ csharp/programming-guide/concepts/attributes/common-attributes.md#Obsolete) 特性，这样，当引用类成员时，将发出错误。</span><span class="sxs-lookup"><span data-stu-id="8fb1f-104">A class member was marked with the [Obsolete]((../../csharp/programming-guide/concepts/attributes/common-attributes.md#Obsolete) attribute, such that an error will be issued when the class member is referenced.</span></span>  
  
 <span data-ttu-id="8fb1f-105">以下示例生成 CS0619：</span><span class="sxs-lookup"><span data-stu-id="8fb1f-105">The following sample generates CS0619:</span></span>  
  
```csharp  
// CS0619.cs  
using System;  
  
public class C  
{  
   [Obsolete("Use newMethod instead", true)]   // generates an error on use  
   public static void m()  
   {  
   }  
  
   // this is the method you should be using  
   public static void newMethod()  
   {  
   }  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
      C.m();   // CS0619  
   }  
}  
```
