# <a name="compiler-error-cs1013"></a>编译器错误 CS1013
无效数字  
  
 编译器检测到一个格式不正确的数字。 有关整型的详细信息，请参阅 [Integral Types Table](../../csharp/language-reference/keywords/integral-types-table.md)。  
  
## <a name="example"></a>示例  
 下面的示例生成 CS1013：  
  
```csharp  
// CS1013.cs  
class Sample  
{  
    static void Main()  
    {  
        int i = 0x;   // CS1013  
    }  
}  
```
