# <a name="compiler-error-cs1003"></a>编译器错误 CS1003
语法错误，应为“char”  
  
 对于若干错误条件中的任何一项，编译器将生成此错误。 查看你的代码以查找语法错误。  
  
 以下示例生成 CS1003：  
  
```csharp  
// CS1003.cs  
public class b  
{  
   public static void Main()  
   {  
      int[] a;  
      a[);   // CS1003  
   }  
}  
```
