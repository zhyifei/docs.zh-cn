# <a name="compiler-error-cs1106"></a>编译器错误 CS1106
扩展方法必须在非泛型静态类中定义。  
  
 扩展方法必须在非泛型静态类中定义为静态方法。  
  
## <a name="example"></a>示例  
 下面的示例生成 CS1106，因为类 `Extensions` 未定义为 `static`：  
  
```csharp  
// cs1106.cs  
public class Extensions // CS1106  
{  
    public  static void Test<T>(this System.String s) {}  
}  
```
