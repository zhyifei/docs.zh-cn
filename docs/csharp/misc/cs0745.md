# <a name="compiler-error-cs0745"></a><span data-ttu-id="d5dec-102">编译器错误 CS0745</span><span class="sxs-lookup"><span data-stu-id="d5dec-102">Compiler Error CS0745</span></span>
<span data-ttu-id="d5dec-103">应为上下文关键字 "by"</span><span class="sxs-lookup"><span data-stu-id="d5dec-103">Expected contextual keyword 'by'</span></span>  
  
 <span data-ttu-id="d5dec-104">`group` 子句的模式是 `group...by` 跟一个可选 `into`，如以下示例所示：</span><span class="sxs-lookup"><span data-stu-id="d5dec-104">The pattern for the `group` clause is `group...by` followed by an optional `into`, as shown in the following example:</span></span>

```csharp  
var query = from name in names  
            group name by name[0];  
```

## <a name="to-correct-this-error"></a><span data-ttu-id="d5dec-106">更正此错误</span><span class="sxs-lookup"><span data-stu-id="d5dec-106">To correct this error</span></span>

## <a name="example"></a><span data-ttu-id="d5dec-108">示例</span><span class="sxs-lookup"><span data-stu-id="d5dec-108">Example</span></span>  
 <span data-ttu-id="d5dec-109">以下代码生成 CS0745：</span><span class="sxs-lookup"><span data-stu-id="d5dec-109">The following code generates CS0745:</span></span>  
  
```csharp  
// cs0745.cs  
using System;  
using System.Linq;  
  
public class C  
{  
    public static int Main()  
    {  
        string[] names = { "Bob", "Bill", "Jonetta", "Mary" };  
  
        var query = from name in names  
                    group name name[0]; // CS0745  
  
        return 1;  
    }  
}  
```
