# <a name="compiler-warning-level-2-cs0162"></a><span data-ttu-id="d05f1-102">编译器警告（等级 2）CS0162</span><span class="sxs-lookup"><span data-stu-id="d05f1-102">Compiler Warning (level 2) CS0162</span></span>
<span data-ttu-id="d05f1-103">检测到无法访问的代码</span><span class="sxs-lookup"><span data-stu-id="d05f1-103">Unreachable code detected</span></span>  
  
 <span data-ttu-id="d05f1-104">编译器检测到永远不执行的代码。</span><span class="sxs-lookup"><span data-stu-id="d05f1-104">The compiler detected code that will never be executed.</span></span>  
  
 <span data-ttu-id="d05f1-105">下面的示例生成 CS0162：</span><span class="sxs-lookup"><span data-stu-id="d05f1-105">The following sample generates CS0162:</span></span>  
  
```csharp  
// CS0162.cs  
// compile with: /W:2  
public class A  
{  
   public static void Main()  
   {  
      goto lab1;  
      {  
         // The following statements cannot be reached:  
         int i = 9;   // CS0162   
         i++;  
      }  
      lab1:  
      {  
      }  
   }  
}  
```
