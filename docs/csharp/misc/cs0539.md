# <a name="compiler-error-cs0539"></a>编译器错误 CS0539
显式接口声明中的“member”不是接口成员  
  
 尝试显式声明不存在的 [接口](../language-reference/keywords/interface.md) 成员。 应删除该声明或将其改为引用有效的接口成员。  
  
 下面的示例生成 CS0539：  
  
```csharp  
// CS0539.cs  
namespace x  
{  
   interface I  
   {  
      void m();  
   }  
  
   public class clx : I  
   {  
      void I.x()   // CS0539  
      {  
      }  
  
      public static int Main()  
      {  
         return 0;  
      }  
   }  
}  
```
