# <a name="compiler-error-cs0516"></a>编译器错误 CS0516
构造函数“constructor”不能调用其自身  
  
 程序不能递归调用构造函数。  
  
 下面的示例生成 CS0516：  
  
```csharp  
// CS0516.cs  
namespace x  
{  
   public class clx  
   {  
      public clx() : this()   // CS0516, delete "this()"  
      {  
      }  
  
      public static void Main()  
      {  
      }  
   }  
}  
```
