# <a name="compiler-error-cs1625"></a><span data-ttu-id="fb141-102">编译器错误 CS1625</span><span class="sxs-lookup"><span data-stu-id="fb141-102">Compiler Error CS1625</span></span>
<span data-ttu-id="fb141-103">无法在 finally 子句体中生成</span><span class="sxs-lookup"><span data-stu-id="fb141-103">Cannot yield in the body of a finally clause</span></span>  
  
 <span data-ttu-id="fb141-104">在 finally 子句体内不允许出现 yield 语句。</span><span class="sxs-lookup"><span data-stu-id="fb141-104">A yield statement is not allowed in the body of a finally clause.</span></span> <span data-ttu-id="fb141-105">若要避免此错误，请将 yield 语句移到 finally 子句的外部。</span><span class="sxs-lookup"><span data-stu-id="fb141-105">To avoid this error, move the yield statement out of the finally clause.</span></span>  
  
 <span data-ttu-id="fb141-106">下面的示例生成 CS1625：</span><span class="sxs-lookup"><span data-stu-id="fb141-106">The following sample generates CS1625:</span></span>  
  
```csharp  
// CS1625.cs  
using System.Collections;  
  
class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()  
   {  
      try  
      {  
      }  
      finally  
      {  
        yield return this;  // CS1625  
      }  
   }  
}  
  
public class CMain  
{  
   public static void Main() { }  
}  
```
