# <a name="compiler-error-cs0538"></a><span data-ttu-id="dfb45-102">编译器错误 CS0538</span><span class="sxs-lookup"><span data-stu-id="dfb45-102">Compiler Error CS0538</span></span>
<span data-ttu-id="dfb45-103">显式接口声明中的“name”不是接口</span><span class="sxs-lookup"><span data-stu-id="dfb45-103">'name' in explicit interface declaration is not an interface</span></span>  
  
 <span data-ttu-id="dfb45-104">尝试显式声明 [接口](../../csharp/language-reference/keywords/interface.md)，但未指定接口。</span><span class="sxs-lookup"><span data-stu-id="dfb45-104">An attempt was made to explicitly declare an [interface](../../csharp/language-reference/keywords/interface.md), but an interface was not specified.</span></span>  
  
 <span data-ttu-id="dfb45-105">下面的示例生成 CS0538：</span><span class="sxs-lookup"><span data-stu-id="dfb45-105">The following sample generates CS0538:</span></span>  
  
```csharp  
// CS0538.cs  
interface MyIFace  
{  
   void F();  
}  
  
public class MyClass  
{  
   public void G()  
   {  
   }  
}  
  
class C: MyIFace  
{  
   void MyIFace.F()  
   {  
   }  
  
   void MyClass.G()   // CS0538, MyClass not an interface  
   {  
   }  
}  
```
