# <a name="compiler-error-cs0538"></a>编译器错误 CS0538
显式接口声明中的“name”不是接口  
  
 尝试显式声明 [接口](../../csharp/language-reference/keywords/interface.md)，但未指定接口。  
  
 下面的示例生成 CS0538：  
  
```csharp  
// CS0538.cs  
interface MyIFace  
{  
   void F();  
}  
  
public class MyClass  
{  
   public void G()  
   {  
   }  
}  
  
class C: MyIFace  
{  
   void MyIFace.F()  
   {  
   }  
  
   void MyClass.G()   // CS0538, MyClass not an interface  
   {  
   }  
}  
```
