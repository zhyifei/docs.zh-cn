<span data-ttu-id="07f87-103">因为“member”不是重写，所以无法将其密封</span><span class="sxs-lookup"><span data-stu-id="07f87-103">'member' cannot be sealed because it is not an override</span></span>  
  
 <span data-ttu-id="07f87-104">对还未使用[密封](../../csharp/language-reference/keywords/sealed.md) 进行标记的成员使用了 [密封](../../csharp/language-reference/keywords/override.md)。</span><span class="sxs-lookup"><span data-stu-id="07f87-104">[sealed](../../csharp/language-reference/keywords/sealed.md) was used on a member that was not also marked with [override](../../csharp/language-reference/keywords/override.md).</span></span> <span data-ttu-id="07f87-105">有关详细信息，请参阅[继承](../../csharp/programming-guide/classes-and-structs/inheritance.md)。</span><span class="sxs-lookup"><span data-stu-id="07f87-105">For more information, see [Inheritance](../../csharp/programming-guide/classes-and-structs/inheritance.md).</span></span>  
  
 <span data-ttu-id="07f87-106">下面的示例生成 CS0238：</span><span class="sxs-lookup"><span data-stu-id="07f87-106">The following sample generates CS0238:</span></span>  
  
```  
// CS0238.cs  
abstract class MyClass  
{  
   public abstract void f();  
}  
  
class MyClass2 : MyClass  
{  
   public static void Main()  
   {  
   }  
  
   public sealed void f() // CS0238  
   // Try the following definition instead:  
   // public override sealed void f()  
   {  
   }  
}  
```
