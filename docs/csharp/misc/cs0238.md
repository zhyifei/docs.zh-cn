 <span data-ttu-id="6482a-104">对还未使用[密封](../../csharp/language-reference/keywords/sealed.md) 进行标记的成员使用了 [密封](../../csharp/language-reference/keywords/override.md)。</span><span class="sxs-lookup"><span data-stu-id="6482a-104">[sealed](../../csharp/language-reference/keywords/sealed.md) was used on a member that was not also marked with [override](../../csharp/language-reference/keywords/override.md).</span></span> <span data-ttu-id="6482a-105">有关详细信息，请参阅[继承](../../csharp/programming-guide/classes-and-structs/inheritance.md)。</span><span class="sxs-lookup"><span data-stu-id="6482a-105">For more information, see [Inheritance](../../csharp/programming-guide/classes-and-structs/inheritance.md).</span></span>  
  
 <span data-ttu-id="6482a-106">下面的示例生成 CS0238：</span><span class="sxs-lookup"><span data-stu-id="6482a-106">The following sample generates CS0238:</span></span>  
  
```csharp  
// CS0238.cs  
abstract class MyClass  
{  
   public abstract void f();  
}

{  
   public static void Main()  
   {  
   }  
  
   public sealed void f() // CS0238  
   // Try the following definition instead:  
   // public override sealed void f()  
   {  
   }  
}  
```
