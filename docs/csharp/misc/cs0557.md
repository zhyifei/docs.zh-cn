# <a name="compiler-error-cs0557"></a>编译器错误 CS0557
类型“class”中有重复的用户定义转换  
  
 类中不允许重复转换例程。  
  
 以下示例生成 CS0557：  
  
```csharp  
// CS0557.cs  
namespace x  
{  
    public class ii  
    {  
        public class iii  
        {  
        public static implicit operator int(iii aa)  
        {  
            return 0;  
        }  
  
    // CS0557, delete duplicate  
        public static explicit operator int(iii aa)  
        {  
            return 0;  
        }  
        }  
  
        public static void Main()  
        {  
        }  
    }  
}  
```
