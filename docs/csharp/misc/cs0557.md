# <a name="compiler-error-cs0557"></a><span data-ttu-id="1ab41-102">编译器错误 CS0557</span><span class="sxs-lookup"><span data-stu-id="1ab41-102">Compiler Error CS0557</span></span>
<span data-ttu-id="1ab41-103">类型“class”中有重复的用户定义转换</span><span class="sxs-lookup"><span data-stu-id="1ab41-103">Duplicate user-defined conversion in type 'class'</span></span>  
  
 <span data-ttu-id="1ab41-104">类中不允许重复转换例程。</span><span class="sxs-lookup"><span data-stu-id="1ab41-104">Duplicate conversion routines are not allowed in a class.</span></span>  
  
 <span data-ttu-id="1ab41-105">以下示例生成 CS0557：</span><span class="sxs-lookup"><span data-stu-id="1ab41-105">The following example generates CS0557:</span></span>  
  
```csharp  
// CS0557.cs  
namespace x  
{  
    public class ii  
    {  
        public class iii  
        {  
        public static implicit operator int(iii aa)  
        {  
            return 0;  
        }  
  
    // CS0557, delete duplicate  
        public static explicit operator int(iii aa)  
        {  
            return 0;  
        }  
        }  
  
        public static void Main()  
        {  
        }  
    }  
}  
```
