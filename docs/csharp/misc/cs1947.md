# <a name="compiler-error-cs1947"></a><span data-ttu-id="bee15-102">编译器错误 CS1947</span><span class="sxs-lookup"><span data-stu-id="bee15-102">Compiler Error CS1947</span></span>
<span data-ttu-id="bee15-103">无法对范围变量“变量名称”赋值 -- 它是只读的。</span><span class="sxs-lookup"><span data-stu-id="bee15-103">Range variable 'variable name' cannot be assigned to -- it is read only.</span></span>  
  
 <span data-ttu-id="bee15-104">在 `foreach` 语句中，范围变量类似于迭代变量。</span><span class="sxs-lookup"><span data-stu-id="bee15-104">A range variable is like an iteration variable in a `foreach` statement.</span></span> <span data-ttu-id="bee15-105">查询表达式中不能对其赋值。</span><span class="sxs-lookup"><span data-stu-id="bee15-105">It cannot be assigned to in a query expression.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="bee15-106">更正此错误</span><span class="sxs-lookup"><span data-stu-id="bee15-106">To correct this error</span></span>

## <a name="example"></a><span data-ttu-id="bee15-109">示例</span><span class="sxs-lookup"><span data-stu-id="bee15-109">Example</span></span>  
 <span data-ttu-id="bee15-110">以下代码生成 CS1947：</span><span class="sxs-lookup"><span data-stu-id="bee15-110">The following code generates CS1947:</span></span>  
  
```csharp  
// cs1947.cs  
using System.Linq;  
class Test  
{  
    static void Main()  
    {  
        int[] array = new int[] { 1, 2, 3, 4, 5 };  
        var x = from i in array  
                let k = i  
                select i = 5; // CS1947  
        x.ToList();  
    }  
}  
```
