# <a name="compiler-warning-level-3-cs0661"></a><span data-ttu-id="22b63-102">编译器警告（等级 3）CS0661</span><span class="sxs-lookup"><span data-stu-id="22b63-102">Compiler Warning (level 3) CS0661</span></span>
<span data-ttu-id="22b63-103">“class”定义运算符 == 或运算符 !=，但不重写 Object.GetHashCode()</span><span class="sxs-lookup"><span data-stu-id="22b63-103">'class' defines operator == or operator != but does not override Object.GetHashCode()</span></span>  
  
 <span data-ttu-id="22b63-104">编译器检测到用户定义的相等运算符或不相等运算符，但没有对 **GetHashCode** 函数的重写。</span><span class="sxs-lookup"><span data-stu-id="22b63-104">The compiler detected the user-defined equality or inequality operator, but no override for the **GetHashCode** function.</span></span> <span data-ttu-id="22b63-105">用户定义的相等或不相等运算符意味着你也想要重写 **GetHashCode** 函数。</span><span class="sxs-lookup"><span data-stu-id="22b63-105">A user-defined equality or inequality operator implies that you also want to override the **GetHashCode** function.</span></span>  
  
 <span data-ttu-id="22b63-106">以下示例生成 CS0661：</span><span class="sxs-lookup"><span data-stu-id="22b63-106">The following sample generates CS0661:</span></span>  
  
```csharp  
// CS0661.cs  
// compile with: /W:3  
class Test   // CS0661  
{  
   public static bool operator == (object o, Test t)  
   {  
      return true;  
   }  
  
   public static bool operator != (object o, Test t)  
   {  
      return true;  
   }  
  
   public override bool Equals(object o)  
   {  
      return true;  
   }  
  
   // uncomment the GetHashCode function to resolve  
   // public override int GetHashCode()  
   // {  
   //    return 0;  
   // }  
  
   public static void Main()  
   {  
   }  
}  
```
