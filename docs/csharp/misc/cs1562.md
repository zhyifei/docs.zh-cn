# <a name="compiler-error-cs1562"></a>编译器错误 CS1562
必须为没有源的输出指定 /out 选项  
  
 编译可以创建输出文件，但没有源代码文件可以作为可从中默示输出文件名的输入。 例如，你可能尝试编译仅元数据文件或仅资源文件。  
  
 使用 [/out](../../csharp/language-reference/compiler-options/out-compiler-option.md) 编译器选项指定输出文件的名称。
