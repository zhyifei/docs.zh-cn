# <a name="compiler-error-cs1660"></a><span data-ttu-id="c5af0-102">编译器错误 CS1660</span><span class="sxs-lookup"><span data-stu-id="c5af0-102">Compiler Error CS1660</span></span>
<span data-ttu-id="c5af0-103">不能将匿名方法块转换为类型“type”，因为它不是委托类型</span><span class="sxs-lookup"><span data-stu-id="c5af0-103">Cannot convert anonymous method block to type 'type' because it is not a delegate type</span></span>  
  
 <span data-ttu-id="c5af0-104">如果你尝试赋值或以其他方式将匿名方法块转换为不是委托类型的类型，则将发生此错误。</span><span class="sxs-lookup"><span data-stu-id="c5af0-104">This error occurs if you try to assign or otherwise convert an anonymous method block to a type which is not a delegate type.</span></span>  
  
 <span data-ttu-id="c5af0-105">下面的示例生成 CS1660：</span><span class="sxs-lookup"><span data-stu-id="c5af0-105">The following sample generates CS1660:</span></span>  
  
```csharp  
// CS1660.cs  
delegate int MyDelegate();  
class C {  
   static void Main()  
   {  
     int i = delegate { return 1; };  // CS1660  
     // Try this instead:  
     // MyDelegate myDelegate = delegate { return 1; };  
     // int i = myDelegate();  
   }  
}  
```
