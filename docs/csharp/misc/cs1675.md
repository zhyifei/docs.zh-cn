# <a name="compiler-error-cs1675"></a>编译器错误 CS1675
枚举不能有类型参数  
  
 若要解决此错误，请删除 `enum` 声明中的类型形参。  
  
## <a name="example"></a>示例  
 下面的示例生成 CS1675：  
  
```csharp  
// CS1675.cs  
enum E<T>  // CS1675  
{  
}  
  
class CMain  
{  
    public static void Main()  
    {  
    }  
}  
```
