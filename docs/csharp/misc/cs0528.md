# <a name="compiler-error-cs0528"></a>编译器错误 CS0528
“interface”已经在接口列表中列出  
  
 接口继承列表包含重复。 [接口](../../csharp/language-reference/keywords/interface.md) 只能在继承列表中指定一次。  
  
 下面的示例生成 CS0528：  
  
```  
// CS0528.cs  
namespace x  
{  
   public interface a  
   {  
   }  
  
   public class b : a, a   // CS0528  
   {  
      public void Main()  
      {  
      }  
   }  
}  
```
