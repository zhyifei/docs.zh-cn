## <a name="example"></a><span data-ttu-id="96d56-106">示例</span><span class="sxs-lookup"><span data-stu-id="96d56-106">Example</span></span>  
 <span data-ttu-id="96d56-107">下面的示例生成 CS3002：</span><span class="sxs-lookup"><span data-stu-id="96d56-107">The following example generates CS3002:</span></span>  
  
```  
// CS3002.cs  
  
[assembly:System.CLSCompliant(true)]  
public class a  
{  
    public ushort bad()   // CS3002, public method  
    {  
        ushort a;  
        a = ushort.MaxValue;  
        return a;  
    }  
  
    private ushort OK()   // OK, private method  
    {  
        ushort a;  
        a = ushort.MaxValue;  
        return a;  
    }  
  
    public static void Main()  
    {  
    }  
}  
```
