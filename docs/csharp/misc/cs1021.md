# <a name="compiler-error-cs1021"></a>编译器错误 CS1021
整数常量太大  
  
 分配为 [整型](../../csharp/language-reference/keywords/integral-types-table.md) 的值超出了可能的最大无符号整数值。  
  
 下面的示例生成 CS1021:  
  
```csharp  
// CS1021.cs  
enum F : int  
{  
   a=(int)2590000000000000000000   // CS1021  
}  
  
public class clx  
{  
   public static void Main()  
   {  
   }  
}  
```
