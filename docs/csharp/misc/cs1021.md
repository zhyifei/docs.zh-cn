# <a name="compiler-error-cs1021"></a>编译器错误 CS1021

整数常量太大  
  
由整数文本表示的值大于<xref:System.UInt64.MaxValue?displayProperty=nameWithType>。  
  
下面的示例生成 CS1021:  

```csharp
// CS1021.cs  
class Program
{
    static void Main(string[] args)
    {
        int a = 18_446_744_073_709_552_000;
    }
}  
```

以下代码还生成 CS1021：

```csharp
using System.Numerics;

class Program
{
    static void Main(string[] args)
    {
        var a = new BigInteger(18_446_744_073_709_552_000);
    }
}
```

有关如何实例化其值超过内置数值类型范围<xref:System.Numerics.BigInteger?displayProperty=nameWithType>的实例的信息，请参阅<xref:System.Numerics.BigInteger>引用页中的"实例化[对象](https://docs.microsoft.com/dotnet/api/System.Numerics.BigInteger#instantiating-a-biginteger-object)"部分。
