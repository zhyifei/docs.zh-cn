# <a name="compiler-error-cs2011"></a>编译器错误 CS2011
无法打开响应文件“file”  
  
 已在编译中指定响应文件，但编译器无法找到和打开此文件。  
  
 有关响应文件的详细信息，请参阅 [@（指定响应文件）](../language-reference/compiler-options/response-file-compiler-option.md)。
