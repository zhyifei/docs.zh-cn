# <a name="compiler-error-cs1517"></a>编译器错误 CS1517
无效的预处理器表达式  
  
 编译器遇到了无效的预处理器表达式。  
  
 有关详细信息，请参阅 [预处理器指令](../../csharp/language-reference/preprocessor-directives/index.md)。  
  
 下面的示例显示了一些有效和无效的预处理器表达式：  
  
```csharp  
// CS1517.cs  
#if symbol      // OK  
#endif  
#if !symbol     // OK  
#endif  
#if (symbol)    // OK  
#endif  
#if true        // OK  
#endif  
#if false       // OK  
#endif  
#if 1           // CS1517  
#endif  
#if ~symbol     // CS1517  
#endif  
#if *           // CS1517  
#endif  
  
class x  
{  
   public static void Main()  
   {  
   }  
}  
```
