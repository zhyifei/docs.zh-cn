# <a name="compiler-error-cs0432"></a><span data-ttu-id="a1a12-102">编译器错误 CS0432</span><span class="sxs-lookup"><span data-stu-id="a1a12-102">Compiler Error CS0432</span></span>
<span data-ttu-id="a1a12-103">未找到别名“identifier”</span><span class="sxs-lookup"><span data-stu-id="a1a12-103">Alias 'identifier' not found</span></span>  
  
 <span data-ttu-id="a1a12-104">在不是别名的标识符的右侧使用“::”时将发生此错误。</span><span class="sxs-lookup"><span data-stu-id="a1a12-104">This error occurs when you use "::" to the right of an identifier that is not an alias.</span></span> <span data-ttu-id="a1a12-105">若要解决此错误，请改用“.”。</span><span class="sxs-lookup"><span data-stu-id="a1a12-105">To resolve the error, use "." instead.</span></span>  
  
 <span data-ttu-id="a1a12-106">下面的示例生成 CS0432：</span><span class="sxs-lookup"><span data-stu-id="a1a12-106">The following example generates CS0432:</span></span>  
  
```csharp  
// CS0432.cs  
namespace A {  
    public class B {  
        public static void Meth() { }  
    }  
}  
  
public class Test  
{  
    public static void Main()  
    {  
        A::B.Meth();   // CS0432  
       // To resolve, use the following line instead:  
       // A.B.Meth();  
    }  
}  
```
