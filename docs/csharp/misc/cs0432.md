# <a name="compiler-error-cs0432"></a>编译器错误 CS0432
未找到别名“identifier”  
  
 在不是别名的标识符的右侧使用“::”时将发生此错误。 若要解决此错误，请改用“.”。  
  
 下面的示例生成 CS0432：  
  
```  
// CS0432.cs  
namespace A {  
    public class B {  
        public static void Meth() { }  
    }  
}  
  
public class Test  
{  
    public static void Main()  
    {  
        A::B.Meth();   // CS0432  
       // To resolve, use the following line instead:  
       // A.B.Meth();  
    }  
}  
```
