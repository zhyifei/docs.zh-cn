<span data-ttu-id="abef0-103">可能是无意的引用比较；若要获取值比较，请将右边转换为类型“type”</span><span class="sxs-lookup"><span data-stu-id="abef0-103">Possible unintended reference comparison; to get a value comparison, cast the right hand side to type 'type'</span></span>  
  
 <span data-ttu-id="abef0-104">编译器正在执行引用比较。</span><span class="sxs-lookup"><span data-stu-id="abef0-104">The compiler is doing a reference comparison.</span></span> <span data-ttu-id="abef0-105">如果你想比较字符串的值，请将表达式右侧转换为 `type`。</span><span class="sxs-lookup"><span data-stu-id="abef0-105">If you want to compare the value of strings, cast the right side of the expression to `type`.</span></span>  
  
 <span data-ttu-id="abef0-106">下面的示例生成 CS0253：</span><span class="sxs-lookup"><span data-stu-id="abef0-106">The following sample generates CS0253:</span></span>  
  
```  
// CS0253.cs  
// compile with: /W:2  
using System;  
class MyClass  
{  
   public static void Main()  
   {  
      string s = "11";  
      object o = s + s;  
  
      bool c = s == o;   // CS0253  
      // try the following line instead  
      // bool c = s == (string)o;  
   }  
}  
```
