<span data-ttu-id="317cf-103">在控件返回调用方之前，自动实现的属性“name”的支持字段必须完全赋值。</span><span class="sxs-lookup"><span data-stu-id="317cf-103">Backing field for automatically implemented property 'name' must be fully assigned before control is returned to the caller.</span></span> <span data-ttu-id="317cf-104">请考虑从构造函数初始值设定项调用无参数构造函数。</span><span class="sxs-lookup"><span data-stu-id="317cf-104">Consider calling the parameterless constructor from a constructor initializer.</span></span>

 <span data-ttu-id="317cf-107">以下示例生成 CS0171：</span><span class="sxs-lookup"><span data-stu-id="317cf-107">The following sample generates CS0171:</span></span>  
  
```csharp  
// CS0171.cs  
struct MyStruct  
{  
   MyStruct(int initField)   // CS0171  
   {  
      // i = initField;      // uncomment this line to resolve this error  
   }  
   public int i;  
}  
  
class MyClass  
{  
   public static void Main()  
   {  
      MyStruct aStruct = new MyStruct();  
   }  
}  
```
