# <a name="compiler-error-cs1594"></a><span data-ttu-id="9d568-102">编译器错误 CS1594</span><span class="sxs-lookup"><span data-stu-id="9d568-102">Compiler Error CS1594</span></span>
<span data-ttu-id="9d568-103">委托“delegate”有一些无效参数</span><span class="sxs-lookup"><span data-stu-id="9d568-103">Delegate 'delegate' has some invalid arguments</span></span>  
  
 <span data-ttu-id="9d568-104">传递给 [委托](../../csharp/language-reference/keywords/delegate.md) 调用的实参类型与委托声明中的形参类型不符。</span><span class="sxs-lookup"><span data-stu-id="9d568-104">The type of an argument passed to a [delegate](../../csharp/language-reference/keywords/delegate.md) invocation does not agree with the type of the parameter in the delegate declaration.</span></span>  
  
 <span data-ttu-id="9d568-105">以下示例生成 CS1594：</span><span class="sxs-lookup"><span data-stu-id="9d568-105">The following sample generates CS1594:</span></span>  
  
```  
// CS1594.cs  
using System;  
delegate string func(int i);   // declare delegate  
  
class a  
{  
   public static void Main()  
   {  
      func dt = new func(z);  
      x(dt);  
   }  
  
   public static string z(int j)  
   {  
      Console.WriteLine(j);  
      return j.ToString();  
   }  
  
   public static void x(func hello)  
   {  
      hello("8");   // CS1594  
      // try the following line instead  
      // hello(8);  
   }  
}  
```
