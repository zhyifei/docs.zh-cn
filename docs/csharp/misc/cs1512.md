 <span data-ttu-id="c91f7-104">在方法、属性或构造函数外部使用了 [base](../../csharp/language-reference/keywords/base.md) 关键字。</span><span class="sxs-lookup"><span data-stu-id="c91f7-104">The [base](../../csharp/language-reference/keywords/base.md) keyword was used outside of a method, property, or constructor.</span></span>

```csharp

{  
    private String xx = base.ToString();   // CS1512  
    // Try putting this initialization in the constructor instead:  
    // public CMyClass()  
    // {  
    //    xx = base.ToString();  
    // }  
  
    public static void Main()  
    {  
        CMyClass z = new CMyClass();  
    }  
}  
```
