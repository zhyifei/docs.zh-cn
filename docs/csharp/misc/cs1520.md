# <a name="compiler-error-cs1520"></a><span data-ttu-id="1587b-102">编译器错误 CS1520</span><span class="sxs-lookup"><span data-stu-id="1587b-102">Compiler Error CS1520</span></span>
<span data-ttu-id="1587b-103">方法必须具有返回类型</span><span class="sxs-lookup"><span data-stu-id="1587b-103">Method must have a return type</span></span>  
  
 <span data-ttu-id="1587b-104">在类、结构或接口中声明的方法必须具有显式返回类型。</span><span class="sxs-lookup"><span data-stu-id="1587b-104">A method that is declared in a class, struct, or interface must have an explicit return type.</span></span> <span data-ttu-id="1587b-105">在下面的示例中，Square 方法具有 [字符串](../../csharp/language-reference/keywords/string.md)返回值：</span><span class="sxs-lookup"><span data-stu-id="1587b-105">In the following example, the Square method has a return value of [string](../../csharp/language-reference/keywords/string.md):</span></span>  
  
```csharp  
class Test  
{  
    string IntToString(int i)  
    {  
        return i.ToString();  
    }  
}  
```  
  
 <span data-ttu-id="1587b-106">以下示例生成 CS1520：</span><span class="sxs-lookup"><span data-stu-id="1587b-106">The following sample generates CS1520:</span></span>  
  
```csharp  
// CS1520a.cs  
public class x  
{  
   // Method declaration missing a return type  
   MyMethod()   // CS1520     
   {}  
   // Add the desired return type:  
   // void MyMethod2()  
  // { }  
  
   public static void Main()  
   {  
   }  
}  
```  
  
 <span data-ttu-id="1587b-107">或者，当构造函数名称的大小写与类或结构声明不同时，则可能遇到此错误，如以下示例所示。</span><span class="sxs-lookup"><span data-stu-id="1587b-107">Alternatively, this error might be encountered when the case of a constructor's name differs from that of the class or struct declaration, as in the following sample.</span></span> <span data-ttu-id="1587b-108">由于该名称与类名称不完全相同，因此编译器将其解释为常规方法而不是构造函数，从而产生此错误：</span><span class="sxs-lookup"><span data-stu-id="1587b-108">Because the name is not exactly the same as the class name, the compiler interprets it as a regular method, not a constructor, and produces the error:</span></span>  
  
```csharp  
// CS1520b.cs  
public class Class1  
{  
   // Should be Class1, not class1  
   public class1()   // CS1520  
   {  
   }  
   static void Main()  
   {  
   }  
}  
```
