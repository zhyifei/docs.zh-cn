 <span data-ttu-id="0848d-107">要避免此错误，找到该标识符的声明，验证拼写是否正确，如果该声明位于单独的程序集中，请确保你具有适当的程序集引用。</span><span class="sxs-lookup"><span data-stu-id="0848d-107">To avoid this error, locate the declaration of the identifier, verify the correct spelling, and if the declaration is in a separate assembly, make sure that you have the appropriate assembly reference.</span></span> <span data-ttu-id="0848d-108">如果该标识符在另一个类型或命名空间内进行了声明，则在 :: 之后使用完全限定的名称。</span><span class="sxs-lookup"><span data-stu-id="0848d-108">If the identifier is declared inside another type or namespace, use the fully-qualified name after the ::.</span></span> <span data-ttu-id="0848d-109">以下示例生成 CS0400：</span><span class="sxs-lookup"><span data-stu-id="0848d-109">The following sample generates CS0400:</span></span>
