## <a name="value-equality"></a><span data-ttu-id="0517f-120">值相等性</span><span class="sxs-lookup"><span data-stu-id="0517f-120">Value Equality</span></span>  
 <span data-ttu-id="0517f-121">值相等性指两个对象包含相同的一个或多个值。</span><span class="sxs-lookup"><span data-stu-id="0517f-121">Value equality means that two objects contain the same value or values.</span></span> <span data-ttu-id="0517f-122">对于基元值类型（例如 [int](../../../csharp/language-reference/keywords/int.md) 或 [bool](../../../csharp/language-reference/keywords/bool.md)），针对值相等性的测试简单明了。</span><span class="sxs-lookup"><span data-stu-id="0517f-122">For primitive value types such as [int](../../../csharp/language-reference/keywords/int.md) or [bool](../../../csharp/language-reference/keywords/bool.md), tests for value equality are straightforward.</span></span> <span data-ttu-id="0517f-123">可以使用 [==](../../../csharp/language-reference/operators/equality-comparison-operator.md) 运算符，如下面的示例所示。</span><span class="sxs-lookup"><span data-stu-id="0517f-123">You can use the [==](../../../csharp/language-reference/operators/equality-comparison-operator.md) operator, as shown in the following example.</span></span>  
  
```csharp  
int a = GetOriginalValue();  
int b = GetCurrentValue();  
  
// Test for value equality.   
if( b == a)   
{  
    // The two integers are equal.  
}  
```  
  
 <span data-ttu-id="0517f-124">对于大多数其他类型，针对值相等性的测试较为复杂，因为它需要用户了解类型对值相等性的定义方式。</span><span class="sxs-lookup"><span data-stu-id="0517f-124">For most other types, testing for value equality is more complex because it requires that you understand how the type defines it.</span></span> <span data-ttu-id="0517f-125">对于具有多个字段或属性的类和结构，值相等性的定义通常指所有字段或属性都具有相同的值。</span><span class="sxs-lookup"><span data-stu-id="0517f-125">For classes and structs that have multiple fields or properties, value equality is often defined to mean that all fields or properties have the same value.</span></span> <span data-ttu-id="0517f-126">例如，如果 pointA.X 等于 pointB.X，并且 pointA.Y 等于 pointB.Y，则可以将两个 `Point` 对象定义为相等。</span><span class="sxs-lookup"><span data-stu-id="0517f-126">For example, two `Point` objects might be defined to be equivalent if pointA.X is equal to pointB.X and pointA.Y is equal to pointB.Y.</span></span>  
  
 <span data-ttu-id="0517f-127">但是，并不要求类型中的所有字段均相等。</span><span class="sxs-lookup"><span data-stu-id="0517f-127">However, there is no requirement that equivalence be based on all the fields in a type.</span></span> <span data-ttu-id="0517f-128">只需子集相等即可。</span><span class="sxs-lookup"><span data-stu-id="0517f-128">It can be based on a subset.</span></span> <span data-ttu-id="0517f-129">比较不具所有权的类型时，应确保明确了解相等性对于该类型是如何定义的。</span><span class="sxs-lookup"><span data-stu-id="0517f-129">When you compare types that you do not own, you should make sure to understand specifically how equivalence is defined for that type.</span></span> <span data-ttu-id="0517f-130">若要详细了解如何在自己的类和结构中定义值相等性，请参阅[如何：为类型定义值相等性](../../../csharp/programming-guide/statements-expressions-operators/how-to-define-value-equality-for-a-type.md)。</span><span class="sxs-lookup"><span data-stu-id="0517f-130">For more information about how to define value equality in your own classes and structs, see [How to: Define Value Equality for a Type](../../../csharp/programming-guide/statements-expressions-operators/how-to-define-value-equality-for-a-type.md).</span></span>
