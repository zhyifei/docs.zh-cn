# <a name="serializing-xml-trees-c"></a>序列化 XML 树 (C#)
序列化 XML 树意味着从 XML 树生成 XML。 可以将 XML 树序列化到文件、<xref:System.IO.TextWriter> 类的具体实现或 <xref:System.Xml.XmlWriter> 的具体实现。  
  
 可以控制序列化的各个方面。 例如，可以控制是否缩进序列化的 XML，以及是否写入 XML 声明。
