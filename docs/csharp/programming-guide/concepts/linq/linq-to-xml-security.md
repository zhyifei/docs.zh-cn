### <a name="limit-the-depth-of-the-xml-hierarchy"></a><span data-ttu-id="a2fc5-170">限制 XML 层次结构的深度</span><span class="sxs-lookup"><span data-stu-id="a2fc5-170">Limit the Depth of the XML Hierarchy</span></span>

### <a name="protect-against-untrusted-xmlreader-or-xmlwriter-implementations"></a><span data-ttu-id="a2fc5-174">防止不受信任的 XmlReader 或 XmlWriter 实现</span><span class="sxs-lookup"><span data-stu-id="a2fc5-174">Protect Against Untrusted XmlReader or XmlWriter Implementations</span></span>

### <a name="annotations-are-accessible-by-software-components-that-share-a-linq-to-xml-tree"></a><span data-ttu-id="a2fc5-183">通过共享 LINQ to XML 树的软件组件可以访问批注</span><span class="sxs-lookup"><span data-stu-id="a2fc5-183">Annotations Are Accessible by Software Components that Share a LINQ to XML Tree</span></span>
