# <a name="-nologo-c-compiler-options"></a>-nologo（C# 编译器选项）
-nologo 选项可在编译器启动时取消显式登录版权标志并在编译期间取消显示信息性消息。  
  
## <a name="syntax"></a>语法  
  
```console  
-nologo  
```  
  
## <a name="remarks"></a>备注  
 此选项不适用于开发环境内，仅当从命令行编译时可用。  
  
 此编译器选项在 Visual Studio 中不可用，并且无法以编程方式更改。
