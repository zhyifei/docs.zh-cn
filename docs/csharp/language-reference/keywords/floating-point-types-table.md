# <a name="floating-point-types-table-c-reference"></a>浮点类型表（C# 参考）

下表显示了浮点类型的精度和大致范围。  
  
|类型|大致范围|精度|  
|----------|-----------------------|---------------|  
|[float](float.md)|±1.5 x 10<sup>−45</sup> 至 ±3.4 x 10<sup>38</sup>|7 位|  
|[double](double.md)|±5.0 × 10<sup>−324</sup> 到 ±1.7 × 10<sup>308</sup>|15-16 位|  
|[decimal](decimal.md)|±1.0 x 10<sup>-28</sup> 至 ±7.9228 x 10<sup>28</sup>|28-29 位|
