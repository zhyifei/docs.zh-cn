# <a name="out-c-reference"></a>out（C# 参考）

可以在以下两种上下文中使用 `out` 关键字：

- 作为[参数修饰符](out-parameter-modifier.md)，它允许按引用而不是按值向方法传递参数。

- 接口和委托的[泛型类型参数声明](out-generic-modifier.md)中，该声明指定类型参数为协变。
