# <a name="compiler-error-cs0310"></a><span data-ttu-id="44232-102">编译器错误 CS0310</span><span class="sxs-lookup"><span data-stu-id="44232-102">Compiler Error CS0310</span></span>
<span data-ttu-id="44232-103">类型“typename”必须是具有公共的无参数构造函数的非抽象类型，才能用作泛型类型或方法“generic”中的参数“parameter”</span><span class="sxs-lookup"><span data-stu-id="44232-103">The type 'typename' must be a non-abstract type with a public parameterless constructor in order to use it as parameter 'parameter' in the generic type or method 'generic'</span></span>  
  
 <span data-ttu-id="44232-104">泛型类型或方法在其 where 子句中定义了一个新约束，因此任何类型必须具有公共的无参数构造函数，才能用作该泛型类型或方法的类型参数。</span><span class="sxs-lookup"><span data-stu-id="44232-104">The generic type or method defines a new constraint in its where clause, so any type must have a public parameterless constructor in order to be used as a type argument for that generic type or method.</span></span> <span data-ttu-id="44232-105">若要避免此错误，请确保类型具有正确的构造函数，或修改泛型类型或方法的约束子句。</span><span class="sxs-lookup"><span data-stu-id="44232-105">To avoid this error, make sure that the type has the correct constructor, or modify the constraint clause of the generic type or method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="44232-106">示例</span><span class="sxs-lookup"><span data-stu-id="44232-106">Example</span></span>  
 <span data-ttu-id="44232-107">下面的示例生成 CS0310：</span><span class="sxs-lookup"><span data-stu-id="44232-107">The following sample generates CS0310:</span></span>  
  
```csharp  
// CS0310.cs  
using System;  
  
class G<T> where T : new()  
{  
    T t;  
  
    public G()  
    {  
        t = new T();  
        Console.WriteLine(t);  
    }  
}  
  
class B  
{  
    private B() { }  
    // Try this instead:  
    // public B() { }  
}  
  
class CMain  
{  
    public static void Main()  
    {  
        G<B> g = new G<B>();   // CS0310  
        Console.WriteLine(g.ToString());  
    }  
}  
```
