# <a name="compiler-error-cs0616"></a>编译器错误 CS0616
“class”不是特性类  
  
 尝试了在属性块中使用非特性类。 所有属性类型都必须从 <xref:System.Attribute?displayProperty=fullName> 继承。  
  
## <a name="example"></a>示例  
 下面的示例生成 CS0616。  
  
```csharp  
// CS0616.cs  
// compile with: /target:library  
[CMyClass(i = 5)]   // CS0616  
public class CMyClass {}  
```  
  
## <a name="example"></a>示例  
 下面的示例演示可以如何定义属性：  
  
```csharp  
// CreateAttrib.cs  
// compile with: /target:library  
using System;  
  
[AttributeUsage(AttributeTargets.Class|AttributeTargets.Interface)]  
public class MyAttr : Attribute  
{  
   public int Name = 0;  
   public int Count = 0;  
  
   public MyAttr (int iCount, int sName)  
   {  
      Count = iCount;  
      Name = sName;  
   }  
}  
  
[MyAttr(5, 50)]  
class Class1 {}  
  
[MyAttr(6, 60)]  
interface Interface1 {}  
```
