 <span data-ttu-id="172ca-106">下面的示例生成 CS0504：</span><span class="sxs-lookup"><span data-stu-id="172ca-106">The following sample generates CS0504:</span></span>

```csharp  
   {  
      static const int i = 0;   // CS0504, cannot be both static and const

   }  
}  
```
