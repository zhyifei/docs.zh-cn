# <a name="compiler-error-cs0188"></a><span data-ttu-id="134c4-102">编译器错误 CS0188</span><span class="sxs-lookup"><span data-stu-id="134c4-102">Compiler Error CS0188</span></span>

<span data-ttu-id="134c4-103">在给“this”对象的所有字段赋值之前，无法使用该对象</span><span class="sxs-lookup"><span data-stu-id="134c4-103">The 'this' object cannot be used before all of its fields are assigned to</span></span>  
  
 <span data-ttu-id="134c4-104">构造函数必须给 `struct` 中的所有字段都赋值，然后才能调用 `struct` 中的方法。</span><span class="sxs-lookup"><span data-stu-id="134c4-104">All fields in a `struct` have to be assigned by a constructor before the constructor can call a method in the `struct`.</span></span>  
  
 <span data-ttu-id="134c4-105">如果在尝试初始化结构构造函数中的属性时发现此错误，则解决方案将更改该构造函数参数以指定支持字段而不指定该属性本身。</span><span class="sxs-lookup"><span data-stu-id="134c4-105">If you see this error when trying to initialize a property in a struct constructor, the solution is to change the constructor parameter to specify the backing field instead of the property itself.</span></span> <span data-ttu-id="134c4-106">由于自动实现的属性不具有支持字段，所以在任何情况下都不能从构造函数进行初始化，而是应在结构中避免使用自动实现的属性。</span><span class="sxs-lookup"><span data-stu-id="134c4-106">Auto-implemented properties should be avoided in structs because they have no backing field and therefore cannot be initialized in any way from the constructor.</span></span>  
  
 <span data-ttu-id="134c4-107">有关详细信息，请参阅[使用结构](../../../csharp/programming-guide/classes-and-structs/using-structs.md)。</span><span class="sxs-lookup"><span data-stu-id="134c4-107">For more information, see [Using Structs](../../../csharp/programming-guide/classes-and-structs/using-structs.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="134c4-108">示例</span><span class="sxs-lookup"><span data-stu-id="134c4-108">Example</span></span>

 <span data-ttu-id="134c4-109">以下示例生成 CS0188：</span><span class="sxs-lookup"><span data-stu-id="134c4-109">The following sample generates CS0188:</span></span>  

```csharp
// CS0188.cs  
// compile with: /t:library  
namespace MyNamespace  
{  
    class MyClass  
    {  
        struct S  
        {  
            public int a;  
  
            void MyMethod()  
            {  
            }  
  
            S(int i)  
            {  
                // a = i;  
                MyMethod();  // CS0188  
            }  
        }  
        public static void Main()  
        { }  
  
    }  
}  
```
