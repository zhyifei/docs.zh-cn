# <a name="compiler-error-cs0151"></a><span data-ttu-id="2b53f-102">编译器错误 CS0151</span><span class="sxs-lookup"><span data-stu-id="2b53f-102">Compiler Error CS0151</span></span>
<span data-ttu-id="2b53f-103">应为整型类型的值</span><span class="sxs-lookup"><span data-stu-id="2b53f-103">A value of an integral type expected</span></span>  
  
 <span data-ttu-id="2b53f-104">在需要整型数据类型的情况中使用了变量。</span><span class="sxs-lookup"><span data-stu-id="2b53f-104">A variable was used in a situation where an integral data type was required.</span></span> <span data-ttu-id="2b53f-105">有关详细信息，请参阅[类型](../../../csharp/programming-guide/types/index.md)。</span><span class="sxs-lookup"><span data-stu-id="2b53f-105">For more information, see [Types](../../../csharp/programming-guide/types/index.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="2b53f-106">示例</span><span class="sxs-lookup"><span data-stu-id="2b53f-106">Example</span></span>  
 <span data-ttu-id="2b53f-107">在没有转换或者可用的隐式转换导致了不明确的情况时，会出现此错误。</span><span class="sxs-lookup"><span data-stu-id="2b53f-107">This error can occur when there is no conversion or if the available implicit conversions result in an ambiguous situation.</span></span> <span data-ttu-id="2b53f-108">下面的示例生成 CS0151。</span><span class="sxs-lookup"><span data-stu-id="2b53f-108">The following sample generates CS0151.</span></span>  
  
```csharp  
// CS0151.cs  
public class MyClass  
{  
   public static implicit operator int (MyClass aa)  
   {  
      return 0;  
   }  
  
   public static implicit operator long (MyClass aa)  
   {  
      return 0;  
   }  
  
   public static void Main()  
   {  
      MyClass a = new MyClass();  
  
      // Compiler cannot choose between int and long  
      switch (a)   // CS0151  
      // try the following line instead  
      // switch ((int)a)  
      {  
         case 1:  
            break;  
      }  
   }  
}  
```

```csharp  
    }

```
