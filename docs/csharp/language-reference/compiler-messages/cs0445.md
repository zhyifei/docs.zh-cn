# <a name="compiler-error-cs0445"></a><span data-ttu-id="064c2-102">编译器错误 CS0445</span><span class="sxs-lookup"><span data-stu-id="064c2-102">Compiler Error CS0445</span></span>
<span data-ttu-id="064c2-103">无法修改取消装箱转换的结果</span><span class="sxs-lookup"><span data-stu-id="064c2-103">Cannot modify the result of an unboxing conversion</span></span>  
  
 <span data-ttu-id="064c2-104">取消装箱转换的结果是一个临时变量。</span><span class="sxs-lookup"><span data-stu-id="064c2-104">The result of an unboxing conversion is a temporary variable.</span></span> <span data-ttu-id="064c2-105">编译器可防止你修改此类变量，因为任何修改都会随临时变量一并消失。</span><span class="sxs-lookup"><span data-stu-id="064c2-105">The compiler prevents you from modifying such variables because any modification would go away when the temporary variable goes away.</span></span> <span data-ttu-id="064c2-106">若要修复此错误，请声明新的值类型变量来存储中间表达式，并将取消装箱的转换的结果赋给该新变量。</span><span class="sxs-lookup"><span data-stu-id="064c2-106">To fix this, declare a new value-type variable to store the intermediate expression, and assign the result of the unboxing conversion to that variable.</span></span>  
  
 <span data-ttu-id="064c2-107">下面的代码生成 CS0455。</span><span class="sxs-lookup"><span data-stu-id="064c2-107">The following code generates CS0455.</span></span>  
  
```csharp  
// CS0445.CS  
class UnboxingTest  
{  
    public static void Main()  
    {  
        Point p;  
        p.x = 1;  
        p.y = 2;  
        object obj = p;  
        // The following line generates CS0445, because the result  
        // of unboxing obj is a temporary variable.  
        ((Point)obj).x = 2;  
  
        // The following lines resolve the error.  
  
        // Store the result of the unboxing conversion in p2.  
        Point p2;
        p2 = (Point)obj;  
        // Then you can modify the unboxed value.  
        p2.x = 2;  
    }  
}  
  
struct Point  
{  
    public int x, y;  
}  
```
