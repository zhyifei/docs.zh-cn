# <a name="compiler-error-cs1674"></a><span data-ttu-id="f9976-102">编译器错误 CS1674</span><span class="sxs-lookup"><span data-stu-id="f9976-102">Compiler Error CS1674</span></span>
<span data-ttu-id="f9976-103">“T”：using 语句中使用的类型必须可以隐式转换为“System.IDisposable”</span><span class="sxs-lookup"><span data-stu-id="f9976-103">'T': type used in a using statement must be implicitly convertible to 'System.IDisposable'</span></span>  
  
 <span data-ttu-id="f9976-104">[using 语句](../../../csharp/language-reference/keywords/using-statement.md)旨在用于确保在 `using` 块的末尾释放对象，因此只有可释放的类型才能在这类语句中使用。</span><span class="sxs-lookup"><span data-stu-id="f9976-104">The [using Statement](../../../csharp/language-reference/keywords/using-statement.md) is intended to be used to ensure the disposal of an object at the end of the `using` block, thus, only types which are disposable may be used in such a statement.</span></span> <span data-ttu-id="f9976-105">例如，值类型不是可释放的，并且不能将没有约束为类的类型参数假定为可释放。</span><span class="sxs-lookup"><span data-stu-id="f9976-105">For example, value types are not disposable, and type parameters which are not constrained to be classes may not be assumed to be disposable.</span></span>  
  
## <a name="example"></a><span data-ttu-id="f9976-106">示例</span><span class="sxs-lookup"><span data-stu-id="f9976-106">Example</span></span>  
 <span data-ttu-id="f9976-107">下面的示例生成 CS1674。</span><span class="sxs-lookup"><span data-stu-id="f9976-107">The following sample generates CS1674.</span></span>  
  
```csharp  
// CS1674.cs  
class C  
{  
   public static void Main()  
   {  
      int a = 0;  
      a++;  
  
      using (a) {}   // CS1674  
   }  
}  
```  
  
## <a name="example"></a><span data-ttu-id="f9976-108">示例</span><span class="sxs-lookup"><span data-stu-id="f9976-108">Example</span></span>  
 <span data-ttu-id="f9976-109">下面的示例生成 CS1674。</span><span class="sxs-lookup"><span data-stu-id="f9976-109">The following sample generates CS1674.</span></span>  
  
```csharp  
// CS1674_b.cs  
using System;  
class C {  
   public void Test() {  
      using (C c = new C()) {}   // CS1674  
   }  
}  
  
// OK  
class D : IDisposable {  
   void IDisposable.Dispose() {}  
   public void Dispose() {}  
  
   public static void Main() {  
      using (D d = new D()) {}  
   }  
}  
```
