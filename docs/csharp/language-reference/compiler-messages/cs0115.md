## <a name="example"></a><span data-ttu-id="51c32-106">示例</span><span class="sxs-lookup"><span data-stu-id="51c32-106">Example</span></span>  
 <span data-ttu-id="51c32-107">下面的示例生成 CS0115。</span><span class="sxs-lookup"><span data-stu-id="51c32-107">The following sample generates CS0115.</span></span> <span data-ttu-id="51c32-108">可通过以下列两种方法之一解决 CS0115 错误：</span><span class="sxs-lookup"><span data-stu-id="51c32-108">You can resolve CS0115 in one of two ways:</span></span>  
  
-   <span data-ttu-id="51c32-109">从 `MyClass2` 中的方法删除 `override` 关键字。</span><span class="sxs-lookup"><span data-stu-id="51c32-109">Remove the `override` keyword from the method in `MyClass2`.</span></span>  
  
-   <span data-ttu-id="51c32-110">将 `MyClass1` 用作 `MyClass2` 的基类。</span><span class="sxs-lookup"><span data-stu-id="51c32-110">Use `MyClass1` as a base class for `MyClass2`.</span></span>  
  
```csharp  
// CS0115.cs  
namespace MyNamespace  
{  
    abstract public class MyClass1  
    {  
        public abstract int f();  
    }  
  
    abstract public class MyClass2  
    {  
        public override int f()   // CS0115  
        {  
            return 0;  
        }  
  
        public static void Main()  
        {  
        }  
    }  
}  
```
