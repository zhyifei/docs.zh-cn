# <a name="compiler-error-cs1029"></a>编译器错误 CS1029
\#error：“文本”  
  
 显示使用 [#error](../../../csharp/language-reference/preprocessor-directives/preprocessor-error.md) 指令定义的错误文本。  
  
 下面的示例演示如何创建用户定义的错误：  
  
```csharp  
// CS1029.cs  
class Sample  
{  
   static void Main()  
   {  
      #error Let's give an error here   // CS1029  
   }  
}  
```
