# <a name="compiler-error-cs0266"></a><span data-ttu-id="cd346-102">编译器错误 CS0266</span><span class="sxs-lookup"><span data-stu-id="cd346-102">Compiler Error CS0266</span></span>
<span data-ttu-id="cd346-103">无法将类型“type1”隐式转换为“type2”。</span><span class="sxs-lookup"><span data-stu-id="cd346-103">Cannot implicitly convert type 'type1' to 'type2'.</span></span> <span data-ttu-id="cd346-104">存在显式转换（是否缺少强制转换？）</span><span class="sxs-lookup"><span data-stu-id="cd346-104">An explicit conversion exists (are you missing a cast?)</span></span>  
  
 <span data-ttu-id="cd346-105">当代码尝试在无法进行隐式转换的两个类型之间转换时，就会发生此错误。</span><span class="sxs-lookup"><span data-stu-id="cd346-105">This error occurs when your code tries to convert between two types that cannot be implicitly converted, but where an explicit conversion is available.</span></span> <span data-ttu-id="cd346-106">有关详细信息，请参阅[显式转换和类型转换](../../programming-guide/types/casting-and-type-conversions.md)。</span><span class="sxs-lookup"><span data-stu-id="cd346-106">For more information, see [Casting and Type Conversions](../../programming-guide/types/casting-and-type-conversions.md).</span></span>  
  
 <span data-ttu-id="cd346-107">下面的代码演示生成 CS0266 的示例。</span><span class="sxs-lookup"><span data-stu-id="cd346-107">The following code shows examples that generate CS0266.</span></span>  
  
```csharp  
// CS0266.cs  
class MyClass  
{  
    public static void Main()  
    {  
        // You cannot implicitly convert a double to an integer.  
        double d = 3.2;  
  
        // The following line causes compiler error CS0266.  
        int i1 = d;  
  
        // However, you can resolve the error by using an explicit conversion.  
        int i2 = (int)d;  
  
        // You cannot implicitly convert an object to a class type.  
        object obj = "MyString";  
  
        // The following assignment statement causes error CS0266.  
        MyClass myClass = obj;    
  
        // You can resolve the error by using an explicit conversion.  
        MyClass c = ( MyClass )obj;  
  
        // You cannot implicitly convert a base class object to a derived class type.  
        MyClass mc = new MyClass();  
        DerivedClass dc = new DerivedClass();  
  
        // The following line causes compiler error CS0266.  
        dc = mc;  
  
        // You can resolve the error by using an explicit conversion.  
        dc = (DerivedClass)mc;  
    }  
}  
  
class DerivedClass : MyClass  
{  
}  
```
