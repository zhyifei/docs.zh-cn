# <a name="compiler-error-cs0845"></a><span data-ttu-id="de65f-102">编译器错误 CS0845</span><span class="sxs-lookup"><span data-stu-id="de65f-102">Compiler Error CS0845</span></span>
<span data-ttu-id="de65f-103">表达式树 lambda 不能包含左侧为 null 文本的合并运算符。</span><span class="sxs-lookup"><span data-stu-id="de65f-103">An expression tree lambda may not contain a coalescing operator with a null literal left-hand side.</span></span>  
  
 <span data-ttu-id="de65f-104">因为 null 本身没有类型，null 合并运算符无法对其进行操作。</span><span class="sxs-lookup"><span data-stu-id="de65f-104">Because null by itself does not have a type, the null coalescing operator cannot operate on it.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="de65f-105">更正此错误</span><span class="sxs-lookup"><span data-stu-id="de65f-105">To correct this error</span></span>  
  
1.  <span data-ttu-id="de65f-106">将 null 文本强制转换对象。</span><span class="sxs-lookup"><span data-stu-id="de65f-106">Cast the null literal to an object.</span></span>  
  
## <a name="example"></a><span data-ttu-id="de65f-107">示例</span><span class="sxs-lookup"><span data-stu-id="de65f-107">Example</span></span>  
 <span data-ttu-id="de65f-108">下面的代码生成 CS0845：</span><span class="sxs-lookup"><span data-stu-id="de65f-108">The following code generates CS0845:</span></span>  
  
```csharp  
// cs0845.cs  
using System;  
using System.Linq;  
using System.Linq.Expressions;  
  
namespace ConsoleApplication1  
{  
    class Program  
    {  
        static void Main(string[] args)  
        {  
            Expression<Func<object>> e = () => null ?? null; // CS0845  
            // Try the following line instead.  
            // Expression<Func<object>> e = () => (object)null ?? null;  
        }  
    }  
}  
```
