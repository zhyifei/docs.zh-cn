# <a name="compiler-error-cs1519"></a><span data-ttu-id="d4f41-102">编译器错误 CS1519</span><span class="sxs-lookup"><span data-stu-id="d4f41-102">Compiler Error CS1519</span></span>
<span data-ttu-id="d4f41-103">类、结构或接口成员声明中的令牌“token”无效</span><span class="sxs-lookup"><span data-stu-id="d4f41-103">Invalid token 'token' in class, struct, or interface member declaration</span></span>  
  
 <span data-ttu-id="d4f41-104">令牌的放置位置无效时会生成此错误。</span><span class="sxs-lookup"><span data-stu-id="d4f41-104">This error is generated whenever a token is encountered in a location where it does not belong.</span></span> <span data-ttu-id="d4f41-105">令牌是关键字；标识符（类、结构、方法等的名称）；字符串、字符或数字文本值（例如 108、“Hello”或 A）；运算符或标点符号（例如 `==` 或 `;`）。</span><span class="sxs-lookup"><span data-stu-id="d4f41-105">A *token* is a keyword; an identifier (the name of a class, struct, method, and so on); a string, character, or numeric literal value such as 108, "Hello", or 'A'; or an operator or punctuator such as `==` or `;`.</span></span>  
  
 <span data-ttu-id="d4f41-106">在任何[类](../../../csharp/language-reference/keywords/class.md)、结构或接口成员声明中，只要在类型前面包含无效修饰符都会生成此错误。</span><span class="sxs-lookup"><span data-stu-id="d4f41-106">Any [class](../../../csharp/language-reference/keywords/class.md), struct, or interface member declaration that contains invalid modifiers before the type will generate this error.</span></span> <span data-ttu-id="d4f41-107">若要修复此错误，请删除无效的修饰符。</span><span class="sxs-lookup"><span data-stu-id="d4f41-107">To fix the error, remove the invalid modifiers.</span></span>  
  
 <span data-ttu-id="d4f41-108">下例在五个位置生成 CS1519，因为这些令牌的放置位置无效：</span><span class="sxs-lookup"><span data-stu-id="d4f41-108">The following sample generates CS1519 in five places because tokens are placed in locations where they are not valid:</span></span>  
  
```csharp  
// CS1519.cs  
// Generates CS1519 because a class name cannot be a number:  
class Test 42   
{  
// Generates CS1519 because of 'j' following 'I'  
// with no comma between them:  
    int i j;   
// Generates CS1519 because of "checked" on void method:  
    checked void f4();     
  
// Generates CS1519 because of "num":  
    void f5(int a num){}        
  
// Generates CS1519 because of namespace inside class:  
    namespace;             
  
}  
```
