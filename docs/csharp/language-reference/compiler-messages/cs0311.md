# <a name="compiler-error-cs0311"></a><span data-ttu-id="0f5da-102">编译器错误 CS0311</span><span class="sxs-lookup"><span data-stu-id="0f5da-102">Compiler Error CS0311</span></span>
<span data-ttu-id="0f5da-103">不能将类型“type1”用作泛型类型或方法“\<name>”中的类型参数“T”。</span><span class="sxs-lookup"><span data-stu-id="0f5da-103">The type 'type1' cannot be used as type parameter 'T' in the generic type or method '\<name>'.</span></span> <span data-ttu-id="0f5da-104">没有从“type1”到“ type2“的隐式引用转换。</span><span class="sxs-lookup"><span data-stu-id="0f5da-104">There is no implicit reference conversion from 'type1' to 'type2'.</span></span>  
  
 <span data-ttu-id="0f5da-105">对泛型类型参数应用约束时，必须存在从具体参数到约束类型的隐式标识或引用转换。</span><span class="sxs-lookup"><span data-stu-id="0f5da-105">When a constraint is applied to a generic type parameter, an implicit identity or reference conversion must exist from the concrete argument to the type of the constraint.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="0f5da-106">更正此错误</span><span class="sxs-lookup"><span data-stu-id="0f5da-106">To correct this error</span></span>  
  
1.  <span data-ttu-id="0f5da-107">更改用来创建类的参数。</span><span class="sxs-lookup"><span data-stu-id="0f5da-107">Change the argument you are using to create the class.</span></span>  
  
2.  <span data-ttu-id="0f5da-108">如果你拥有类，则可以删除该约束，否则需要执行一些操作来启用隐式引用或标识转换。</span><span class="sxs-lookup"><span data-stu-id="0f5da-108">If you own the class, you can remove the constraint or else do something to enable an implicit reference or identity conversion.</span></span> <span data-ttu-id="0f5da-109">例如，可以从第一个类型继承第二个类型。</span><span class="sxs-lookup"><span data-stu-id="0f5da-109">For example, you can make the second type inherit from the first.</span></span>  
  
## <a name="example"></a><span data-ttu-id="0f5da-110">示例</span><span class="sxs-lookup"><span data-stu-id="0f5da-110">Example</span></span>  
  
```csharp  
// cs0311.cs  
class B{}  
class C{}  
class Test<T> where T : C  
{ }  
  
class Program  
{  
    static void Main()  
    {  
        Test<B> test = new Test<B>(); //CS0311  
    }  
}  
```  
  
 <span data-ttu-id="0f5da-111">如果在尝试使用值类型参数时发生此错误，请注意，隐式数字转换（例如从 `short` 到 `int`）不符合泛型类型参数。</span><span class="sxs-lookup"><span data-stu-id="0f5da-111">If this error occurs when trying to use a value-type argument, notice that an implicit numeric conversion, for example from `short` to `int`, does not satisfy a generic type parameter.</span></span>
