# <a name="compiler-error-cs0120"></a><span data-ttu-id="8724b-102">编译器错误 CS0120</span><span class="sxs-lookup"><span data-stu-id="8724b-102">Compiler Error CS0120</span></span>
<span data-ttu-id="8724b-103">非静态的字段、方法或属性“member”要求对象引用</span><span class="sxs-lookup"><span data-stu-id="8724b-103">An object reference is required for the nonstatic field, method, or property 'member'</span></span>  
  
 <span data-ttu-id="8724b-104">必须先创建对象实例，才能使用非静态的字段、方法或属性。</span><span class="sxs-lookup"><span data-stu-id="8724b-104">In order to use a non-static field, method, or property, you must first create an object instance.</span></span> <span data-ttu-id="8724b-105">有关静态方法的详细信息，请参阅[静态类和静态类成员](../../../csharp/programming-guide/classes-and-structs/static-classes-and-static-class-members.md)。</span><span class="sxs-lookup"><span data-stu-id="8724b-105">For more information about static methods, see [Static Classes and Static Class Members](../../../csharp/programming-guide/classes-and-structs/static-classes-and-static-class-members.md).</span></span> <span data-ttu-id="8724b-106">有关创建类的实例的详细信息，请参阅[实例构造函数](../../../csharp/programming-guide/classes-and-structs/instance-constructors.md)。</span><span class="sxs-lookup"><span data-stu-id="8724b-106">For more information about creating instances of classes, see [Instance Constructors](../../../csharp/programming-guide/classes-and-structs/instance-constructors.md).</span></span>  
  
 <span data-ttu-id="8724b-107">以下示例生成 CS0120：</span><span class="sxs-lookup"><span data-stu-id="8724b-107">The following sample generates CS0120:</span></span>  
  
```csharp  
// CS0120_1.cs  
public class MyClass  
{  
   // Non-static field  
   public int i;   
   // Non-static method  
   public void f(){}  
   // Non-static property  
   int Prop  
   {  
      get   
      {  
         return 1;   
      }  
   }  
  
   public static void Main()  
   {  
      i = 10;   // CS0120  
      f();   // CS0120  
      int p = Prop;   // CS0120  
      // try the following lines instead  
      // MyClass mc = new MyClass();  
      // mc.i = 10;  
      // mc.f();  
      // int p = mc.Prop;  
   }  
}  
```  
  
 <span data-ttu-id="8724b-108">如果从静态方法调用非静态方法，也会生成 CS0120，如下所示：</span><span class="sxs-lookup"><span data-stu-id="8724b-108">CS0120 will also be generated if there is a call to a non-static method from a static method, as follows:</span></span>  
  
```csharp  
// CS0120_2.cs  
// CS0120 expected  
using System;  
  
public class MyClass  
{  
   public static void Main()  
   {  
      TestCall();   // CS0120  
      // To call a non-static method from Main,  
      // first create an instance of the class.  
      // Use the following two lines instead:  
      // MyClass anInstanceofMyClass = new MyClass();  
      // anInstanceofMyClass.TestCall();  
   }  
  
   public void TestCall()  
   {  
   }  
}  
```  
  
 <span data-ttu-id="8724b-109">同样，静态方法不能调用实例方法，除非显式给它提供了类的实例，如下所示：</span><span class="sxs-lookup"><span data-stu-id="8724b-109">Similarly, a static method cannot call an instance method unless you explicitly give it an instance of the class, as follows:</span></span>  
  
```csharp  
// CS0120_3.cs  
using System;  
  
public class MyClass  
{  
   public static void Main()  
   {  
      do_it("Hello There");   // CS0120  
   }  
  
   private void do_it(string sText)  
   // You could also add the keyword static to the method definition:  
   // private static void do_it(string sText)  
   {  
      Console.WriteLine(sText);  
   }  
}  
```
