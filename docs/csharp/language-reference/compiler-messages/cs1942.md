# <a name="compiler-error-cs1942"></a><span data-ttu-id="1535c-102">编译器错误 CS1942</span><span class="sxs-lookup"><span data-stu-id="1535c-102">Compiler Error CS1942</span></span>
<span data-ttu-id="1535c-103">“clause”子句中表达式的类型不正确。</span><span class="sxs-lookup"><span data-stu-id="1535c-103">The type of the expression in the 'clause' clause is incorrect.</span></span> <span data-ttu-id="1535c-104">对“method”的调用中，类型推理失败。</span><span class="sxs-lookup"><span data-stu-id="1535c-104">Type inference failed in the call to 'method'.</span></span>  
  
 <span data-ttu-id="1535c-105">如果为范围变量指定了不正确的显式类型，通常会生成此错误。</span><span class="sxs-lookup"><span data-stu-id="1535c-105">This error is typically generated when the range variable has been given an incorrect explicit type.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="1535c-106">更正此错误</span><span class="sxs-lookup"><span data-stu-id="1535c-106">To correct this error</span></span>  
  
1.  <span data-ttu-id="1535c-107">如果范围变量是显式类型的，请确保该类型与变量循环访问的集合中的元素类型相同，或者可从后者隐式转换。</span><span class="sxs-lookup"><span data-stu-id="1535c-107">If the range variable is explicitly typed, make sure that the type is either the same as, or implicitly convertible from, the type of the elements in the collection it iterates.</span></span> <span data-ttu-id="1535c-108">如果范围变量前面带有 `var` 关键字，请删除 `var`。</span><span class="sxs-lookup"><span data-stu-id="1535c-108">If the range variable is preceded with the `var` keyword, remove `var`.</span></span>  
  
## <a name="example"></a><span data-ttu-id="1535c-109">示例</span><span class="sxs-lookup"><span data-stu-id="1535c-109">Example</span></span>  
 <span data-ttu-id="1535c-110">下面的代码生成 CS1942：</span><span class="sxs-lookup"><span data-stu-id="1535c-110">The following code generates CS1942:</span></span>  
  
```csharp  
// cs1942.cs  
class Program  
    {  
        static void Main(string[] args)  
        {  
            var x = from var i in Enumerable.Range(1, 100) // CS1949  
                    select i; //CS1942  
        }  
    }  
```  
  
 <span data-ttu-id="1535c-111">CS1942 与 CS1949 相关，因为对范围变量使用 `var` 会导致因 `var` 不是类型而使基础 `Cast<T>` 运算失败。</span><span class="sxs-lookup"><span data-stu-id="1535c-111">CS1942 is related to CS1949 because the use of `var` with a range variable causes the underlying `Cast<T>` operation to fail because `var` is not a type.</span></span>
