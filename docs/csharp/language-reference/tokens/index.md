# <a name="c-special-characters"></a>C# 特殊字符

特殊字符是预定义的上下文字符，用于修改最前面插入了此类字符的程序元素（文本字符串、标识符或属性名称）。 C# 支持以下特殊字符： 

- [@](../../../csharp/language-reference/tokens/verbatim.md)：逐字字符串标识符字符。 

- [$](../../../csharp/language-reference/tokens/interpolated.md)：内插的字符串字符。
