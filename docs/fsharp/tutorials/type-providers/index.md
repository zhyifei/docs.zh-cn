# <a name="type-providers"></a><span data-ttu-id="9920f-103">类型提供程序</span><span class="sxs-lookup"><span data-stu-id="9920f-103">Type Providers</span></span>

<span data-ttu-id="9920f-104">F# 类型提供程序是提供在程序中使用的类型、属性和方法的组件。</span><span class="sxs-lookup"><span data-stu-id="9920f-104">An F# type provider is a component that provides types, properties, and methods for use in your program.</span></span> <span data-ttu-id="9920f-105">类型提供程序生成所谓**提供类型**，它由 F # 编译器生成并基于外部数据源。</span><span class="sxs-lookup"><span data-stu-id="9920f-105">Type Providers generate what are known as **Provided Types**, which are generated by the F# compiler and are based on an external data source.</span></span>

<span data-ttu-id="9920f-106">例如，F # 类型提供程序的 SQL 可以生成表示关系数据库中表和列的类型。</span><span class="sxs-lookup"><span data-stu-id="9920f-106">For example, an F# Type Provider for SQL can generate types representing tables and columns in a relational database.</span></span> <span data-ttu-id="9920f-107">实际上，这是什么[SQLProvider](https://fsprojects.github.io/SQLProvider/)类型提供程序执行。</span><span class="sxs-lookup"><span data-stu-id="9920f-107">In fact, this is what the [SQLProvider](https://fsprojects.github.io/SQLProvider/) Type Provider does.</span></span>

<span data-ttu-id="9920f-108">提供类型取决于输入参数类型提供程序。</span><span class="sxs-lookup"><span data-stu-id="9920f-108">Provided Types depend on input parameters to a Type Provider.</span></span> <span data-ttu-id="9920f-109">此类输入可以是示例数据源 （如 JSON 架构中的文件），直接指向外部服务时或数据源的连接字符串的 URL。</span><span class="sxs-lookup"><span data-stu-id="9920f-109">Such input can be a sample data source (such as a JSON schema file), a URL pointing directly to an external service, or a connection string to a data source.</span></span> <span data-ttu-id="9920f-110">类型提供程序还可确保按需; 的用户仅能展开类型组也就是说，它们会展开如果你的程序实际上引用的类型。</span><span class="sxs-lookup"><span data-stu-id="9920f-110">A Type Provider can also ensure that groups of types are only expanded on demand; that is, they are expanded if the types are actually referenced by your program.</span></span> <span data-ttu-id="9920f-111">这允许以强类型的方式直接按需集成大型信息空间（如联机数据市场）。</span><span class="sxs-lookup"><span data-stu-id="9920f-111">This allows for the direct, on-demand integration of large-scale information spaces such as online data markets in a strongly typed way.</span></span>

## <a name="generative-and-erased-type-providers"></a><span data-ttu-id="9920f-112">生成和清除类型提供程序</span><span class="sxs-lookup"><span data-stu-id="9920f-112">Generative and Erased Type Providers</span></span>

<span data-ttu-id="9920f-113">类型提供程序有两种形式： 生成和清除。</span><span class="sxs-lookup"><span data-stu-id="9920f-113">Type Providers come in two forms: Generative and Erased.</span></span>

<span data-ttu-id="9920f-114">生成类型提供程序生成可在生成的程序集写入为.NET 类型的类型。</span><span class="sxs-lookup"><span data-stu-id="9920f-114">Generative Type Providers produce types that can be written as .NET types into the assembly in which they are produced.</span></span> <span data-ttu-id="9920f-115">这使他们能够从其他程序集中的代码使用。</span><span class="sxs-lookup"><span data-stu-id="9920f-115">This allows them to be consumed from code in other assemblies.</span></span> <span data-ttu-id="9920f-116">这意味着数据源的类型化表示形式通常必须是一个可行表示，并将.NET 类型。</span><span class="sxs-lookup"><span data-stu-id="9920f-116">This means that the typed representation of the data source must generally be one that is feasible to represent with .NET types.</span></span>

<span data-ttu-id="9920f-117">擦除类型提供程序生成仅可以在程序集或从生成的项目中使用的类型。</span><span class="sxs-lookup"><span data-stu-id="9920f-117">Erasing Type Providers produce types that can only be consumed in the assembly or project they are generated from.</span></span> <span data-ttu-id="9920f-118">类型是临时;也就是说，它们不会写入到程序集和其他程序集中的代码不能使用。</span><span class="sxs-lookup"><span data-stu-id="9920f-118">The types are ephemeral; that is, they are not written into an assembly and cannot be consumed by code in other assemblies.</span></span> <span data-ttu-id="9920f-119">它们可以包含*延迟*成员，从而可以使用提供的类型从可能的无限信息空间。</span><span class="sxs-lookup"><span data-stu-id="9920f-119">They can contain *delayed* members, allowing you to use provided types from a potentially infinite information space.</span></span> <span data-ttu-id="9920f-120">它们可用于使用大型和相互关联的数据源的一小部分。</span><span class="sxs-lookup"><span data-stu-id="9920f-120">They are useful for using a small subset of a large and interconnected data source.</span></span>
