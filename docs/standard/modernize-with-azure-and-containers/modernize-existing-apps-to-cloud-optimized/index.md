# <a name="modernize-existing-net-apps-to-cloud-optimized-applications"></a>更新运行的.NET 应用到云优化的应用程序

> 设想：更新现有.NET Framework 应用程序到云优化的应用程序以显著提高您的部署灵活性，从而可以更快地交付，降低应用程序的交付成本。

若要利用云和新技术（如容器）的优势，至少应部分更新现有 .NET 应用程序。 最终，更新企业应用程序将降低总拥有成本。

部分更新应用并不一定意味着完全迁移和体系结构重建。 最初，你可以完成现代化重要但容易实现现代化你现有的应用程序。 可以维护当前基本代码，这些代码使用现有.NET Framework 版本编写并具有任意 Windows 和 IIS 依赖项。 图 4-1 突出显示了如何云优化的应用程序在 Azure 应用程序更新成熟度模型中的位置。
