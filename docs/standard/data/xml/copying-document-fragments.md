# 复制文档片段
可以创建一个 **XmlDocumentFragment** 节点，然后在它下面添加节点。  当使用 **InsertNode** 方法插入 **XmlDocumentFragment** 时，不会复制 **XmlDocumentFragment** 节点，但会在 XML 文档对象模型 \(DOM\) 中插入它的子节点。
