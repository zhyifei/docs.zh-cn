### <a name="escaping-braces"></a><span data-ttu-id="5fa39-166">转义大括号</span><span class="sxs-lookup"><span data-stu-id="5fa39-166">Escaping Braces</span></span>  
 <span data-ttu-id="5fa39-167">左大括号和右大括号被解释为格式项的开始和结束。</span><span class="sxs-lookup"><span data-stu-id="5fa39-167">Opening and closing braces are interpreted as starting and ending a format item.</span></span> <span data-ttu-id="5fa39-168">因此，必须使用转义序列显示文本左大括号或右大括号。</span><span class="sxs-lookup"><span data-stu-id="5fa39-168">Consequently, you must use an escape sequence to display a literal opening brace or closing brace.</span></span> <span data-ttu-id="5fa39-169">在固定文本中指定两个左大括号 ("{{") 以显示一个左大括号 ("{")，或指定两个右大括号 ("}}") 以显示一个右大括号 ("}")。</span><span class="sxs-lookup"><span data-stu-id="5fa39-169">Specify two opening braces ("{{") in the fixed text to display one opening brace ("{"), or two closing braces ("}}") to display one closing brace ("}").</span></span> <span data-ttu-id="5fa39-170">按照在格式项中遇到大括号的顺序依次解释它们。</span><span class="sxs-lookup"><span data-stu-id="5fa39-170">Braces in a format item are interpreted sequentially in the order they are encountered.</span></span> <span data-ttu-id="5fa39-171">不支持解释嵌套的大括号。</span><span class="sxs-lookup"><span data-stu-id="5fa39-171">Interpreting nested braces is not supported.</span></span>  
  
 <span data-ttu-id="5fa39-172">解释转义大括号的方式会导致意外的结果。</span><span class="sxs-lookup"><span data-stu-id="5fa39-172">The way escaped braces are interpreted can lead to unexpected results.</span></span> <span data-ttu-id="5fa39-173">例如，假设格式项为“{{{0:D}}}”，旨在显示左大括号、采用十进制数格式的数值和右大括号。</span><span class="sxs-lookup"><span data-stu-id="5fa39-173">For example, consider the format item "{{{0:D}}}", which is intended to display an opening brace, a numeric value formatted as a decimal number, and a closing brace.</span></span> <span data-ttu-id="5fa39-174">但是，实际是按照以下方式解释该格式项：</span><span class="sxs-lookup"><span data-stu-id="5fa39-174">However, the format item is actually interpreted in the following manner:</span></span>  
  
1.  <span data-ttu-id="5fa39-175">前两个左大括号 ("{{") 被转义，生成一个左大括号。</span><span class="sxs-lookup"><span data-stu-id="5fa39-175">The first two opening braces ("{{") are escaped and yield one opening brace.</span></span>

3.  <span data-ttu-id="5fa39-177">下一个字符 ("D") 将被解释为 Decimal 标准数值格式说明符，但后面的两个转义大括号 ("}}") 生成单个大括号。</span><span class="sxs-lookup"><span data-stu-id="5fa39-177">The next character ("D") would be interpreted as the Decimal standard numeric format specifier, but the next two escaped braces ("}}") yield a single brace.</span></span> <span data-ttu-id="5fa39-178">由于得到的字符串 ("D}") 不是标准数值格式说明符号，所以得到的字符串会被解释为用于显示字符串“D}”的自定义格式字符串。</span><span class="sxs-lookup"><span data-stu-id="5fa39-178">Because the resulting string ("D}") is not a standard numeric format specifier, the resulting string is interpreted as a custom format string that means display the literal string "D}".</span></span>  
  
4.  <span data-ttu-id="5fa39-179">最后一个大括号 ("}") 被解释为格式项的结束。</span><span class="sxs-lookup"><span data-stu-id="5fa39-179">The last brace ("}") is interpreted as the end of the format item.</span></span>
