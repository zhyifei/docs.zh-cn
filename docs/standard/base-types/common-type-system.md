### <a name="overloading"></a><span data-ttu-id="ea526-390">重载</span><span class="sxs-lookup"><span data-stu-id="ea526-390">Overloading</span></span>  
 <span data-ttu-id="ea526-391">每个类型成员都有一个唯一的签名。</span><span class="sxs-lookup"><span data-stu-id="ea526-391">Each type member has a unique signature.</span></span> <span data-ttu-id="ea526-392">方法签名由方法名称和一个参数列表（方法的参数的顺序和类型）组成。</span><span class="sxs-lookup"><span data-stu-id="ea526-392">Method signatures consist of the method name and a parameter list (the order and types of the method's arguments).</span></span> <span data-ttu-id="ea526-393">可以在一种类型内定义具有相同名称的多种方法，只要这些方法的签名不同。</span><span class="sxs-lookup"><span data-stu-id="ea526-393">Multiple methods with the same name can be defined within a type as long as their signatures differ.</span></span> <span data-ttu-id="ea526-394">当定义两种或多种具有相同名称的方法时，就称作重载。</span><span class="sxs-lookup"><span data-stu-id="ea526-394">When two or more methods with the same name are defined, the method is said to be overloaded.</span></span> <span data-ttu-id="ea526-395">例如，在 <xref:System.Char?displayProperty=nameWithType> 中，重载了 <xref:System.Char.IsDigit%2A> 方法。</span><span class="sxs-lookup"><span data-stu-id="ea526-395">For example, in <xref:System.Char?displayProperty=nameWithType>, the <xref:System.Char.IsDigit%2A> method is overloaded.</span></span> <span data-ttu-id="ea526-396">一个方法采用 <xref:System.Char>。</span><span class="sxs-lookup"><span data-stu-id="ea526-396">One method takes a <xref:System.Char>.</span></span> <span data-ttu-id="ea526-397">另一个方法采用 <xref:System.String> 和 <xref:System.Int32>。</span><span class="sxs-lookup"><span data-stu-id="ea526-397">The other method takes a <xref:System.String> and an <xref:System.Int32>.</span></span>
