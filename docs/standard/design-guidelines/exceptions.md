# <a name="design-guidelines-for-exceptions"></a>异常设计准则
异常处理有很多优于基于返回值的错误报告。 良好的框架设计可帮助应用程序开发人员的好处的异常。 本部分讨论其优势的异常，并提供了有效地使用它们的准则。
