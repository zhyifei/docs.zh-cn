# Parallel Diagnostic Tools
[!INCLUDE[vs_dev10_ext](../../../includes/vs-dev10-ext-md.md)] 为调试和分析多线程应用程序提供了广泛的支持。  
  
## 调试  
 Visual Studio 调试器新增了用于调试并行应用程序的窗口。  有关更多信息，请参见下列主题：  
  
-   [使用“并行堆栈”窗口](../Topic/Using%20the%20Parallel%20Stacks%20Window.md)  
  
-   [使用“任务”窗口](../Topic/Using%20the%20Tasks%20Window.md)  
  
-   [演练：调试并行应用程序](../Topic/Walkthrough:%20Debugging%20a%20Parallel%20Application.md).  
  
## 分析  
 您可利用“并发可视化工具”报告视图直观显示并行程序中的线程如何彼此进行交互，以及如何与系统上其他进程中的线程进行交互。  有关更多信息，请参见[并发可视化工具](../Topic/Concurrency%20Visualizer.md)。
