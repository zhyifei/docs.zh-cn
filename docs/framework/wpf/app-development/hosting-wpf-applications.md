# 承载 WPF 应用程序
WPF XAML 浏览器应用程序 \(XBAP\) 是丰富客户端应用程序，可以部署到 Web 服务器并在浏览器中启动。  WPF 主机 \(PresentationHost.exe\) 注册为 XBAP 和 XAML 文件的 shell 和 MIME 处理程序。  因此，Internet Explorer 知道在启动 XBAP 时启动 WPF 主机。  Firefox 用户可以安装使 Firefox 也可以承载 XBAP 的 Firefox 加载项。  可以使用 WPF 提供的本机浏览器宿主 API 在其他计算机或独立应用程序中承载 XBAP。
