# 菜单
<xref:System.Windows.Controls.Menu> 是一个控件，使用该控件可以对那些与命令或事件处理程序相关联的元素以分层方式进行组织。  每个 <xref:System.Windows.Controls.Menu> 可以包含多个 <xref:System.Windows.Controls.MenuItem> 控件。  每个 <xref:System.Windows.Controls.MenuItem> 都可调用命令或调用 `Click` 事件处理程序。  <xref:System.Windows.Controls.MenuItem> 也可以有多个 <xref:System.Windows.Controls.MenuItem> 元素作为子项，从而构成子菜单。  
  
 下图演示了菜单控件的三种不同状态。  默认状态是没有设备（如鼠标指针）停留在 <xref:System.Windows.Controls.Menu> 上时的状态。  当鼠标指针悬停在 <xref:System.Windows.Controls.Menu> 上时显示焦点状态，当在 <xref:System.Windows.Controls.Menu> 上单击鼠标按钮时显示按下状态。
