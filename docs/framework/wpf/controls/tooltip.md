# ToolTip
工具提示是用户将鼠标指针暂停在元素上（例如暂停在 <xref:System.Windows.Controls.Button> 上）时出现的小型弹出窗口。  
  
 下图显示了指向**“关闭”**<xref:System.Windows.Controls.Button> 的鼠标指针，该按钮随后会显示其标识 <xref:System.Windows.Controls.ToolTip>。
