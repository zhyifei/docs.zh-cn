# 数据绑定 (WPF)
Windows Presentation Foundation \(WPF\) 数据绑定为应用程序提供了一种表示数据和与数据交互的简单而又一致的方法。  元素能够以公共语言运行时 \(CLR\) 对象和 XML 的形式绑定到各种数据源中的数据。
