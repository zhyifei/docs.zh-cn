# 1143 - FlowchartNextNull
## 属性  
  
|||  
|-|-|  
|ID|1143|  
|关键字|WFActivities|  
|级别|信息|  
|通道|Microsoft\-Windows\-应用程序服务器\-应用程序\/调试|  
  
## 描述  
 指示 Flowchart 中的下一个节点为 null。  Flowchart 执行将结束。  
  
## 消息  
 Flowchart“%1”\/FlowStep \- 下一个节点为 null。  Flowchart 执行将结束。  
  
## 详细信息  
  
|数据项名称|数据项类型|描述|  
|-----------|-----------|--------|  
|FlowChart|xs:string|FlowChart 的显示名称。|  
|AppDomain|xs:string|由 AppDomain.CurrentDomain.FriendlyName 返回的字符串。|
