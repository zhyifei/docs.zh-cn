# 2025 - InternalCacheMetadataStop
## 属性  
  
|||  
|-|-|  
|ID|2025|  
|关键字|WFRuntime|  
|级别|详细|  
|通道|Microsoft\-Windows\-应用程序服务器\-应用程序\/调试|  
  
## 描述
