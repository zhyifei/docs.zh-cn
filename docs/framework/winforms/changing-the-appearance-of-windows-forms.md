# <a name="changing-the-appearance-of-windows-forms"></a>更改 Windows 窗体外观
可以按多种不同的方式自定义 Windows 窗体应用程序的外观，例如，更改 Windows 窗体应用程序的边框、不透明度、形状、样式，或设置它的背景图像。
