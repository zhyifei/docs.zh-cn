# 将设计器与 Windows 窗体 DataGridView 控件结合使用
Visual Studio 为 `DataGridView` 控件提供设计器支持，这可令您在不编写代码的情况下执行很多安装任务。  这些任务包括将控件绑定到数据源，修改用来显示数据的列以及调整控件的外观和基本行为。
