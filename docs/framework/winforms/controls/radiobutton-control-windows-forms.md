# <a name="radiobutton-control-windows-forms"></a>RadioButton 控件（Windows 窗体）
Windows 窗体`RadioButton`控件向用户显示的一组两个或多个互斥选项。 尽管单选按钮和复选框似乎作用类似，但还有一项重大差异： 如果用户选择的单选按钮，不能同时选中同一个组中的其他单选按钮。
