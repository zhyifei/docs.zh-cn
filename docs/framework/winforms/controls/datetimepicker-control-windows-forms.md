# DateTimePicker 控件（Windows 窗体）
使用 Windows 窗体 `DateTimePicker` 控件，用户可以从日期或时间列表中选择单个项。  在用来表示日期时，它显示为两部分：一个下拉列表（带有以文本形式表示的日期）和一个网格（在单击列表旁边的向下箭头时显示）。
