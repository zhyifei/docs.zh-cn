# ErrorProvider 组件（Windows 窗体）
使用 Windows 窗体 `ErrorProvider` 组件，可以在不打扰用户的情况下向用户显示有错误发生。  当验证用户在窗体中的输入或显示数据集内的错误时，一般要用到该控件。
