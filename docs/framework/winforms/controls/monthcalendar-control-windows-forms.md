# <a name="monthcalendar-control-windows-forms"></a>MonthCalendar 控件（Windows 窗体）
Windows 窗体 `MonthCalendar` 控件提供了一个直观的图形界面，用户可在其中查看和设置日期信息。 控件将显示一个网格，其中包含月份的编号日期，并按周中各天的顺序排列。 可以通过单击月份标题两侧的箭头按钮来选择其他月份。 与类似的 <xref:System.Windows.Forms.DateTimePicker> 控件不同，您可以使用此控件选择日期范围。不过，<xref:System.Windows.Forms.DateTimePicker> 控件允许您设置时间和日期。
