# <a name="trackbar-control-windows-forms"></a>TrackBar 控件（Windows 窗体）
Windows 窗体 `TrackBar` 控件（有时也称为 "滑块" 控件）用于在大量信息中导航，或用于直观调整数字设置。 `TrackBar` 控件包含两部分：滚动块（也称为滑块）和刻度线。 滚动块是可以调整的部分。 其位置与 <xref:System.Windows.Forms.TrackBar.Value%2A> 属性相对应。 刻度线是按固定间隔分隔的可视指示器。 跟踪条以指定的增量移动，并可水平或垂直对齐。 跟踪条的示例用法是设置光标闪烁速率或鼠标速度。
