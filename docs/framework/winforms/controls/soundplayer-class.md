# <a name="soundplayer-class"></a>SoundPlayer 类
`SoundPlayer` 类使你能够轻松地将声音纳入应用程序。  
  
 还可使用 <xref:System.Media.SystemSounds> 类播放常见的系统声音，包括提示音。
