# <a name="textbox-control-windows-forms"></a>TextBox 控件（Windows 窗体）
若要获取用户输入或显示文本使用 Windows 窗体的文本框。 `TextBox`控件通常用于可编辑文本，尽管也可将其只读的。 文本框可以显示多个行，该控件的大小的文本换行并添加基本格式设置。 `TextBox`控件允许文本在控件中输入或显示一种格式。
