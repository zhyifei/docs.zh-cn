# CheckBox 控件（Windows 窗体）
Windows 窗体 `CheckBox` 控件指示特定条件处于打开开始关闭状态。  通常用于向用户显示 Yes\/No 或 True\/False 选项。  可以使用组中的复选框控件来显示用户可从中选择一个或多选的多个选项。  它类似于 <xref:System.Windows.Forms.RadioButton> 控件，但可以选择任意数量的已分组 `CheckBox` 控件。
