# <a name="panel-control-windows-forms"></a>Panel 控件（Windows 窗体）
Windows 窗体 `Panel` 控件用于为其他控件提供可识别的分组。 通常使用面板按功能细分窗体。 `Panel` 控件类似于 <xref:System.Windows.Forms.GroupBox> 控件;但是，只有 `Panel` 控件才能具有滚动条，并且只有 <xref:System.Windows.Forms.GroupBox> 控件才能显示标题。
