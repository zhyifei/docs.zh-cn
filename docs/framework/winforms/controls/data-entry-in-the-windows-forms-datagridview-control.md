# Windows 窗体 DataGridView 控件中的数据输入
`DataGridView` 控件提供几个功能，使您可以更改用户在控件中添加或修改数据的方式。  例如，通过为新行提供默认值以及在出错时通知用户，可以提高数据输入的效率。
