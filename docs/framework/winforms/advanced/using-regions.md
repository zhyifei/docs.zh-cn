# 使用区域
[!INCLUDE[ndptecgdiplus](../../../../includes/ndptecgdiplus-md.md)] <xref:System.Drawing.Region> 类可用于定义自定义形状。  该形状可由直线、多边形和曲线组成。  
  
 区域的两个常见用途是命中检测和剪辑。  命中检测确定鼠标是否在屏幕的某个区域单击。  剪辑是将绘图限制在某个区域。
