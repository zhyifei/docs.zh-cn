# 使用钢笔绘制线条和形状
使用 [!INCLUDE[ndptecgdiplus](../../../../includes/ndptecgdiplus-md.md)] `Pen` 对象绘制线段、曲线和形状的轮廓线。  在本节中，除非特别说明只表示线段，否则“线条”指上述任何一种线。  设置钢笔的属性以控制用其绘制的线条的颜色、宽度、对齐方式和样式。
