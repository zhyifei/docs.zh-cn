# 使用应用程序协议
.NET Framework 支持常用的 Internet 应用程序协议。 本部分包括有关使用 [HTTP](../../../docs/framework/network-programming/http.md)、“TCP”和“UDP”协议的信息，以及使用 [Windows Sockets](../../../docs/framework/network-programming/sockets.md) 接口实现自定义协议的相关信息。
