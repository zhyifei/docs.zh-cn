## <a name="user-instances"></a><span data-ttu-id="096a8-114">用户实例</span><span class="sxs-lookup"><span data-stu-id="096a8-114">User Instances</span></span>

 <span data-ttu-id="096a8-118">用户实例是从 SQL Server 或 SQL Server Express 的主实例代表用户生成的。</span><span class="sxs-lookup"><span data-stu-id="096a8-118">A user instance is generated from a primary instance of SQL Server or SQL Server Express on behalf of a user.</span></span> <span data-ttu-id="096a8-119">它在用户的 Windows 安全上下文下作为用户进程运行，而不作为服务运行。</span><span class="sxs-lookup"><span data-stu-id="096a8-119">It runs as a user process under the Windows security context of the user, not as a service.</span></span> <span data-ttu-id="096a8-120">不允许 SQL Server 登录；仅支持 Windows 登录。</span><span class="sxs-lookup"><span data-stu-id="096a8-120">SQL Server logins are disallowed; only Windows logins are supported.</span></span> <span data-ttu-id="096a8-121">这可防止在用户实例上运行的软件进行用户无权限进行的系统范围的更改。</span><span class="sxs-lookup"><span data-stu-id="096a8-121">This prevents software executing on a user instance from making system-wide changes that the user would not have permissions to make.</span></span> <span data-ttu-id="096a8-122">用户实例也称为子实例或客户端实例，有时可通过使用“run as normal user”（作为正常用户运行）的首字母缩略词 RANU 来引用。</span><span class="sxs-lookup"><span data-stu-id="096a8-122">A user instance is also known as a child or client instance, and is sometimes referred to by using the RANU acronym ("run as normal user").</span></span>
