# 连接池
连接到数据源可能需要很长时间。  为了最大程度地降低打开连接的成本，ADO.NET 使用一种称为*连接池* 的优化技术，这种技术可最大程度地降低重复打开和关闭连接所造成的成本。  .NET Framework 数据提供程序处理连接池的方式有所不同。
