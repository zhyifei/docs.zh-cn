# &lt;unrecognizedPolicyAssertion&gt;
表示用于指定策略断言的绑定元素。  此元素没有属性并且以空开关形式存在。  
  
## 语法  
  
```  
  
<unrecognizedPolicyAssertion/>  
```  
  
## 类型  
 `Type`  
  
## 特性和元素  
 下列各节描述了特性、子元素和父元素。  
  
### 特性  
 无。  
  
### 子元素  
 无  
  
### 父元素  
  
|元素|描述|  
|--------|--------|  
|[\<绑定\>](../../../../../docs/framework/misc/binding.md)|定义自定义绑定的所有绑定功能。|
