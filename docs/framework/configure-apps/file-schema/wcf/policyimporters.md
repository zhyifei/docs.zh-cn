# &lt;policyImporters&gt;
此配置元素指定用于控制有关绑定的自定义策略断言的导入的所有策略导入程序。  每个子元素均为一个 \<`policyImporter`\>，可用于搜索有关绑定功能的自定义策略断言，并附加一个实现断言所需功能的自定义绑定元素。
