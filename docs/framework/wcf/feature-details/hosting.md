# 承载
本节中的主题介绍服务承载。服务可以承载于 Internet Information Services \(IIS\)、Windows 进程激活服务 \(WAS\)、Windows Server AppFabric、Windows 服务中，也可以承载于托管应用程序中（这种情况通常称为“自承载”）。  
  
 值得注意的是，从不受信任的主机运行服务或任何扩展会危害安全。
