## <a name="endpoint-addresses"></a><span data-ttu-id="3ae76-121">终结点地址</span><span class="sxs-lookup"><span data-stu-id="3ae76-121">Endpoint Addresses</span></span>

## <a name="ntlm"></a><span data-ttu-id="3ae76-140">NTLM</span><span class="sxs-lookup"><span data-stu-id="3ae76-140">NTLM</span></span>
