## <a name="distributed-processing-and-compute-clusters"></a><span data-ttu-id="8bf91-112">分布式处理和计算群集</span><span class="sxs-lookup"><span data-stu-id="8bf91-112">Distributed Processing and Compute Clusters</span></span>
