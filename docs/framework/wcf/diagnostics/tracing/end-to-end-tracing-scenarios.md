# <a name="end-to-end-tracing-scenarios"></a>端到端跟踪方案
本节包含的主题描述使用跟踪的不同方案。  
  
 使用下面的方法为回复/响应、单向和双工方案实现活动跟踪  
  
- 同步或异步  
  
- BasicHttpBinding、netNamedPipeBinding、netTcpBinding、wsHttpBinding 或 netMsmqBinding  
  
- COM 服务  
  
- 用户活动 ID 传播
