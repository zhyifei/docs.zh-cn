# System.ServiceModel.ServiceOperationMissingReplyContext
System.ServiceModel.ServiceOperationMissingReplyContext  
  
## 说明  
 对于双向操作，ReplyContext 为 Null。跟踪数据显示以下消息：“答复操作没有可用于该答复的 IRequestContext”。  
  
 这种跟踪对于开发过程中的调试非常有用。
