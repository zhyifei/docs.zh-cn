# System.ServiceModel.DroppedAMessage
System.ServiceModel.DroppedAMessage  
  
## 说明  
 收到了响应消息，但没有任何未处理的请求等候此消息。[!INCLUDE[indigo1](../../../../../includes/indigo1-md.md)] 运行时正在丢弃该消息。  
  
 此跟踪可用于调试丢失的消息。
