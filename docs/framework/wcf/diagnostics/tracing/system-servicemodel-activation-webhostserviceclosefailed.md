# System.ServiceModel.Activation.WebHostServiceCloseFailed
当服务无法正常关闭并中止时发生。  
  
## 说明  
 此错误代码仅出现在日志文件中。它通常指示编程错误，例如，在已经调用 Abort 之后尝试关闭服务的时候。  
  
## 疑难解答  
 请检查应用程序源代码。
