# System.ServiceModel.TxCompletionStatusCompletedForTACOSC
在关闭会话时完成了特定的事务。  
  
## 说明  
 在关闭当前活动会话并且 `TransactionAutoCompleteOnSessionClose` 设置为 `true` 时进行了跟踪。
