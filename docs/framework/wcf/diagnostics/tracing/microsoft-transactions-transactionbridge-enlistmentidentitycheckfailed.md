# Microsoft.Transactions.TransactionBridge.EnlistmentIdentityCheckFailed
传入消息发送方的标识与派生自登记的终结点引用的标识不匹配。  
  
## 说明  
 如果消息安全验证失败，则进行跟踪。  
  
## 疑难解答  
 确保为参与该事务的所有计算机正确配置了 Transaction 安全。
