# System.ServiceModel.TxCompletionStatusCompletedForSetComplete
由于调用 SetTransactionComplete 导致指定操作的指定事务完成。  
  
## 说明  
 跟踪用户是否已显式决定通过 OperationContext 来完成当前事务。
