# Microsoft.Transactions.TransactionBridge.ReplayMessageRetry
已向无响应的协调程序发送重播消息重试。  
  
## 描述  
 如果本地事务管理器因未在给定的时间内收到响应而需要向上级协调程序重新发送重播消息，则进行跟踪。  
  
## 疑难解答  
 调查导致响应未及时传送的潜在网络或产品问题。  如果看到很多这样的消息，可能表明基础结构有问题或响应时间异常长。  这两种问题都会明显降低系统中事务的吞吐量。
