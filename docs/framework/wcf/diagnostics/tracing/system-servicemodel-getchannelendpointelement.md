# System.ServiceModel.GetChannelEndpointElement
System.ServiceModel.GetChannelEndpointElement  
  
## 描述  
 在构造通道工厂时在信息级别发出此跟踪。  此跟踪提供与客户端通信的终结点的说明（例如，远程地址、绑定、协定名称）。
