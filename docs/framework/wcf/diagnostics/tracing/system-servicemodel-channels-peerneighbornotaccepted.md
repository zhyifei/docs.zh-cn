# System.ServiceModel.Channels.PeerNeighborNotAccepted
未接受对等邻居。  
  
## 说明  
 处理与邻居的连接时会出现此跟踪。可能的原因包括安全设置不匹配、超时、邻居不能接受连接或存在暂时性的网络问题。  
  
## 疑难解答  
 检查是否有不匹配的安全设置、是否超时、邻居是否不能接受连接或是否存在暂时性的网络问题。
