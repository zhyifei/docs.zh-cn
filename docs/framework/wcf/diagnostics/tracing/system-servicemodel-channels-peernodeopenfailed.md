# System.ServiceModel.Channels.PeerNodeOpenFailed
尝试打开邻居连接失败。  
  
## 描述  
 此跟踪在尝试建立邻居连接后发生。  可能的原因包括无响应的解析程序、不正确的绑定设置或暂时的网络故障。  
  
## 疑难解答  
 确保这两个节点均有有效的地址并正确进行了配置。
