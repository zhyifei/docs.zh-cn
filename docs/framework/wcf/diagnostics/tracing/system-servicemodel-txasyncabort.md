# System.ServiceModel.TxAsyncAbort
指定的事务被异步中止。  
  
## 描述  
 由于另一名参与者赞成中止，发生超时，或事务的参与者内部出现另一错误，因此当前事务被中止。  
  
## 疑难解答  
 如果该中止是意料之外的，请检查所有系统日志，以便确定发生该中止的真正原因。
