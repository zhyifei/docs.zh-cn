# System.ServiceModel.Channels.IncompatibleExistingTransportManager
发现指定的 URI 存在不兼容的传输管理器。  
  
## 说明  
 此信息跟踪指示已为要打开的侦听器找到匹配的注册，但是，现有的传输管理器不兼容。侦听器将尝试使用其他传输管理器，如果该尝试失败，则会引发异常。
