# System.ServiceModel.TxSourceTxScopeRequiredIsAttachedTransaction
从 InstanceContext 事务收到了指定操作的指定事务。  
  
## 说明  
 如果用户代码可用的事务被在此调用之前调用的 AutoComplete\=false 方法遗留，则进行跟踪。
