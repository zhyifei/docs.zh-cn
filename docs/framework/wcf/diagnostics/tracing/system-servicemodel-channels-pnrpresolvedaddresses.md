# System.ServiceModel.Channels.PnrpResolvedAddresses
Pnrp 解析程序完成了一个解析操作。  
  
## 说明  
 此警告跟踪指示 Pnrp 解析程序未能找到任何地址。此跟踪在 PNRP 解析程序执行解析操作期间出现。
