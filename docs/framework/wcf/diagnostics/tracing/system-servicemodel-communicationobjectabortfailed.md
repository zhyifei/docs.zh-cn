# System.ServiceModel.CommunicationObjectAbortFailed
通信对象中止失败。  
  
## 描述  
 通信对象中止失败，原因是在其 Abort 方法的执行过程中引发了异常。  另一个邻近的跟踪可能包含有关异常的详细信息。
