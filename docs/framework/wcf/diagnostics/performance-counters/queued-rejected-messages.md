# Queued Rejected Messages（已排队的被拒绝消息数）
计数器名称：Queued Messages Rejected（已排队的被拒绝消息数）。  
  
## 说明  
 此服务中被排队传输拒绝的消息的数目。  
  
 有关何时消息被拒绝的更多信息，请参见 [Poison\-Message Handling](http://go.microsoft.com/fwlink/?LinkID=96233)（病毒消息处理）。
