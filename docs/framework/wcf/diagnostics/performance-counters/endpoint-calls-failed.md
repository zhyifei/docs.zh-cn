# <a name="endpoint-calls-failed"></a>终结点：Calls Failed（失败的调用次数）
计数器名称：Calls Failed（失败的调用次数）。  
  
## <a name="description"></a>描述  
 在此终结点上有未处理异常的调用的数量。 每当此终结点处有未处理的异常时，此计数器就会增加。
