# Queue Dropped Messages（队列丢弃的消息数）
计数器名称：Queued Messages Dropped（已丢弃的排队消息数）。  
  
## 说明  
 此服务中由排队传输丢弃的消息数。  
  
 有关何时丢弃消息的更多信息，请参见 [Poison\-Message Handling](http://go.microsoft.com/fwlink/?LinkId=96233)（病毒消息处理）。
