# COM+ 集成
本主题列出由 COM\+ 集成生成的所有异常。  
  
## 异常列表  
  
|资源代码|资源字符串|  
|----------|-----------|  
|CannotResolveTypeForParamInMessageDescription|无法解析指定命名空间中的指定参数的类型。|  
|NoEqualSignFound|指定关键字的后面没有等号。请确保每个关键字的后面都有一个等号和一个值。|  
|UdtNotFoundInAssembly|无法找到指定的用户定义类型。请确保注册和指定了正确的类型和类型库。|
