# MessageEncodingBindingElement
MessageEncodingBindingElement  
  
## 语法  
 `class MessageEncodingBindingElement : BindingElement`  
  
 `{`  
  
 `string MessageVersion;`  
  
 `};`  
  
## 方法  
 MessageEncodingBindingElement 类不定义任何方法。  
  
## 属性  
 MessageEncodingBindingElement 类具有以下属性：  
  
### MessageVersion  
 数据类型：String  
  
 访问类型：只读  
  
 使用绑定发送的 SOAP 版本的消息。
