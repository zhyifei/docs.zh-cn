# ChannelPoolSettings
ChannelPoolSettings  
  
## 语法  
  
```  
class ChannelPoolSettings  
{  
  datetime IdleTimeout;  
  datetime LeaseTimeout;  
  sint32 MaxOutboundChannelsPerEndpoint;  
};  
```  
  
## 方法  
 ChannelPoolSettings 类未定义任何方法。  
  
## 属性  
 ChannelPoolSettings 类具有下列属性：  
  
### IdleTimeout  
 数据类型：DateTime  
  
 访问类型：只读  
  
 连接在断开前可以空闲的最长时间。  
  
### LeaseTimeout  
 数据类型：DateTime  
  
 访问类型：只读  
  
 超时前完成租约操作的最大时间。  
  
### MaxOutboundChannelsPerEndpoint  
 数据类型：sint32  
  
 访问类型：只读  
  
 每个终结点的最大出站通道数。
