# GetOperationCounterInstanceName
检索操作性能计数器实例名称。  
  
## 语法  
  
```  
string GetOperationCounterInstanceName(  
  string Operation  
);  
```  
  
## 参数  
  
### Operation  
 \[in\] string  
  
## 要求  
  
|MOF|在 Servicemodel.mof 中声明。|  
|---------|-----------------------------|  
|命名空间|已在 root\\ServiceModel 中定义|
