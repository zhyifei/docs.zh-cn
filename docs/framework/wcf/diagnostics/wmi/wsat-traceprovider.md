# WSAT_TraceProvider
WSAT\_TraceProvider  
  
## 语法  
  
```  
class WSAT_TraceProvider : EventTrace  
{  
};  
```  
  
## 方法  
 WSAT\_TraceProvider 类从 EventTrace 继承而来，但是不会定义其他方法或属性。  
  
## 要求  
  
|MOF|已在 Servicemodel.mof 中声明。|  
|---------|------------------------------|  
|命名空间|已在 root\\ServiceModel 中定义|
