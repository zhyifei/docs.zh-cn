# ServiceModel 注册工具
ServiceModel 注册工具是一个命令行工具，可用于管理 ServiceModel 在单个计算机上的注册。有关此工具的更多信息，请参见 [ServiceModel 注册工具 \(ServiceModelReg.exe\)](../../../../docs/framework/wcf/servicemodelreg-exe.md)。
