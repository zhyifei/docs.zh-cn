# <a name="wmiexecqueryfailed"></a>WmiExecQueryFailed
ID:124  
  
 严重性：Error  
  
 类别：ServiceModel  
  
## <a name="description"></a>描述  
 此事件指示未处理 WMI ExecQuery 请求。 此事件列出了异常、进程名称和进程 ID。
