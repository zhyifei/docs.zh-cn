# ProtocolRecoveryBeginningFailure
Id：144  
  
 严重性：错误  
  
 类别：TransactionBridge  
  
## 说明  
 此事件指示 MSDTC WS\-AtomicTransaction 协议在恢复过程开始时即已失败。因此，禁用 WS\-AtomicTransaction 功能。此事件将列出协议 ID、协议名称、异常、进程名称和进程 ID。
