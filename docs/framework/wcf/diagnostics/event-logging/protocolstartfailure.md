# ProtocolStartFailure
ID：143  
  
 严重性：错误  
  
 类别：TransactionBridge  
  
## 描述  
 此事件指示 WS\-AtomicTransaction 协议服务启动失败。  因此，禁用 WS\-AtomicTransaction 功能。  此事件将列出协议 ID、协议名称、异常、进程名称和进程 ID。
