# WmiDeleteInstanceFailed
Id：122  
  
 严重性：错误  
  
 类别：ServiceModel  
  
## 说明  
 此事件指示未处理 WMI DeleteInstance 请求。此事件列出了异常、进程名称和进程 ID。
