# LAFailedToListenForApp
ID：160  
  
 严重性：错误  
  
 类别：SMSvcHost  
  
## 说明  
 此事件指示试图侦听特定站点时指定协议的指定激活服务中出错，因此将为该站点临时禁用该协议。该事件将列出 URL、状态、异常、进程名称和进程 ID。
