# SslNoPrivateKey
Id：154  
  
 严重性：错误  
  
 类别：TransactionBridge  
  
## 说明  
 此事件指示带有特定主题名称和指纹的标识证书不具有私钥。此事件将列出进程名称和进程 ID。
