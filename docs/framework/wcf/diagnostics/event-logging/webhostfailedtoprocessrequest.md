# <a name="webhostfailedtoprocessrequest"></a>WebHostFailedToProcessRequest
ID:111  
  
 对应Error  
  
 类别ServiceModel  
  
## <a name="description"></a>描述  
 此事件指示 WebHost 无法处理请求。 此事件列出了发送方信息、异常、进程名称和进程 ID。
