# <a name="wmiexecmethodfailed"></a>WmiExecMethodFailed
ID:125  
  
 严重性：Error  
  
 类别：ServiceModel  
  
## <a name="description"></a>描述  
 此事件指示尚未处理 WMI ExecMethod 请求。 此事件列出了异常、进程名称和进程 ID。
