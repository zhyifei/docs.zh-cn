# 603 - AspNetRoutingService
## 属性  
  
|||  
|-|-|

|关键字|RoutingServices|  
|级别|详细|  
|通道|Microsoft\-Windows\-应用程序服务器\-应用程序\/调试|  
  
## 描述  
 当传入请求与在带有地址 %1 的 ASP.NET 路由中定义的 WCF 服务匹配时，发出此事件。  
  
## 消息  
 传入请求与带有地址 %1 的 ASP.NET 路由中定义的某个 WCF 服务匹配。
