# 4806 - DiscoveryMessageWithInvalidRelatesToOrOperationCompleted
## 属性  
  
|||  
|-|-|  
|ID|4806|  
|关键字|发现|  
|级别|警告|  
|通道|Microsoft\-Windows\-应用程序服务器\-应用程序\/调试|  
  
## 描述  
 当因为相应的操作已完成或 relatesTo 值无效而导致发现消息已被 DiscoveryClient 丢弃时，发出此事件。  
  
## 消息  
 messageId 为“%2”且 relatesTo 为“%3”的 %1 消息已被 DiscoveryClient 丢弃，因为相应的 %4 操作已完成或 relatesTo 值无效。
