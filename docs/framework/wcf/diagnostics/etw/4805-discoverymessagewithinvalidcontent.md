# 4805 - DiscoveryMessageWithInvalidContent
## 属性  
  
|||  
|-|-|  
|ID|4805|  
|关键字|发现|  
|级别|警告|  
|通道|Microsoft\-Windows\-应用程序服务器\-应用程序\/调试|  
  
## 描述  
 当发现消息包含无效内容时，将发出此事件。  
  
## 消息  
 messageId 为“%2”的 %1 消息由于具有无效内容而被丢弃。
