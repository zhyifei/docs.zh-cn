# 4809 - DiscoveryMessageWithNullMessageId
## 属性  
  
|||  
|-|-|  
|ID|4809|  
|关键字|发现|  
|级别|警告|  
|通道|Microsoft\-Windows\-应用程序服务器\-应用程序\/调试|  
  
## 描述  
 在发现消息由于消息标头不包含所需的 MessageId 属性而被丢弃时，将发出此事件。  
  
## 消息  
 %1 消息由于消息标头不包含所需的 MessageId 属性而被丢弃。
