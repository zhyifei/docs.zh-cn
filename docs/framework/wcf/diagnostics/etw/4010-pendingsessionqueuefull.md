# 4010 - PendingSessionQueueFull
## 属性  
  
|||  
|-|-|  
|ID|4010|  
|关键字|ActivationServices|  
|级别|警告|  
|通道|Microsoft\-Windows\-应用程序服务器\-应用程序\/分析|  
  
## 描述  
 当会话调度因为挂起的会话队列已满而失败时，发出此事件。  
  
## 消息  
 对“%1”的会话调度失败，因为挂起的会话队列已满，其中包含“%2”个挂起项。
