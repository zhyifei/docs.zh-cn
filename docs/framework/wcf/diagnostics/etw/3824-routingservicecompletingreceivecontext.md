# 3824 - RoutingServiceCompletingReceiveContext
## 属性  
  
|||  
|-|-|  
|ID|3824|  
|关键字|RoutingServices|  
|级别|详细|  
|通道|Microsoft\-Windows\-应用程序服务器\-应用程序\/调试|  
  
## 描述  
 当路由服务针对具有指定 ID 的消息调用 ReceiveContext.Complete 时发出此事件。  
  
## 消息  
 路由服务正在为 ID 是“%1”的消息调用 ReceiveContext.Complete。
