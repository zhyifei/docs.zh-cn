# 4811 - DiscoveryMessageWithNullRelatesTo
## 属性  
  
|||  
|-|-|  
|ID|4811|  
|关键字|发现|  
|级别|警告|  
|通道|Microsoft\-Windows\-应用程序服务器\-应用程序\/调试|  
  
## 描述  
 在发现消息由于消息标头不包含所需的 RelatesTo 属性而被 DiscoveryClient 丢弃时，将发出此事件。  
  
## 消息  
 messageId 为“%2”的 %1 消息由于消息标头不包含所需的 RelatesTo 属性而被 DiscoveryClient 丢弃。
