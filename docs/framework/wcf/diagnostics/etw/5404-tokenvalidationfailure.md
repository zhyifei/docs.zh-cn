# 5404 - TokenValidationFailure
## 属性  
  
|||  
|-|-|  
|ID|5404|  
|关键字|安全性|  
|级别|错误|  
|通道|Microsoft\-Windows\-应用程序服务器\-应用程序\/调试|  
  
## 描述  
 当 SecurityToken 验证失败时，将发出此事件。  
  
## 消息  
 SecurityToken（类型为“%1”，ID 为“%2”）验证已失败。  %3
