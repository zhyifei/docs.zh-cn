# 3821 - RoutingServiceCloseFailed
## 属性  
  
|||  
|-|-|  
|ID|3821|  
|关键字|RoutingServices|  
|级别|警告|  
|通道|Microsoft\-Windows\-应用程序服务器\-应用程序\/调试|  
  
## 描述  
 当路由服务无法关闭出站客户端时，将发出此事件。  
  
## 消息  
 关闭出站客户端“%1”时，路由服务失败。
