## <a name="partially-qualified-assembly-names-and-side-by-side-execution"></a><span data-ttu-id="4b706-188">部分限定的程序集名称和并行执行</span><span class="sxs-lookup"><span data-stu-id="4b706-188">Partially Qualified Assembly Names and Side-by-Side Execution</span></span>
