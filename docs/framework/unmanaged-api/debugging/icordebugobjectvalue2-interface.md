# <a name="icordebugobjectvalue2-interface"></a>ICorDebugObjectValue2 接口

|方法|描述|  
|------------|-----------------|  
|[GetVirtualMethodAndType 方法](icordebugobjectvalue2-getvirtualmethodandtype-method.md)|`GetVirtualMethodAndType`尚未实现。|
