# <a name="icordebugchainismanaged-method"></a>ICorDebugChain::IsManaged 方法
获取一个值，该值指示此链是否正在运行托管的代码。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT IsManaged (  
    [out] BOOL               *pManaged  
);  
```  
  
#### <a name="parameters"></a>参数  
 `pManaged`  
 [out]`true`如果此证书链正在运行托管的代码; 否则为`false`。
