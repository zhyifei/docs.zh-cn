# <a name="icordebugcodegetaddress-method"></a>ICorDebugCode::GetAddress 方法
获取此"ICorDebugCode"接口表示代码段的相对虚拟地址 (RVA)。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT GetAddress (  
    [out] CORDB_ADDRESS *pStart  
);  
```  
  
#### <a name="parameters"></a>参数  
 `pStart`  
 [out]一个指向代码段的 RVA。
