# <a name="icordebugvalueenumnext-method"></a>ICorDebugValueEnum::Next 方法
从当前位置开始，从枚举中获取指定的 "ICorDebugValue" 实例数。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT Next (  
    [in]  ULONG  celt,  
    [out, size_is(celt), length_is(*pceltFetched)]  
        ICorDebugValue *values[],  
    [out] ULONG *pceltFetched  
);  
```  
  
## <a name="parameters"></a>参数  
 `celt`  
 中`ICorDebugValue`要检索的实例数。  
  
 `values`  
 弄指针的数组，其中每个都指向一个 `ICorDebugValue` 对象。  
  
 `pceltFetched`  
 弄一个指针，指向 `ICorDebugValue` 实际返回的实例数。 如果为1，则此值可以为 null `celt` 。
