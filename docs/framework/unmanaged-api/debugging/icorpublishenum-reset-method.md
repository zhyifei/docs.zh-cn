# <a name="icorpublishenumreset-method"></a>ICorPublishEnum::Reset 方法
将光标移到枚举的开头。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT Reset ();  
```  
  
#### <a name="parameters"></a>参数  
 `Reset` 不带任何参数。
