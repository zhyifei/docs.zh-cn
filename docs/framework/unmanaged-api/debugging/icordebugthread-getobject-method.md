# <a name="icordebugthreadgetobject-method"></a>ICorDebugThread::GetObject 方法
获取公共语言运行时 (CLR) 线程的接口指针。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT GetObject (  
    [out] ICorDebugValue   **ppObject  
);  
```  
  
#### <a name="parameters"></a>参数  
 `ppObject`  
 [out]指向表示 CLR 线程 ICorDebugValue 接口对象的地址的指针。
