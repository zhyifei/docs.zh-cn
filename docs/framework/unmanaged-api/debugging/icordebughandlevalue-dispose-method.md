# <a name="icordebughandlevaluedispose-method"></a>ICorDebugHandleValue::Dispose 方法
释放此 ICorDebugHandleValue 对象而无需显式地释放接口指针引用的句柄。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT Dispose ();  
```
