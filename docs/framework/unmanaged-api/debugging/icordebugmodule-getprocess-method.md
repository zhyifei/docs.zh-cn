# <a name="icordebugmodulegetprocess-method"></a>ICorDebugModule::GetProcess 方法
获取此模块的包含进程。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT GetProcess (  
    [out] ICorDebugProcess **ppProcess  
);  
```  
  
## <a name="parameters"></a>参数  
 `ppProcess`  
 弄指向 ICorDebugProcess 对象的地址的指针，该对象表示包含此模块的进程。
