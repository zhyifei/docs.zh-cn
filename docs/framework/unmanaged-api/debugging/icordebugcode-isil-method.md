# <a name="icordebugcodeisil-method"></a>ICorDebugCode::IsIL 方法
获取一个值，该值指示此"icor 调试代码"是表示在 Microsoft 中间语言 (MSIL) 中编译的代码。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT IsIL (  
    [out] BOOL       *pbIL  
);  
```  
  
#### <a name="parameters"></a>参数  
 `pbIL`  
 [out]`true`如果此`ICorDebugCode`表示代码编译在 MSIL 中; 否则为`false`。
