# <a name="icordebugarrayvaluegetrank-method"></a>ICorDebugArrayValue::GetRank 方法
获取数组中的维度数。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT GetRank (  
    [out] ULONG32   *pnRank  
);  
```  
  
## <a name="parameters"></a>参数  
 `pnRank`  
 [out]指向在此维度的数目的`ICorDebugArrayValue`对象。
