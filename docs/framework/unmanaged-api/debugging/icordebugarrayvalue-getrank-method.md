# <a name="icordebugarrayvaluegetrank-method"></a>ICorDebugArrayValue::GetRank 方法
获取数组中的维度数。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT GetRank (  
    [out] ULONG32   *pnRank  
);  
```  
  
## <a name="parameters"></a>参数  
 `pnRank`  
 弄一个指针，指向该`ICorDebugArrayValue`对象中的维度数。
