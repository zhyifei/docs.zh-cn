# <a name="icordebugvalueenum-interface1"></a>ICorDebugValueEnum 接口 1
实现"ICorDebugEnum"方法，并枚举"ICorDebugValue"数组。  
  
## <a name="methods"></a>方法  
  
|方法|描述|  
|------------|-----------------|  
|[Next 方法](../../../../docs/framework/unmanaged-api/debugging/icordebugvalueenum-next-method.md)|获取指定的数目的`ICorDebugValue`枚举，从当前位置开始中的实例。|  
  
## <a name="remarks"></a>备注  
  
> [!NOTE]
>  此接口不支持跨计算机或跨进程远程调用。
