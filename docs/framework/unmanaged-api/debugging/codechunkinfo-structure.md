# <a name="codechunkinfo-structure"></a>CodeChunkInfo 结构

表示内存中的单一代码块。  
  
## <a name="syntax"></a>语法  
  
```cpp  
typedef struct _CodeChunkInfo {  
    CORDB_ADDRESS startAddr;  
    ULONG32       length;  
} CodeChunkInfo;  
```  
  
## <a name="members"></a>成员  
  
|成员|描述|  
|------------|-----------------|  
|`startAddr`|一个`CORDB_ADDRESS`值，该值指定块区的起始地址。|  
|`length`|块区的大小（以字节为单位）。|  
  
## <a name="remarks"></a>备注  
 单个代码块是本机代码区域，它是代码对象（例如函数）的一部分。
