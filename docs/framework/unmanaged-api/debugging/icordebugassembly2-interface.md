# <a name="icordebugassembly2-interface"></a>ICorDebugAssembly2 接口

表示一个程序集。 此接口是 ICorDebugAssembly 接口的扩展。  
  
## <a name="methods"></a>方法  
  
|方法|说明|  
|------------|-----------------|  
|[IsFullyTrusted 方法](icordebugassembly2-isfullytrusted-method.md)|获取一个值，该值指示运行时安全系统是否已向程序集授予完全信任。|  
  
## <a name="remarks"></a>备注  
  
> [!NOTE]
> 此接口不支持跨计算机或跨进程远程调用。
