# <a name="icordebugmodulegetassembly-method"></a>ICorDebugModule::GetAssembly 方法
获取此模块包含程序集。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT GetAssembly(  
    [out] ICorDebugAssembly **ppAssembly  
);  
```  
  
#### <a name="parameters"></a>参数  
 `ppAssembly`  
 [out]指向一个表示包含此模块的程序集的 icor 调试程序集对象的指针。
