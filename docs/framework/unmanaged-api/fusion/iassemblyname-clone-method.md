# <a name="iassemblynameclone-method"></a>IAssemblyName::Clone 方法

## <a name="syntax"></a>语法  
  
```  
HRESULT Clone (  
    [out] IAssemblyName **pName  
);  
```  
  
#### <a name="parameters"></a>参数  
 `pName`  
 [out]此返回的副本`IAssemblyName`对象。
