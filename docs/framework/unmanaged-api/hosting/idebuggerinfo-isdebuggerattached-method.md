# <a name="idebuggerinfoisdebuggerattached-method"></a>IDebuggerInfo::IsDebuggerAttached 方法
获取一个值，该值指示是否有托管的调试器附加到此进程。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT IsDebuggerAttached (  
    [out] BOOL *pbAttached  
);  
```  
  
## <a name="parameters"></a>参数  
 `pbAttached`  
 [out]指向一个值，则该值`true`如果托管的调试器附加到进程; 否则为`false`。
