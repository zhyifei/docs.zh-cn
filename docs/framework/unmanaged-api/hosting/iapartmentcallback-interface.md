# <a name="iapartmentcallback-interface"></a>IApartmentCallback 接口
提供用于在单元中进行回调的方法。 *单元*是进程内共享相同线程访问要求的对象的逻辑容器。  
  
## <a name="methods"></a>方法  
  
|方法|说明|  
|------------|-----------------|  
|[DoCallback 方法](iapartmentcallback-docallback-method.md)|在单元内执行指定的函数。|
