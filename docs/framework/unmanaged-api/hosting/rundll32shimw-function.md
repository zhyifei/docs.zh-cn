# <a name="rundll32shimw-function"></a>RunDll32ShimW 函数
执行指定的命令。  
  
 此函数中不推荐[!INCLUDE[net_v40_long](../../../../includes/net-v40-long-md.md)]。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT RunDll32ShimW (  
    [in] HWND        hwnd,  
    [in] HINSTANCE   hinst,  
    [in] LPCWSTR     lpszCmdLine,  
    [in] int         nCmdShow  
);  
```  
  
## <a name="parameters"></a>参数  
 `hwnd`  
 [in]将在其中显示命令输出窗口的句柄。  
  
 `hinst`  
 [in]包含该命令的库句柄。  
  
 `lpszCmdLine`  
 [in]一个字符串，指定要执行的命令。  
  
 `nCmdShow`  
 [in]一个整数，指定的输出窗口的显示模式。
