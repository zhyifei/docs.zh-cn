# <a name="isymunmanagedwriter3commit-method"></a>ISymUnmanagedWriter3::Commit 方法
提交到目前为止写入到流的更改。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT Commit();  
```  
  
## <a name="return-value"></a>返回值  
 如果该方法成功; 则为 S_OK否则为 E_FAIL 或某些其他错误代码。  
  
## <a name="requirements"></a>惠?  
 **标头：** CorSym.idl、 CorSym.h
