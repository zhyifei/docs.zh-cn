|方法|描述|  
|------------|-----------------|  
|[SetCheckSum 方法](../../../../docs/framework/unmanaged-api/diagnostics/isymunmanageddocumentwriter-setchecksum-method.md)|设置校验和信息。|  
|[SetSource 方法](../../../../docs/framework/unmanaged-api/diagnostics/isymunmanageddocumentwriter-setsource-method.md)|正在写入的文档集将嵌入的源。|
