# <a name="imetadataimportisvalidtoken-method"></a>IMetaDataImport::IsValidToken 方法
获取指示指定的标记是否包含对代码对象的有效引用的值。  
  
## <a name="syntax"></a>语法  
  
```  
BOOL IsValidToken (  
   [in] mdToken     tk  
);  
```  
  
## <a name="parameters"></a>参数  
 `tk`  
 [in]要检查有关引用有效性的标记。  
  
## <a name="return-value"></a>返回值  
 `true` 如果`tk`是当前作用域内的有效的元数据标记。 否则为 `false`。
