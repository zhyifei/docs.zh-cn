# <a name="imetadataemitdefinememberref-method"></a>IMetaDataEmit::DefineMemberRef 方法
定义对当前范围之外的模块成员的引用，并获取该引用定义的标记。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT DefineMemberRef (   
    [in]  mdToken           tkImport,   
    [in]  LPCWSTR           szName,   
    [in]  PCCOR_SIGNATURE   pvSigBlob,   
    [in]  ULONG             cbSigBlob,   
    [out] mdMemberRef       *pmr   
);  
```  
  
## <a name="parameters"></a>参数  
 `tkImport`  
 中目标成员的类或接口的标记（如果成员不是全局成员）;如果成员是全局成员，则为该其他文件的 `mdModuleRef` 标记。  
  
 `szName`  
 中目标成员的名称。  
  
 `pvSigBlob`  
 中目标成员的签名。  
  
 `cbSigBlob`  
 中`pvSigBlob`中的字节计数。  
  
 `pmr`  
 弄分配 `mdMemberRef` 标记。
