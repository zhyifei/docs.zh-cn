# <a name="imetadataemitdeletetoken-method"></a>IMetaDataEmit::DeleteToken 方法
从当前元数据作用域中删除指定的令牌。  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT DeleteToken (
    [in]  mdToken     tkObj
);  
```  
  
## <a name="parameters"></a>parameters  
 `tkObj`  
 [在]要删除的令牌。
