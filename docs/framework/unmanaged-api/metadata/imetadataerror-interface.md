# <a name="imetadataerror-interface"></a>IMetaDataError 接口
提供回调机制，用于在元数据合并期间报告错误。  
  
> [!NOTE]
>  `IMetaDataError`接口必须由客户端实现。  
  
## <a name="methods"></a>方法  
  
|方法|描述|  
|------------|-----------------|  
|[OnError 方法](../../../../docs/framework/unmanaged-api/metadata/imetadataerror-onerror-method.md)|提供元数据合并过程中发生的错误的通知。|
