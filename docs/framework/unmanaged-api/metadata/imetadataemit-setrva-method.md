# <a name="imetadataemitsetrva-method"></a>IMetaDataEmit::SetRVA 方法
设置指定的方法的相对虚拟地址。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT SetRVA (  
    [in]  mdMethodDef  md,   
    [in]  ULONG        ulRVA   
);  
```  
  
## <a name="parameters"></a>参数  
 `md`  
 [in]目标方法或方法的实现标记。  
  
 `ulRVA`  
 [in]在代码或数据区域的地址。
