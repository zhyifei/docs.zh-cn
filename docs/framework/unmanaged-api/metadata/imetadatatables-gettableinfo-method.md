# <a name="imetadatatablesgettableinfo-method"></a>IMetaDataTables::GetTableInfo 方法
获取名称、 行大小、 行数、 列数和指定的表的键列索引。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT GetTableInfo (  
    [in]  ULONG       ixTbl,  
    [out] ULONG       *pcbRow,  
    [out] ULONG       *pcRows,  
    [out] ULONG       *pcCols,  
    [out] ULONG       *piKey,  
    [out] const char  **ppName  
);  
```  
  
## <a name="parameters"></a>参数  
 `ixTbl`  
 [in]表的标识符要返回其属性。  
  
 `pcbRow`  
 [out]指向大小 （字节） 的表行的指针。  
  
 `pcRows`  
 [out]指向表中的行数的指针。  
  
 `pcCols`  
 [out]指向表中的列数的指针。  
  
 `piKey`  
 [out]索引键列，则为-1 如果表没有键列的指针。  
  
 `ppName`  
 [out]为表名的指针指向的指针。
