# <a name="iceegenemitstring-method"></a>ICeeGen::EmitString 方法
将指定的字符串发出到基本代码。  
  
 此方法已过时，不应使用。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT EmitString (  
    [in]  LPWSTR    lpString,  
    [out] ULONG     *RVA  
);  
```  
  
#### <a name="parameters"></a>参数  
 `lpString`  
 [in]要发出的字符串。  
  
 `RVA`  
 [out]发出的字符串的相对虚拟地址。
