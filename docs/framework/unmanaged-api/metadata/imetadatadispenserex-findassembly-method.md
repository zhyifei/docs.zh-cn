# <a name="imetadatadispenserexfindassembly-method"></a>IMetaDataDispenserEx::FindAssembly 方法
未实现此方法。 If called, it returns E_NOTIMPL.  
  
## <a name="syntax"></a>语法  
  
```cpp  
HRESULT FindAssembly(  
    [in]  LPCWSTR  szAppBase,  
    [in]  LPCWSTR  szPrivateBin,  
    [in]  LPCWSTR  szGlobalBin,  
    [in]  LPCWSTR  szAssemblyName,  
    [out] LPCWSTR  szName,  
    [in]  ULONG    cchName,  
    [out] ULONG    *pcName  
);  
```  
  
## <a name="parameters"></a>参数  
 `szAppBase`  
 [in] Not used.  
  
 `szPrivateBin`  
 [in] Not used.  
  
 `szGlobalBin`  
 [in] Not used.  
  
 `szAssemblyName`  
 [in] The assembly to be found.  
  
 `szName`  
 [out] The simple name of the assembly.  
  
 `cchName`  
 [in] The size, in bytes, of `szName`.  
  
 `pcName`  
 [out] The number of characters actually returned in `szName`.
