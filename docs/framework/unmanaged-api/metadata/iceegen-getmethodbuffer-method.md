# <a name="iceegengetmethodbuffer-method"></a>ICeeGen::GetMethodBuffer 方法
获取在指定的相对虚拟地址的方法的适当大小的缓冲区。  
  
 此方法已过时，不应使用。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT GetMethodBuffer (  
    [in]  ULONG       RVA,  
    [out] UCHAR       **lpBuffer  
);  
```  
  
#### <a name="parameters"></a>参数  
 `RVA`  
 [in]要为其返回缓冲区的方法相对虚拟地址。  
  
 `lpBuffer`  
 [out]指向返回的缓冲区的指针。
