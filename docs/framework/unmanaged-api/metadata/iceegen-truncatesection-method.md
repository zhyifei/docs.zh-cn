# <a name="iceegentruncatesection-method"></a>ICeeGen::TruncateSection 方法
通过指定长度截断指定的代码段。  
  
 此方法已过时，不应使用。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT TruncateSection (  
    [in]  HCEESECTION     section,  
    [in]  ULONG           len  
);  
```  
  
#### <a name="parameters"></a>参数  
 `section`  
 [in]要截断的部分。  
  
 `len`  
 [in]长度 （以字节为单位，用来截断部分）。  
  
## <a name="remarks"></a>备注  
 调用`TruncateSection`仅当你有特殊的段中要求未由其他方法。
