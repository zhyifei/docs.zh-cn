# <a name="imetadataemitsavetostream-method"></a>IMetaDataEmit::SaveToStream 方法
将所有元数据保存在当前范围内指定`IStream`。  
  
## <a name="syntax"></a>语法  
  
```  
HRESULT SaveToStream (   
    [in]  IStream     *pIStream,  
    [in]  DWORD       dwSaveFlags  
);  
```  
  
#### <a name="parameters"></a>参数  
 `pIStream`  
 [in]要将保存到的可写流。  
  
 `dwSaveFlags`  
 [in]保留。 必须为零。
