`Connection`类分析服务器响应，队列请求，请求管道。

## <a name="syntax"></a>语法
  
```csharp

```

> `Connection`类内部使用并且不应在代码中直接使用。

> 在生产应用程序在任何情况下，Microsoft 不支持此类使用。
