# <a name="enumerations-overview-visual-basic"></a>枚举概述 (Visual Basic)
枚举提供一种简便方式，以使用集相关的常数的并将与名称关联常量值。 例如，可以为一组与星期几相关联的整数常量声明一个枚举，然后在代码中使用星期几的名称而不是整数值。
