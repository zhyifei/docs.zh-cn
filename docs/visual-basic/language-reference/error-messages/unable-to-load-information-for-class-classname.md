# <a name="unable-to-load-information-for-class-classname"></a>无法加载类 "\<classname >" 的信息
对不提供的类进行了引用。  
  
 **错误 ID：** BC30712  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1. 验证是否定义了类，并且是否正确拼写了名称。  
  
2. 尝试访问该模块中声明的其中一个成员。 在某些情况下，调试环境找不到成员，因为尚未加载在其中声明成员的模块。
