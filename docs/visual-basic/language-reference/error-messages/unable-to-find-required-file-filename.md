# <a name="unable-to-find-required-file-filename"></a>找不到所需的文件 "\<filename >"
Visual Studio 所需的文件丢失或已损坏。  
  
 **错误 ID：** BC30655  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 重新安装 Visual Studio。
