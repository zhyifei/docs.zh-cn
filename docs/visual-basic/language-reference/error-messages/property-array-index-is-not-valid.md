# <a name="property-array-index-is-not-valid"></a>属性数组索引无效
提供的值的属性数组索引无效。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   请查看组件的文档，请确保你的索引的指定属性的有效范围内。
