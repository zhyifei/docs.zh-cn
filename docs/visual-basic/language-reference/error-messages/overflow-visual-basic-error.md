# <a name="overflow-visual-basic-error"></a>溢出（Visual Basic 错误）
文本表示的值超出所分配的数据类型的限制。  
  
 **错误 ID:** BC30036  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 目标数据类型和重写该文本，使其符合该范围，请查阅其值的范围。
