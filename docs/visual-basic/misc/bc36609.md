# <a name="query-expressions-are-not-valid-in-debug-windows"></a>查询表达式在调试窗口中无效
调试会话期间在“监视”  或“即时”  窗口中输入了 LINQ 查询。 查询在调试窗口中不可用。  
  
 **错误 ID:** BC36609
