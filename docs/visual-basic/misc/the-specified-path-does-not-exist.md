# <a name="the-specified-path-does-not-exist"></a>指定的路径不存在
指定的路径不存在，可能是由于组合字符串时发生了错误。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   请确保路径正确无误。
