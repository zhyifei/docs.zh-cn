<span data-ttu-id="dc26b-103">一个 `Continue` 语句后面必须接 `Do`、 `For`或 `While`，这取决于 `Continue` 语句是否出现在 `Do...Loop` 循环、 `For...Next` 循环或是 `While...End While` 循环中。</span><span class="sxs-lookup"><span data-stu-id="dc26b-103">A `Continue` statement must be followed by `Do`, `For`, or `While`, depending on if the `Continue` statement appears within a `Do...Loop` loop, `For...Next` loop, or `While...End While` loop.</span></span>  
  
 <span data-ttu-id="dc26b-104">**错误 ID:** BC30781</span><span class="sxs-lookup"><span data-stu-id="dc26b-104">**Error ID:** BC30781</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="dc26b-105">更正此错误</span><span class="sxs-lookup"><span data-stu-id="dc26b-105">To correct this error</span></span>
