# <a name="39sub-new39-cannot-handle-events"></a>&#39;Sub New&#39;无法处理事件
你试图将 `Sub New` 与 `Handles`组合在一起，这样做无效。 在过程声明的结尾使用 `Handles` 关键字，以使其处理由使用 `WithEvents` 关键字声明的对象变量引发的事件。  
  
 **错误 ID：** BC30497  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  请不要在此上下文中使用 `New` 。
