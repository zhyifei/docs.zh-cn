# <a name="static-local-variable-39ltvariablenamegt39-is-already-declared"></a>静态局部变量&#39; &lt;variablename&gt; &#39;已声明
已声明具有相同名称的另一个静态局部变量。  
  
 **错误 ID：** BC31401  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  删除冗余的静态局部声明。  
  
2.  给每个静态局部变量一个唯一的名称。
