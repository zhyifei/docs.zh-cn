# <a name="array-dimensions-cannot-have-a-negative-size"></a>数组维数的大小不能为负
一个或多个指定的数组边界是负数。 仅当使用 -1 作为上限声明空数组时，才能指定负下标。 这样的数组具有零个元素，但它不为 `Nothing`。  
  
 **错误 ID:** BC30611  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   使用负数替换负的数组界限。
