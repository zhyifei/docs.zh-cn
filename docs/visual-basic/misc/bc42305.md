# <a name="xml-comment-tag-39lttaggt39-appears-with-identical-attributes-more-than-once-in-the-same-xml-comment-block"></a>XML 注释标记&#39;&lt;标记&gt;&#39;具有相同特性不止一次出现在同一个 XML 注释块
XML 注释标记\<标记 > 具有相同特性不止一次出现在同一个 XML 注释块中。 将忽略 XML 注释。  
  
 XML 注释包含多个具有相同特性的标记。  
  
 **错误 ID：** BC42305  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   删除重复的标记。
