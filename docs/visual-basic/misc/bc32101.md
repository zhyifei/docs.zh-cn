# <a name="39class39-constraint-cannot-be-specified-multiple-times-for-the-same-type-parameter"></a>&#39;类&#39;约束不能为同一类型参数指定多次
约束列表包括[类 (Visual Basic)](https://msdn.microsoft.com/library/0777c6e6-46bc-451b-ad70-57b49d4ef4f7)约束不止一次。  
  
 类型参数上的约束列表可以指定传递到该类型形参的类型实参必须是值类型 (与[结构 (Visual Basic)](https://msdn.microsoft.com/library/263ce115-ac36-4c05-8cb7-0e0eead5c6d0)约束) 或者必须是引用类型 (与`Class`约束). 不能为同一类型形参同时指定两个约束，并且不能多次指定其中任意一个约束。  
  
 错误 ID：BC32101  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   删除任何多余的 `Class` 关键字。 约束列表中只能有一个关键字。
