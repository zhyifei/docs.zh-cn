# <a name="39before39-and-39after39-arguments-cannot-be-combined"></a>&#39;之前&#39;和&#39;后&#39;自变量不能组合
参数列表同时包含相互排斥的 `Before` 和 `After` 参数。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  删除 `Before` 或 `After` 参数。
