# <a name="namespace-or-type-ltnamegt-has-already-been-imported"></a>Namespace 或类型&lt;名称&gt;已导入
多次尝试导入命名空间。  
  
 **错误 ID:** BC31051  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   删除多余的 `Imports` 语句。
