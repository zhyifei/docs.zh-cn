# <a name="39resume39-or-39goto39-expected"></a>&#39;恢复 &#39;或 &#39;转到 &#39;预期
`On Error` 语句不包含 `Resume` 子句或 `GoTo` 子句。 它必须使用一个子句或另一个子句。  
  
 **错误 ID：** BC32019  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   将 `Resume` 子句或 `GoTo` 子句添加到 `On Error` 语句。
