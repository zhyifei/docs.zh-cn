# <a name="cannot-calculate-rate-using-the-arguments-provided"></a>无法使用提供的参数计算速率。
速率计算不包含所需的参数。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  检查表达式中参数的拼写是否正确。 拼写错误的变量名可能会隐式创建一个初始化为零的数值变量。  
  
2.  检查之前对表达式中的变量进行的操作，尤其是那些从其他过程作为参数传递给该过程的操作。
