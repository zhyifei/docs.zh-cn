# <a name="xml-comment-must-be-the-first-statement-on-a-line"></a>XML 注释必须是一行上的第一条语句
XML 注释必须是一行上的第一条语句。 将忽略 XML 注释。  
  
 不允许将 XML 注释置于另一个语言元素之后。  
  
 **错误 ID:** BC42302  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 将 XML 注释移到新行。
