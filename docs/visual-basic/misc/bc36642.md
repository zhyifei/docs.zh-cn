<span data-ttu-id="3f0c6-103">已经声明了 lambda 表达式中的参数，但未使用 `As` 子句，并且将 `Option Strict` 设置为了“on”。</span><span class="sxs-lookup"><span data-stu-id="3f0c6-103">You have declared a parameter in a lambda expression without using an `As` clause, with `Option Strict` on.</span></span>

```  
' Dim increment1 = Function (n) n + 1  
```  
  
 <span data-ttu-id="3f0c6-104">如果可以推断出 `n` 的类型，则上一个声明有效。</span><span class="sxs-lookup"><span data-stu-id="3f0c6-104">The previous declaration is valid if the type of `n` can be inferred.</span></span> <span data-ttu-id="3f0c6-105">例如，如果将上一个 lambda 表达式分配到函数委托 `Del`：</span><span class="sxs-lookup"><span data-stu-id="3f0c6-105">For example, if you are assigning the previous lambda expression to a function delegate, `Del`:</span></span>
