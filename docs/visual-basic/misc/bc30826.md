# <a name="39endif39-statements-are-no-longer-supported-use-39end-if39-instead"></a>&#39;EndIf &#39;不再支持语句;使用 &#39;如果端 &#39;改为
不再支持 `EndIf` 语句。  
  
 **错误 ID：** BC30826  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   使用 `End If` 终止 `If` 语句。
