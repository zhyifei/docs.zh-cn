# <a name="error-number-must-be-within-the-range-0-and-65535"></a>错误号必须在 0 到 65535 的范围内
错误号不在 0 到 65535 的允许范围内。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   确保错误号在有效的范围内。
