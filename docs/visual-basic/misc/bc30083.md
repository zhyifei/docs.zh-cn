# <a name="do-must-end-with-a-matching-loop"></a>“Do”必须以匹配的“Loop”结束
出现 `Do` 语句而没有相应的 `Loop` 语句。 必须使用 `Loop` 语句结束 `Do` 循环。  
  
 **错误 ID:** BC30083  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   如果此 `Do` 循环是一组嵌套循环的一部分，请确保每个循环正常终止。  
  
-   将 `Loop` 语句添加到 `Do` 循环末尾。
