# <a name="object-no-longer-exists-due-to-compile-error-or-deletion"></a>由于编译错误或删除操作，对象不再存在
Visual Basic 编译器内部发生代码元素访问错误。  
  
 **错误 ID：** BC32300  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  再次编译此程序以查看错误是否重复出现。  
  
2.  如果错误重复出现，请重新安装 Visual Basic 编译器。  
  
3.  如果重新安装后仍然出现错误，请收集有关该情况的信息并通知 Microsoft 产品支持服务。
