# <a name="too-many-type-arguments"></a>类型参数太多
已使用多于类型形参的类型实参调用了泛型方法。  
  
 在调用泛型方法时，必须为此方法定义的每一个类型形参提供一个类型实参。  
  
 **错误 ID：** BC36579  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   从类型实参列表中删除类型实参，以便正在调用的泛型方法的每一个类型形参都有一个类型实参。
