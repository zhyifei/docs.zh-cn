# <a name="type-of-variablename-cannot-be-inferred-from-an-expression-containing-variablename"></a><span data-ttu-id="5d1e8-102">无法从包含 "\<variablename >" 的表达式中推断 "\<variablename >" 的类型</span><span class="sxs-lookup"><span data-stu-id="5d1e8-102">Type of '\<variablename>' cannot be inferred from an expression containing '\<variablename>'</span></span>
<span data-ttu-id="5d1e8-103">如果在声明中使用变量建立其初始值，则编译器无法推断该变量的数据类型。</span><span class="sxs-lookup"><span data-stu-id="5d1e8-103">The compiler cannot infer the data type of a variable if the variable is used in establishing its initial value in the declaration.</span></span>  
  
 <span data-ttu-id="5d1e8-104">例如，在 `Option Infer` 设置为 `On`的情况下，下面的示例不编译：</span><span class="sxs-lookup"><span data-stu-id="5d1e8-104">For example, with `Option Infer` set to `On`, the following examples do not compile:</span></span>  
  
- <span data-ttu-id="5d1e8-105">声明</span><span class="sxs-lookup"><span data-stu-id="5d1e8-105">Declarations</span></span>  
  
    ```vb  
    ' Does not compile with Option Infer on.  
    Dim m = m  
    Dim d = someFunction(d)  
    ```  
  
- <span data-ttu-id="5d1e8-106">`For` 循环</span><span class="sxs-lookup"><span data-stu-id="5d1e8-106">`For` loop</span></span>  
  
    ```vb  
    ' Does not compile with Option Infer on.  
    For j = 1 To j  
    Next  
    ```  
  
- <span data-ttu-id="5d1e8-107">`For Each` 循环</span><span class="sxs-lookup"><span data-stu-id="5d1e8-107">`For Each` loop</span></span>  
  
    ```vb  
    ' Does not compile with Option Infer on.  
    For Each customer In customer.Orders  
    Next  
    ```  
  
 <span data-ttu-id="5d1e8-108">**错误 ID：** BC30980</span><span class="sxs-lookup"><span data-stu-id="5d1e8-108">**Error ID:** BC30980</span></span>
