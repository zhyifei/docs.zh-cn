# <a name="the-file-is-currently-open-for-writing"></a>文件当前处于打开状态，可以写入
已尝试访问当前处于打开状态并可以写入的文件。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 关闭文件，然后重试该操作。
