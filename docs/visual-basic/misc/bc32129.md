# <a name="enum-types-cannot-be-nullable"></a>枚举类型不可以为 null
用于声明枚举的基础类型不可以为 null。 例如，下列代码会导致此错误：  
  
```vb  
'' Not valid.  
' Enum exampleEnum As Integer?  
'     Member declarations.  
' End Enum  
```  
  
 **错误 ID：** BC32129  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   不在 `Enum` 声明中使用可以为 null 的基础类型。
