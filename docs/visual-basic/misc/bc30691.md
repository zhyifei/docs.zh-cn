# <a name="typename-is-a-type-in-containername-and-cannot-be-used-as-an-expression"></a>'\<类型名称 > 是中的类型\<容器名称 >，不能用作表达式
使用中的其中一个名称已声明为类型的名称，且不能在表达式中使用。  
  
 **错误 ID:** BC30691  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   对成员进行唯一命名。  
  
-   用方括号括起成员名称。
