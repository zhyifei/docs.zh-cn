## <a name="example"></a><span data-ttu-id="f4d99-108">示例</span><span class="sxs-lookup"><span data-stu-id="f4d99-108">Example</span></span>

```  
Namespace StringUtility  
    <Extension()> _  
    Module StringExtensions  
        <Extension()> _

        End Sub  
    End Module  
End Namespace  
```
