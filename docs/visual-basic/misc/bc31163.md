# <a name="expected-matching-closing-single-quote-for-xml-attribute-value"></a>XML 特性值应有匹配的右单引号
由单引号分隔的特性值缺少右引号字符。  
  
 **错误 ID：** BC31163  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   添加右单引号字符 (') 来分割特性值。
