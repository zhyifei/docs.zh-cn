# <a name="specifiers-and-attributes-are-not-valid-on-this-statement"></a>说明符和特性在此语句上无效
无法修改的声明包含一个或多个特性（例如 `Inherits`），或者一个或多个关键字（例如 `Public` 或 `ReadOnly`）。 仅可修改的声明可以包含这些关键字或特性。  
  
 **错误 ID:** BC30193  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 从声明中删除特性或关键字。
