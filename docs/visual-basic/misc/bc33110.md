# <a name="cannot-infer-a-common-type-for-the-first-and-second-operands-of-the-binary-39if39-operator"></a><span data-ttu-id="0e851-102">无法推断二进制文件的通用类型的第一个和第二个操作数&#39;如果&#39;运算符</span><span class="sxs-lookup"><span data-stu-id="0e851-102">Cannot infer a common type for the first and second operands of the binary &#39;If&#39; operator</span></span>
<span data-ttu-id="0e851-103">无法推断二进制“If”运算符的第一个和第二个操作数的通用类型。</span><span class="sxs-lookup"><span data-stu-id="0e851-103">Cannot infer a common type for the first and second operands of the binary 'If' operator.</span></span> <span data-ttu-id="0e851-104">其中一个必须有对另一个的扩大转换。</span><span class="sxs-lookup"><span data-stu-id="0e851-104">One must have a widening conversion to the other.</span></span>  
  
 <span data-ttu-id="0e851-105">该二进制 `If` 运算符要求其中一个参数和另一个参数之间必须有扩大转换。</span><span class="sxs-lookup"><span data-stu-id="0e851-105">The binary `If` operator requires that there be a widening conversion between one of the arguments and the other argument.</span></span> <span data-ttu-id="0e851-106">例如，由于 `Integer` 和 `String`之间的两个方向均没有扩大转换，因此以下代码导致此错误。</span><span class="sxs-lookup"><span data-stu-id="0e851-106">For example, because there is not a widening conversion in either direction between `Integer` and `String`, the following code causes this error.</span></span>  
  
```vb  
Dim first? As Integer  
Dim second As String = "First is Nothing"  
'' Not valid.  
' Console.WriteLine(If(first, second))  
```  
  
 <span data-ttu-id="0e851-107">**错误 ID：** BC33110</span><span class="sxs-lookup"><span data-stu-id="0e851-107">**Error ID:** BC33110</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="0e851-108">更正此错误</span><span class="sxs-lookup"><span data-stu-id="0e851-108">To correct this error</span></span>  
  
-   <span data-ttu-id="0e851-109">如果可能，请在你的代码中为其中一个操作数提供显式转换：</span><span class="sxs-lookup"><span data-stu-id="0e851-109">Provide an explicit conversion for one of the operands, if that is possible in your code:</span></span>  
  
    ```  
    Console.WriteLine(If(first, CInt(second)))   
    ```  
  
-   <span data-ttu-id="0e851-110">通过使用不同的条件构造重写代码。</span><span class="sxs-lookup"><span data-stu-id="0e851-110">Rewrite the code by using a different conditional construction.</span></span>  
  
    ```  
    If first IsNot Nothing Then  
        Console.WriteLine(first)  
    Else  
        Console.WriteLine(second)  
    End If  
    ```
