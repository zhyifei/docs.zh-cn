# <a name="structure-in-an-interface-cannot-be-declared-39ltspecifiergt39"></a>不能声明为接口中的结构&#39;&lt;说明符&gt;&#39;
为在接口中声明的结构指定了无效的修饰符。  
  
 **错误 ID：** BC31071  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   删除修饰符，如 `Public`、 `Private`、 `Shared`、 `Friend`、 `Protected`、 `Protected Friend`或 `Overridable`。
