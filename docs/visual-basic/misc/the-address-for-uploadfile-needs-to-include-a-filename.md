# <a name="the-address-for-uploadfile-needs-to-include-a-filename"></a>UploadFile 的地址中需要包括文件名
当用 `UploadFile` 方法上载文件时，必须提供目标路径中的文件名。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   提供文件名。
