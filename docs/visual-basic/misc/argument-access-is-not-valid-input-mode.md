# <a name="argument-39access39-is-not-valid-valid-values-for-input-mode-are-39openaccessread39-and-39openaccessdefault39"></a>自变量&#39;访问&#39;无效。 输入模式的有效值是&#39;OpenAccess.Read&#39;和&#39;OpenAccess.Default&#39;
访问值在 `Input` 模式中无效。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   将值更改为 `OpenAccess.Read` 或 `OpenAccess.Default`。
