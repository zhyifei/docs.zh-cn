# <a name="automation-object-does-not-have-a-default-value"></a>自动化对象没有默认值
Visual Basic 无法确定指定的对象的默认成员。  
  
### <a name="to-correct-the-error"></a>更正此错误  
  
- 检查对象的文档，并对属性或方法提供显式规范。
