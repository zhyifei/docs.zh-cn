# <a name="data-types-of-the-type-parameters-in-method-ltmethodnamegt-cannot-be-inferred-from-these-arguments-because-they-do-not-convert-to-the-same-type"></a><span data-ttu-id="b3308-102">在方法中类型形参的数据类型&lt;methodname&gt;因为它们没有转换为相同的类型不能从这些实参推断</span><span class="sxs-lookup"><span data-stu-id="b3308-102">Data type(s) of the type parameter(s) in method '&lt;methodname&gt;' cannot be inferred from these arguments because they do not convert to the same type</span></span>
<span data-ttu-id="b3308-103">在方法中类型形参的数据类型\<方法名称 > 因为它们没有转换为相同的类型不能从这些实参推断。</span><span class="sxs-lookup"><span data-stu-id="b3308-103">Data type(s) of the type parameter(s) in method '\<methodname>' cannot be inferred from these arguments because they do not convert to the same type.</span></span> <span data-ttu-id="b3308-104">显式指定数据类型可更正此错误。</span><span class="sxs-lookup"><span data-stu-id="b3308-104">Specifying the data type(s) explicitly might correct this error.</span></span>  
  
 <span data-ttu-id="b3308-105">在计算对泛型过程的调用时，试图使用类型推断功能来确定类型形参的数据类型。</span><span class="sxs-lookup"><span data-stu-id="b3308-105">An attempt has been made to use type inference to determine the data type or types of the type parameter or parameters when evaluating a call to a generic procedure.</span></span> <span data-ttu-id="b3308-106">编译器找不到符合所有实参约束的数据类型。</span><span class="sxs-lookup"><span data-stu-id="b3308-106">The compiler could not find a data type that meets the constraints of all the arguments.</span></span> <span data-ttu-id="b3308-107">因此，编译器报告此错误。</span><span class="sxs-lookup"><span data-stu-id="b3308-107">Therefore, it reported this error.</span></span>  
  
> [!NOTE]
>  <span data-ttu-id="b3308-108">当无法指定实参时（例如，对于查询表达式中的查询运算符），显示的错误消息不包括第二个句子。</span><span class="sxs-lookup"><span data-stu-id="b3308-108">When specifying arguments is not an option (for example, for query operators in query expressions), the error message appears without the second sentence.</span></span>  
  
 <span data-ttu-id="b3308-109">下面的代码演示了此错误。</span><span class="sxs-lookup"><span data-stu-id="b3308-109">The following code demonstrates the error.</span></span>  
  
```vb  
Option Strict Off  
Module Module1  
    Sub Main()  
  
        '' Not valid. Integer and Date do not convert to the same type.  
        'targetMethod(19, #3/4/2007#)  
  
    End Sub  
  
    Sub targetMethod(Of T)(ByVal p1 As T, ByVal p2 As T)  
    End Sub  
  
End Module  
```  
  
 <span data-ttu-id="b3308-110">**错误 ID:** BC36660 和 BC36657</span><span class="sxs-lookup"><span data-stu-id="b3308-110">**Error ID:** BC36660 and BC36657</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="b3308-111">更正此错误</span><span class="sxs-lookup"><span data-stu-id="b3308-111">To correct this error</span></span>  
  
-   <span data-ttu-id="b3308-112">你或许能够将一个或多个实参显式转换为某个兼容类型，如以下代码所示：</span><span class="sxs-lookup"><span data-stu-id="b3308-112">You may be able to convert one or more arguments explicitly to a compatible type, as shown in the following code:</span></span>  
  
    ```  
    targetMethod(19, #3/4/2007#.ToOADate)  
    ```  
  
-   <span data-ttu-id="b3308-113">你或许能够为实参转换到的类型形参指定数据类型，如以下代码所示：</span><span class="sxs-lookup"><span data-stu-id="b3308-113">You may be able to specify a data type for the type parameter or parameters to which the arguments convert, as shown in the following code:</span></span>  
  
    ```  
    targetMethod(Of String)(19, #3/4/2007#)  
    ```
