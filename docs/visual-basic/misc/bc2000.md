# <a name="initialization-error"></a>初始化错误
初始化程序时出错。 这可能由在安装正在进行时将其取消造成。  
  
 **错误 ID：** BC2000  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   重新尝试安装。
