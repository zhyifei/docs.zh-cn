# <a name="raiseevent-method-must-have-the-same-signature-as-the-containing-events-delegate-type-ltsignaturegt"></a>RaiseEvent 方法必须具有相同的签名包含事件的委托类型&lt;签名&gt;
`Custom Event` 声明必须具有 `RaiseEvent` 声明，此声明具有和自定义事件的 `As` 子句指定的委托类型相同的签名。  
  
 对于要匹配的签名， `RaiseEvent` 声明和委托必须具有参数数量，并且参数类型必须匹配。  
  
 **错误 ID:** BC31137  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   更改 `RaiseEvent` 声明的参数以匹配委托类型的参数。
