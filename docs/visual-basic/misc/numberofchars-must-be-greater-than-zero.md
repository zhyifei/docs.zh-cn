# <a name="numberofchars-must-be-greater-than-zero"></a>NumberOfChars 必须大于 0
使用 `PeekChars` 对象的 `TextFieldParser` 方法时，必须提供一个大于 `NumberOfChars` 的 `0`值。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   将 `NumberOfChars` 更改为大于 `0`的值。
