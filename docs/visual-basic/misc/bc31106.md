# <a name="property-accessors-cannot-be-declared-39ltaccessmodifiergt39-in-a-39notoverridable39-property"></a><span data-ttu-id="5313e-102">属性访问器不能声明为&#39;&lt;访问修饰符&gt;&#39;中&#39;NotOverridable&#39;属性</span><span class="sxs-lookup"><span data-stu-id="5313e-102">Property accessors cannot be declared &#39;&lt;accessmodifier&gt;&#39; in a &#39;NotOverridable&#39; property</span></span>
<span data-ttu-id="5313e-103">A [Get 语句](../../visual-basic/language-reference/statements/get-statement.md)或[Set 语句](../../visual-basic/language-reference/statements/set-statement.md)中`NotOverridable`属性包括`Private`关键字。</span><span class="sxs-lookup"><span data-stu-id="5313e-103">A [Get Statement](../../visual-basic/language-reference/statements/get-statement.md) or [Set Statement](../../visual-basic/language-reference/statements/set-statement.md) in a `NotOverridable` property includes the `Private` keyword.</span></span>  
  
 <span data-ttu-id="5313e-104">以下推理行解释了 `NotOverridable` 中不可同时包含 `Private` 和 [Private](../../visual-basic/language-reference/statements/property-statement.md)的原因：</span><span class="sxs-lookup"><span data-stu-id="5313e-104">The following line of reasoning explains why `NotOverridable` and `Private` cannot be combined in a [Property Statement](../../visual-basic/language-reference/statements/property-statement.md):</span></span>  
  
1.  <span data-ttu-id="5313e-105">不重写基类属性（或过程）的属性（或过程）具有 [NotOverridable](../../visual-basic/language-reference/modifiers/notoverridable.md)这一默认设置。</span><span class="sxs-lookup"><span data-stu-id="5313e-105">A property or procedure that does not override a base class property or procedure has a default setting of [NotOverridable](../../visual-basic/language-reference/modifiers/notoverridable.md).</span></span>  
  
2.  <span data-ttu-id="5313e-106">但是，对于派生类中重写基类属性（或过程）的属性（或过程），其默认设置为 [Overridable](../../visual-basic/language-reference/modifiers/overridable.md)。</span><span class="sxs-lookup"><span data-stu-id="5313e-106">However, a property or procedure in a derived class that overrides a base class property or procedure has a default setting of [Overridable](../../visual-basic/language-reference/modifiers/overridable.md).</span></span> <span data-ttu-id="5313e-107">若要终止重写层次结构，可将其声明为 `NotOverridable`。</span><span class="sxs-lookup"><span data-stu-id="5313e-107">To terminate the hierarchy of overriding, you can declare it `NotOverridable`.</span></span> <span data-ttu-id="5313e-108">这是唯一可使用 `NotOverridable`的上下文。</span><span class="sxs-lookup"><span data-stu-id="5313e-108">This is the only context in which you can use `NotOverridable`.</span></span> <span data-ttu-id="5313e-109">也就是说，你只能将 `NotOverridable` 和 [Overrides](../../visual-basic/language-reference/modifiers/overrides.md)这一默认设置。</span><span class="sxs-lookup"><span data-stu-id="5313e-109">That is, you can use `NotOverridable` only in combination with [Overrides](../../visual-basic/language-reference/modifiers/overrides.md).</span></span>  
  
3.  <span data-ttu-id="5313e-110">如果基类属性或过程声明为 [Private](../../visual-basic/language-reference/modifiers/private.md)，则派生类无法访问此属性或过程，因而不能重写它。</span><span class="sxs-lookup"><span data-stu-id="5313e-110">If a base class property or procedure is declared [Private](../../visual-basic/language-reference/modifiers/private.md), a derived class cannot override that property or procedure because it cannot access it.</span></span> <span data-ttu-id="5313e-111">因此，不能结合使用 `Private` 和 `Overridable`。</span><span class="sxs-lookup"><span data-stu-id="5313e-111">Because of this, you cannot use `Private` in combination with `Overridable`.</span></span>  
  
4.  <span data-ttu-id="5313e-112">若要重写属性或过程，则重写的属性或过程必须具有相同签名及相同的访问级别。</span><span class="sxs-lookup"><span data-stu-id="5313e-112">To override a property or procedure, the overriding property or procedure must have not only the identical signature but also the same access level.</span></span> <span data-ttu-id="5313e-113">这意味着，重写的属性或过程不能指定 `Private`，因为可重写的属性或过程不能指定 `Private`。</span><span class="sxs-lookup"><span data-stu-id="5313e-113">This means that an overriding property or procedure cannot specify `Private`, because an overridable property or procedure cannot specify `Private`.</span></span>  
  
5.  <span data-ttu-id="5313e-114">因为你只可在重写的属性或过程上指定 `NotOverridable` ，因此不能将其与 `Private`结合使用。</span><span class="sxs-lookup"><span data-stu-id="5313e-114">Because you can specify `NotOverridable` only on an overriding property or procedure, you cannot combine it with `Private`.</span></span>  
  
 <span data-ttu-id="5313e-115">同理，重写属性的个别属性过程（`Get` 和 `Set`）不能为 `Private`。</span><span class="sxs-lookup"><span data-stu-id="5313e-115">By the same reasoning, the individual property procedures (`Get` and `Set`) of an overriding property cannot be `Private`.</span></span>
