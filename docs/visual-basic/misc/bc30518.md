# <a name="overload-resolution-failed-because-no-accessible-method-can-be-called-with-these-argumentserror"></a>重载决策失败，因为没有可访问\<方法 > 可以用这些参数调用：\<错误 >
你已调用重载方法，但编译器没有找到具有匹配参数列表的方法。  
  
 **错误 ID:** BC30518  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   检查所提供的参数列表。
