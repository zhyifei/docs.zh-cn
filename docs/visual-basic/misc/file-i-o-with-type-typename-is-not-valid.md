# <a name="file-io-with-type-39lttypenamegt39-is-not-valid"></a>包含类型的文件 I/O &#39; &lt;typename&gt; &#39;无效
尝试将不受支持的类型写入文件。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   检查结构来确定不支持的类型出现的位置，然后将其更改为有效类型。
