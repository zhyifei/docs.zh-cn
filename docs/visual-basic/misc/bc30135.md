# <a name="39synclock39-statements-are-not-valid-in-the-immediate-window"></a>&#39;SyncLock &#39;语句中不能使用即时窗口
`SyncLock` 语句会同步线程，不允许用于调试上下文中。  
  
 **错误 ID：** BC30135  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   不要在“即时” `SyncLock`**窗口中发出** 语句。
