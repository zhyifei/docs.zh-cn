# <a name="39ltnamegt39-is-not-a-member-of-39ltclassnamegt39"></a>&#39;&lt;名称&gt;&#39;不是成员的&#39;&lt;类名&gt;&#39;
已提供的成员不是类的成员。  
  
 **错误 ID:** BC30456  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  检查成员名称以确保其准确无误。  
  
2.  使用类的实际成员。
