# <a name="on-goto-and-on-gosub-statements-are-no-longer-supported"></a>不再支持“On GoTo”和“On GoSub”语句
将变量或表达式的值用于 `GoTo` 和 `GoSub` 语句以控制程序流不再有效。  
  
 **错误 ID:** BC30817  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 重构应用程序以使用 `If...Then...Else` 或 `Case` 语句。
