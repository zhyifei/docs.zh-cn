# <a name="39is39-expected"></a>&#39; 是 &#39;预期
在没有 `TypeOf` 子句的情况下出现了 `Is` 运算符。  
  
 **错误 ID：** BC30224  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   在 `Is` 运算符后面添加 `TypeOf` 子句；例如 `TypeOf MyControl Is ComboBox`。
