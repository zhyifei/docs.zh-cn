# <a name="removehandler-is-already-declared"></a>已经声明“RemoveHandler”
自定义事件声明中出现多个 `RemoveHandler` 声明。 `RemoveHandler` 声明会声明用于删除事件处理程序的过程。  
  
 **错误 ID:** BC31128  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   删除多余的 `RemoveHandler` 语句。
