# <a name="xml-namespace-prefix-39prefix39-is-reserved-for-use-by-xml-and-the-namespace-uri-cannot-be-changed"></a>XML 命名空间前缀&#39;前缀&#39;已保留供 XML 和 URI 不能更改命名空间
一个保留的 XML 命名空间前缀（如 `xmlns`）已用于 XML 文本中。  
  
 **错误 ID：** BC31152  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   将 XML 命名空间前缀更改为非保留的值。
