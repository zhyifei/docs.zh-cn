# <a name="method-methodname1-must-be-declared-private-in-order-to-implement-partial-method-methodname2"></a><span data-ttu-id="237df-102">方法\<methodname1 > 必须声明为 Private 才能实现分部方法\<methodname2 ></span><span class="sxs-lookup"><span data-stu-id="237df-102">Method '\<methodname1>' must be declared 'Private' in order to implement partial method '\<methodname2>'</span></span>
<span data-ttu-id="237df-103">分部方法的实现必须声明为 `Private`。</span><span class="sxs-lookup"><span data-stu-id="237df-103">The implementation of a partial method must be declared `Private`.</span></span> <span data-ttu-id="237df-104">例如，下面的代码会导致此错误。</span><span class="sxs-lookup"><span data-stu-id="237df-104">For example, the following code causes this error.</span></span>  
  
```vb  
Partial Class Product  
  
    ' Declaration of the partial method.  
    Partial Private Sub valueChanged()  
    End Sub  
  
End Class  
```  
  
```vb  
Partial Class Product  
  
    ' Implementation of the partial method, with Private missing,   
    ' causes this error.   
    'Sub valueChanged()  
    '    MsgBox(Value was changed to " & Me.Quantity)  
    'End Sub  
  
End Class  
```  
  
 <span data-ttu-id="237df-105">**错误 ID:** BC31441</span><span class="sxs-lookup"><span data-stu-id="237df-105">**Error ID:** BC31441</span></span>

```vb  
End Sub  
```
