# <a name="argument-not-specified-for-parameter-39ltparameternamegt39"></a>没有为参数 &#39; 指定的自变量&lt;parametername&gt;&#39;
没有为所需形参提供实参。  
  
 **错误 ID：** BC36569  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   为指定形参提供实参。
