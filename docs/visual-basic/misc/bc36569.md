# <a name="argument-not-specified-for-parameter-ltparameternamegt"></a>没有为参数指定的参数&lt;parametername&gt;
没有为所需形参提供实参。  
  
 **错误 ID:** BC36569  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   为指定形参提供实参。
