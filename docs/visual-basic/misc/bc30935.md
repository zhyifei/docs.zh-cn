```

    Public Overridable Sub doSomething(ByVal inputValue As String)  
    End Sub  
    Public Overridable Sub doSomething(ByVal inputValue As t)  
    End Sub  
End Class

    Inherits baseClass(Of String)  
    Overrides Sub doSomething(ByVal inputValue As String)  
    End Sub  
End Class  
```  
  
 <span data-ttu-id="d3721-105">因为 `derivedClass` 继承为其类型参数 `baseClass` 提供 `String` 的 `t`， `doSomething` 中两个版本的 `baseClass` 使用就 `derivedClass` 而言完全相同的签名。</span><span class="sxs-lookup"><span data-stu-id="d3721-105">Because `derivedClass` inherits `baseClass` supplying `String` to its type parameter `t`, the two versions of `doSomething` in `baseClass` take on identical signatures as far as `derivedClass` is concerned.</span></span> <span data-ttu-id="d3721-106">因此，编译器无法确定要重写的版本。</span><span class="sxs-lookup"><span data-stu-id="d3721-106">As a result, the compiler cannot determine which version to override.</span></span>
