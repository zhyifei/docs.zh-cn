# <a name="at-least-one-parameter-type-of-this-binary-operator-must-be-the-containing-type-typename"></a>此二元运算符的至少一个参数类型必须是包含类型\<类型名称 >
二元运算符的定义使用除了在其中定义此运算符的类或结构之外的类型来指定两个参数。  
  
 当你在类或结构中定义运算符时，至少一个参数应为此类或结构的类型。  
  
 **错误 ID:** BC33021  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   将两个参数之一或两个参数更改为在其中定义了运算符的类或结构的类型。
