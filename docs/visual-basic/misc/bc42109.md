# <a name="variable-ltvariablenamegt-is-used-before-it-has-been-assigned-a-value-visual-basic-error"></a>变量&lt;variablename&gt;使用之前已分配的值 （Visual Basic 错误）
变量在赋值前已被使用，导致 <xref:System.NullReferenceException>。  
  
 **错误 ID:** BC42109  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   请确保已为变量赋值。
