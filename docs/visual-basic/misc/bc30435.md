# <a name="members-in-a-structure-cannot-be-declared-specifier"></a>不能声明为结构中的成员\<说明符 >
你使用了在 `Structure` 语句内的成员上无效的说明符。  
  
 **错误 ID:** BC30435  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 删除说明符。
