# <a name="referenced-object-objectname-has-a-value-of-nothing"></a>引用的对象\<对象名称 > 的值为 Nothing
尚未设置引用的对象，或已将其值设置为 `Nothing`。  
  
 **错误 ID:** BC30705  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 验证对象名称的拼写是否正确，并验证是否设置了该对象。
