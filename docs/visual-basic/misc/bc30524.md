# <a name="property-39ltpropertynamegt39-is-39writeonly39"></a>属性&#39; &lt;propertyname&gt; &#39;是&#39;WriteOnly&#39;
已尝试从声明 `WriteOnly`的属性中检索值。  
  
 **错误 ID：** BC30524  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   从属性声明中删除 `WriteOnly` 说明符。
