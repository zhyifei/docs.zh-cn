# <a name="property-propertyname-is-writeonly"></a>属性 '\<属性名称 > 为 WriteOnly
已尝试从声明 `WriteOnly`的属性中检索值。  
  
 **错误 ID:** BC30524  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   从属性声明中删除 `WriteOnly` 说明符。
