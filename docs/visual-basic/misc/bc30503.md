# <a name="properties-in-a-module-cannot-be-declared-39ltspecifiergt39"></a>模块中的属性不能声明&#39;&lt;说明符&gt;&#39;
你尝试对模块中的属性使用说明符，例如 `MustOverride`、 `NotOverridable`或 `Overridable`。  
  
 **错误 ID：** BC30503  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   删除说明符。
