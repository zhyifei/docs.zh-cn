# <a name="value-of-type-type1-cannot-be-converted-to-type2-because-type3-is-not-derived-from-type4"></a>类型的值\<type1 > 无法转换为\<y p e 2 > 由于 '\<type3 > 不派生自\<type4 >
当两个数组元素的数据类型并非同为引用类型时，或无法在两个数组的元素类型之间进行扩大转换或收缩转换时，语句试图将一个数组类型转换为另一个数组类型。  
  
 **错误 ID:** BC30332  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   检查两个数组的元素数据类型以确定错误源。
