# <a name="type-declaration-characters-are-not-valid-in-this-context"></a>类型声明字符在此上下文中无效。
你试图使用在此上下文中无效的字符，如 $。  
  
 **错误 ID:** BC30468  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  删除字符。
