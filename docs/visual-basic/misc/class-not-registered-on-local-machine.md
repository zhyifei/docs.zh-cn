# <a name="class-not-registered-on-local-machine"></a>未在本地计算机上注册的类
此错误通常是由于文件损坏而出现的。 你可能需要重新安装 Visual Studio。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   重新安装 Visual Studio。
