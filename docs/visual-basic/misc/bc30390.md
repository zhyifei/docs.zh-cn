# <a name="ltqualifiedmethodgt-is-not-accessible-in-this-context-because-it-is-ltmodifiergt"></a>'&lt;qualifiedmethod&gt;是不在此上下文中可访问，因此它是&lt;修饰符&gt;
尝试访问一个由于应用了修饰符而在此上下文中不可访问的方法。  
  
 **错误 ID:** BC30390  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   删除此修饰符。
