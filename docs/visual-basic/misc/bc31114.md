# <a name="39custom-event39-must-end-with-a-matching-39end-event39"></a>&#39;自定义事件 &#39;必须以匹配 &#39; 结尾结束事件 &#39;
`Custom Event` 声明必须以 `End Event` 语句结束。  
  
 **错误 ID：** BC31114  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   检查 `Custom Event` 声明是否以 `End Event` 语句结束。
