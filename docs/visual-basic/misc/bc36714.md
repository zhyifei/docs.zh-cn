# <a name="expanded-properties-cannot-be-initialized"></a><span data-ttu-id="73b5a-102">不能初始化扩展属性</span><span class="sxs-lookup"><span data-stu-id="73b5a-102">Expanded properties cannot be initialized</span></span>
<span data-ttu-id="73b5a-103">可以初始化自动实现的属性作为其声明的一部分，但不能初始化扩展的属性。</span><span class="sxs-lookup"><span data-stu-id="73b5a-103">An auto-implemented property can be initialized as part of its declaration, but an expanded property cannot be.</span></span>  
  
 <span data-ttu-id="73b5a-104">**错误 ID：** BC36714</span><span class="sxs-lookup"><span data-stu-id="73b5a-104">**Error ID:** BC36714</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="73b5a-105">更正此错误</span><span class="sxs-lookup"><span data-stu-id="73b5a-105">To correct this error</span></span>  
  
-   <span data-ttu-id="73b5a-106">使用自动实现的属性，或者从属性声明中删除该初始化。</span><span class="sxs-lookup"><span data-stu-id="73b5a-106">Either use an auto-implemented property or remove the initialization from the property declaration.</span></span>  
  
## <a name="example"></a><span data-ttu-id="73b5a-107">示例</span><span class="sxs-lookup"><span data-stu-id="73b5a-107">Example</span></span>  
 <span data-ttu-id="73b5a-108">下面的示例演示了自动实现的和扩展的属性。</span><span class="sxs-lookup"><span data-stu-id="73b5a-108">The following examples show both auto-implemented and expanded properties.</span></span> <span data-ttu-id="73b5a-109">自动实现的属性可通过使用分配或 `New` 子句进行初始化，但扩展的属性不能。</span><span class="sxs-lookup"><span data-stu-id="73b5a-109">Auto-implemented properties can be initialized by using assignment or a `New` clause, but expanded properties cannot be.</span></span>  
  
```vb  
Class AutoImplementedExample  
    ' An auto-implemented property can be assigned an initial value.  
    Property IDNum As Integer = 33542  
    ' An auto-implemented property can be initialized with New.  
    Property Name As New String("Don Hall")  
End Class  
  
Class ExpandedExample  
    ' Attempting to expand an initialized auto-implemented property  
    ' causes this error.  
    'Property IDNum As Integer = 33542  
    '    Get  
    '    End Get  
    '    Set(ByVal value As Integer)  
    '    End Set  
    'End Property  
  
    ' Instead, you can assign the initial value to the backing field.  
    Private _IDNum As Integer = 33542  
    Property IDNum As Integer  
        Get  
        End Get  
        Set(ByVal value As Integer)  
        End Set  
    End Property  
End Class  
```
