# <a name="argument-39access39-is-not-valid-valid-values-for-append-mode-are-39openaccesswrite39-and-39openaccessdefault39"></a>自变量 &#39;访问 &#39;不是有效的。 有效值为追加模式是 #39;OpenAccess.Write &#39;和 &#39;OpenAccess.Default &#39;
访问值在追加模式中无效。

1.  将值更改为 `OpenAccess.Write` 或 `OpenAccess.Default`。
