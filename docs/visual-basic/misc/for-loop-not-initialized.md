# <a name="for-loop-not-initialized-visual-basic"></a>未初始化 For 循环 (Visual Basic)
跳转到 `For...Next` 循环中间。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1. 删除向循环的跳转。
