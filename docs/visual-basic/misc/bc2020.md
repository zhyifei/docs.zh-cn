# <a name="the-maximum-number-of-errors-has-been-exceeded"></a>已超过错误的最大数量
已达到编译器错误数的最大值：100 个。  
  
 **错误 ID:** BC2020  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 通过逐个依次调试和解决，减少生成的错误数。
