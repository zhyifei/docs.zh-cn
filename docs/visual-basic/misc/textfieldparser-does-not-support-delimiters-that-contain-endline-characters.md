# <a name="textfieldparser-does-not-support-delimiters-that-contain-endline-characters"></a>TextFieldParser 不支持包含行尾字符的分隔符
提供包含行尾字符的分隔符，但`TextFieldParser`不支持包含行尾字符的分隔符。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 提供正确的分隔符。
