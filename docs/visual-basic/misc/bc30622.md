# <a name="39end-module39-must-be-preceded-by-a-matching-39module39"></a>&#39;End Module&#39;前面必须是匹配的&#39;模块&#39;
Module 块以 `Module` 语句开头并以 `End Module` 构造结尾。  
  
 **错误 ID：** BC30622  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   请确保模块以 `Module` 语句开头。
