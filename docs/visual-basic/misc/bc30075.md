# <a name="named-arguments-are-not-valid-as-array-subscripts"></a>命名参数作为数组下标无效
使用用于通过名称传递过程参数的语法指定数组索引，例如 `Array(Index := 10)`。 此语法对数组下标无效。  
  
 **错误 ID：** BC30075  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   使用普通变量或常量表达式作为数组索引，例如 `Array(10)`。
