# <a name="type-typename-has-no-type-parameters-and-so-cannot-have-type-arguments"></a><span data-ttu-id="b5e90-102">类型\<类型名称 > 没有类型参数，因此不能有类型实参</span><span class="sxs-lookup"><span data-stu-id="b5e90-102">Type '\<typename>' has no type parameters and so cannot have type arguments</span></span>
<span data-ttu-id="b5e90-103">在调用非泛型类型时，声明或赋值语句包括 [Of](../../visual-basic/language-reference/statements/of-clause.md) 子句。</span><span class="sxs-lookup"><span data-stu-id="b5e90-103">A declaration or assignment statement includes an [Of](../../visual-basic/language-reference/statements/of-clause.md) clause when invoking a non-generic type.</span></span>  
  
 <span data-ttu-id="b5e90-104">根据其定义， *泛型类型* 是作用于可通过一个或多个 *类型参数*来指定的数据类型的类、结构、接口、过程或委托。</span><span class="sxs-lookup"><span data-stu-id="b5e90-104">By its definition, a *generic type* is a class, structure, interface, procedure, or delegate that operates on data types you can specify through one or more *type parameters*.</span></span> <span data-ttu-id="b5e90-105">当使用的代码从该泛型类型创建类型时，它会为每个类型形参提供 *类型实参* 。</span><span class="sxs-lookup"><span data-stu-id="b5e90-105">When the using code creates a type from this generic type, it supplies a *type argument* to each type parameter.</span></span> <span data-ttu-id="b5e90-106">在创建该类型的过程中，每个类型实参在生成的代码中替换其对应的类型形参的每个匹配项。</span><span class="sxs-lookup"><span data-stu-id="b5e90-106">As part of the creation of the type, each type argument replaces every occurrence of its corresponding type parameter in the generated code.</span></span>  
  
 <span data-ttu-id="b5e90-107">括号内的 `Of` 子句用于定义类型形参，括号内的 `Of` 子句则用于提供类型实参。</span><span class="sxs-lookup"><span data-stu-id="b5e90-107">Type parameters are defined with an `Of` clause inside parentheses, and type arguments are supplied using an `Of` clause inside parentheses.</span></span> <span data-ttu-id="b5e90-108">仅在处理泛型类型时才使用 `Of` 子句。</span><span class="sxs-lookup"><span data-stu-id="b5e90-108">The `Of` clause is used only when dealing with generic types.</span></span>  
  
 <span data-ttu-id="b5e90-109">非泛型类型不接受类型形参，并且在调用此类类型时不能指定任何类型实参。</span><span class="sxs-lookup"><span data-stu-id="b5e90-109">Non-generic types do not accept type parameters, and you cannot specify any type arguments when you invoke such a type.</span></span>
