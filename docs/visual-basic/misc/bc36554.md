<span data-ttu-id="d67bf-103">“ParamArray”无法应用于扩展方法的第一个参数。</span><span class="sxs-lookup"><span data-stu-id="d67bf-103">'ParamArray' cannot be applied to the first parameter of an extension method.</span></span> <span data-ttu-id="d67bf-104">第一个参数指定要扩展哪个类型。</span><span class="sxs-lookup"><span data-stu-id="d67bf-104">The first parameter specifies which type to extend.</span></span>  
  
 <span data-ttu-id="d67bf-105">扩展方法的第一个参数指定该方法扩展的数据类型。</span><span class="sxs-lookup"><span data-stu-id="d67bf-105">The first parameter of an extension method specifies the data type that the method extends.</span></span> <span data-ttu-id="d67bf-106">因此，第一个参数是必需的，而不能是可选的。</span><span class="sxs-lookup"><span data-stu-id="d67bf-106">Therefore, the first parameter is required and cannot be optional.</span></span> <span data-ttu-id="d67bf-107">因为形参数组是自动可选的，所以不能作为扩展方法的第一个实参。</span><span class="sxs-lookup"><span data-stu-id="d67bf-107">Because a parameter array is automatically optional, it is not valid as the first argument of an extension method.</span></span>

 <span data-ttu-id="d67bf-110">**错误 ID：** BC36554</span><span class="sxs-lookup"><span data-stu-id="d67bf-110">**Error ID:** BC36554</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="d67bf-111">更正此错误</span><span class="sxs-lookup"><span data-stu-id="d67bf-111">To correct this error</span></span>  
  
-   <span data-ttu-id="d67bf-112">如果形参数组未指定要扩展的数据类型，请添加新的第一个形参以指定此类型。</span><span class="sxs-lookup"><span data-stu-id="d67bf-112">If the parameter array does not specify the data type you want to extend, add a new first parameter that specifies this type.</span></span>  
  
    ```  
    <Extension()>  
    Public Sub AddTo(ByRef str As String, ByVal ParamArray addOns() As String)  
    ' Concatenate the strings in addOns to str.  
    End Sub  
    ```  
  
-   <span data-ttu-id="d67bf-113">如果形参数组指定了要扩展的数据类型，请考虑将其更改为需要实参的常规数组，而不要使用形参数组。</span><span class="sxs-lookup"><span data-stu-id="d67bf-113">If the parameter array does specify the data type you want to extend, consider changing it to a regular array, requiring an argument, instead of a parameter array.</span></span> <span data-ttu-id="d67bf-114">常规数组可以进行扩展。</span><span class="sxs-lookup"><span data-stu-id="d67bf-114">Regular arrays can be extended.</span></span>  
  
    ```  
    <Extension()>  
    Public Function Sum(ByVal ints() As Integer) As Integer  
        Dim total As Integer = 0  
        For Each i As Integer In ints  
            total = total + i  
        Next i  
        Return total  
    End Function  
    ```
