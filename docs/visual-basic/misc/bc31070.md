# <a name="class-in-an-interface-cannot-be-declared-specifier"></a>不能声明为接口中的类\<说明符 >
使用在接口中声明的类指定了无效的修饰符。  
  
 **错误 ID:** BC31070  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 删除修饰符，如 `Public`、 `Private`、 `Shared`、 `Friend`、 `Protected`、 `Protected Friend`或 `Overridable`。
