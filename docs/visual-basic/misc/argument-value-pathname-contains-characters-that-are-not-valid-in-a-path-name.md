# <a name="argument-value-39ltpathnamegt39-contains-characters-that-are-not-valid-in-a-path-name"></a>自变量值 &#39;&lt;路径名&gt;&#39; 包含在路径名中无效的字符
路径名包含一个或多个无效字符。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   从路径名中删除无效字符。
