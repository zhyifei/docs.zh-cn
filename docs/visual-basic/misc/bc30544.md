# <a name="method-cannot-contain-both-a-39try39-statement-and-an-39on-error39-or-39resume39-statement"></a>方法不能同时包含 &#39;请尝试 &#39;语句和 &#39; On Error &#39;或 &#39;恢复 &#39;语句
你已将 `Try` 语句和 `On Error` 或 `Resume`合并。  
  
 **错误 ID：** BC30544  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  删除 `On Error` 或 `Resume`。
