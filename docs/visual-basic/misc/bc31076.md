# <a name="modulename-is-a-module-and-cannot-be-referenced-as-an-assembly"></a>\<模块名称 > 是一个模块，不能作为程序集引用
已尝试使用模块作为程序集。  
  
 **错误 ID:** BC31076  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 更改对程序集的引用。
