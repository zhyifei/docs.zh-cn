# <a name="withevents-variable-does-not-raise-any-instance-events-that-are-accessible-to-containername"></a>"WithEvents" 变量不会引发 "\<容器 >" 可访问的任何实例事件
正在引用的对象不会引发可访问的事件。  
  
 **错误 ID:** BC30600  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1. 修改事件的访问修饰符，或选择在当前上下文中可访问的事件。
