# <a name="end-get-must-be-the-first-statement-on-a-line"></a>“End Get”必须是一行中的第一条语句
`End Get` 语句跟在冒号 (:) 语句分隔符之后。 `End Get` 必须是其源行的唯一语句。  
  
 **错误 ID:** BC32033  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   将多个语句分隔到不同的行。
