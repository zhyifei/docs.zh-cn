# <a name="cannot-calculate-number-of-periods-using-the-arguments-provided"></a>无法计算使用提供的参数的周期数
调用`NPer`函数不包含所有必需自变量。  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   确保函数调用中包含 `Rate`、 `Prnt` 和 `PV` 值。
