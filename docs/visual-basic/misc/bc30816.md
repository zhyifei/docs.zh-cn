# <a name="membername-is-not-declared"></a>\<成员名称 > 未声明
未声明“`<membername>`”。 `Debug` 对象功能在 `System.Diagnostics.Debug` 程序集中的 `System` 中可用。  
  
 无法找到指定的成员。  
  
 **错误 ID:** BC30816  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  检查成员的拼写。  
  
2.  使用 `Imports` 语句或完全限定在其他命名空间中定义的成员。 例如，在 `WriteLine` 命名空间中声明 `System.Diagnostics.Debug` 函数。
