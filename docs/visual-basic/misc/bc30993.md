# <a name="property-39ltpropertynamegt39-cannot-be-initialized-in-an-object-initializer-expression-because-all-accessible-overloads-require-arguments"></a><span data-ttu-id="d7255-102">属性 &#39;&lt;propertyname&gt;&#39; 因为所有可访问的重载都需要参数不能在对象初始值设定项表达式中初始化</span><span class="sxs-lookup"><span data-stu-id="d7255-102">Property &#39;&lt;propertyname&gt;&#39; cannot be initialized in an object initializer expression because all accessible overloads require arguments</span></span>
<span data-ttu-id="d7255-103">对象初始值设定项列表中初始化的成员必须是字段或属性。</span><span class="sxs-lookup"><span data-stu-id="d7255-103">The members initialized in an object initializer list must be either fields or properties.</span></span> <span data-ttu-id="d7255-104">此外，初始值设定项列表中的属性不能有参数。</span><span class="sxs-lookup"><span data-stu-id="d7255-104">Additionally, properties in an initializer list cannot have parameters.</span></span> <span data-ttu-id="d7255-105">导致此错误的属性重载，其每个版本都需要参数。</span><span class="sxs-lookup"><span data-stu-id="d7255-105">The property causing this error is overloaded, and each of its versions requires arguments.</span></span> <span data-ttu-id="d7255-106">因此，无法在对象初始值设定项列表中初始化该属性。</span><span class="sxs-lookup"><span data-stu-id="d7255-106">Therefore, the property cannot be initialized in an object initializer list.</span></span>  
  
 <span data-ttu-id="d7255-107">**错误 ID：** BC30993</span><span class="sxs-lookup"><span data-stu-id="d7255-107">**Error ID:** BC30993</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="d7255-108">更正此错误</span><span class="sxs-lookup"><span data-stu-id="d7255-108">To correct this error</span></span>  
  
-   <span data-ttu-id="d7255-109">删除需要初始值设定项列表中的参数的属性。</span><span class="sxs-lookup"><span data-stu-id="d7255-109">Remove the property that requires arguments from the initializer list.</span></span>  
  
## <a name="example"></a><span data-ttu-id="d7255-110">示例</span><span class="sxs-lookup"><span data-stu-id="d7255-110">Example</span></span>  
 <span data-ttu-id="d7255-111">以下类包括三种属性定义：一种用于 `TotalItems` ，两种用于 `Item`，这种情况为重载。</span><span class="sxs-lookup"><span data-stu-id="d7255-111">The following class contains three property definitions: one for `TotalItems` and two for `Item`, which is overloaded.</span></span>  
  
```  
Class CollectionOfItems  
    Property TotalItems() As Integer  
        Get  
        End Get  
        Set(ByVal value As Integer)  
        End Set  
    End Property  
    Property Item(ByVal Key As String) As Object  
        Get  
        End Get  
        Set(ByVal value As Object)  
        End Set  
    End Property  
    Property Item(ByVal Index As Integer) As Object  
        Get  
        End Get  
        Set(ByVal value As Object)  
        End Set  
    End Property  
End Class  
```  
  
 <span data-ttu-id="d7255-112">`TotalItems` 属性不需要任何参数，并可在对象初始化列表中初始化，如以下声明中所示。</span><span class="sxs-lookup"><span data-stu-id="d7255-112">The `TotalItems` property requires no arguments and can be initialized in an object initialization list, as shown in the following declaration.</span></span>  
  
```  
Dim coinCollection As New CollectionOfItems With { .TotalItems = 0 }  
```  
  
 <span data-ttu-id="d7255-113">`Item` 属性为重载，并且每个重载都需要一个参数。</span><span class="sxs-lookup"><span data-stu-id="d7255-113">The `Item` property is overloaded, and each overload requires an argument.</span></span> <span data-ttu-id="d7255-114">因此， `Item` 不能出现在对象初始值设定项列表中。</span><span class="sxs-lookup"><span data-stu-id="d7255-114">Therefore, `Item` cannot appear in an object initializer list.</span></span>  
  
```  
' The following declaration is not valid.  
' Dim coinCollection As New CollectionOfItems With { .TotalItems = 0, _  
'    .Item = aCoinObject }  
```  
  
 <span data-ttu-id="d7255-115">若要避免此错误，在对象初始值设定项之外初始化 `Item` 属性。</span><span class="sxs-lookup"><span data-stu-id="d7255-115">To avoid this error, initialize the `Item` property outside the object initializer.</span></span>  
  
```  
Dim coinCollection As New CollectionOfItems With { .TotalItems = 0 }  
coinCollection.Item(1) = aCoinObject  
```
