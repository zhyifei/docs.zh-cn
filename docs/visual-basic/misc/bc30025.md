# <a name="property-missing-end-property"></a>属性缺少“End Property”。
`End` 语句必须与它终止的块匹配。 未正确终止 `Property` 定义。  
  
 **错误 ID:** BC30025  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 将 `End Property` 语句添加到属性定义的末尾。
