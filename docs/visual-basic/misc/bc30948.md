# <a name="unscheduled-fiber"></a>未计划纤程
由于表达式处于物理线程上未计划的逻辑纤程中，因此调试器无法计算其值。 如果进程在使用纤程的 SQL 服务器上运行，则可能会发生这种情况。  
  
 纤程由堆栈和寄存器上下文组成，它可以在任何线程上运行。 纤程可以交换出某个线程，稍后在另一个线程上重新启动。  
  
 **错误 ID：** BC30948  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   确保在物理线程上计划了纤程。
