# <a name="anonymous-type-member-or-property-propertyname-is-already-declared"></a><span data-ttu-id="e5dfa-102">匿名类型成员或属性 '\<属性名称 > 已声明</span><span class="sxs-lookup"><span data-stu-id="e5dfa-102">Anonymous type member or property '\<propertyname>' is already declared</span></span>
<span data-ttu-id="e5dfa-103">属性名在匿名类型的声明中只能建立一次。</span><span class="sxs-lookup"><span data-stu-id="e5dfa-103">A property name can be established only once in the declaration of an anonymous type.</span></span> <span data-ttu-id="e5dfa-104">例如，下面的声明无效：</span><span class="sxs-lookup"><span data-stu-id="e5dfa-104">For example, the following declarations are not valid:</span></span>  
  
```vb  
'' Not valid, because the Label property is assigned to twice.  
' Dim anonType1 = New With {.Label = "Debra Garcia", .Label = .Label & ", President"}  
'' Not valid, because the property name inferred for both properties is  
'' Name.  
' Dim anonType2 = New With {Key product.Name, Key car1.Name}  
```  
  
 <span data-ttu-id="e5dfa-105">**错误 ID:** BC36547</span><span class="sxs-lookup"><span data-stu-id="e5dfa-105">**Error ID:** BC36547</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="e5dfa-106">更正此错误</span><span class="sxs-lookup"><span data-stu-id="e5dfa-106">To correct this error</span></span>

```vb  
' Valid.

```

```vb  
' Valid.

```
