# <a name="end-statement-not-valid"></a>"End" 语句无效
`End` 关键字使用不当或不适用于当前上下文。  
  
 **错误 ID:** BC30678  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1.  单独使用 `End` ，使过程失去范围。  
  
2.  将 `End` 与其他关键字一起使用，以终止代码块（如结构或属性）。
