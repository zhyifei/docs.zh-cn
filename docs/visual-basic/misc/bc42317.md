# <a name="xml-comment-type-parameter-ltparametergt-does-not-match-a-type-parameter-on-the-corresponding-ltkeywordgt-statement"></a>XML 注释类型参数&lt;参数&gt;与上相应的类型参数不匹配&lt;关键字&gt;语句
XML 注释参数\<参数 > 与相应的类型形参不匹配\<关键字 > 语句。 将忽略 XML 注释。  
  
 为方法中的某些（而非所有）类型形参指定了 XML 注释标记。  
  
 **错误 ID：** BC42317  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   为方法中的所有类型形参提供 XML 注释标记。
