 <span data-ttu-id="08fe2-110">默认情况下，此消息是一个警告。</span><span class="sxs-lookup"><span data-stu-id="08fe2-110">By default, this message is a warning.</span></span> <span data-ttu-id="08fe2-111">有关隐藏警告或将警告视为错误的信息，请参见 [Configuring Warnings in Visual Basic](/visualstudio/ide/configuring-warnings-in-visual-basic)。</span><span class="sxs-lookup"><span data-stu-id="08fe2-111">For information on hiding warnings or treating warnings as errors, see [Configuring Warnings in Visual Basic](/visualstudio/ide/configuring-warnings-in-visual-basic).</span></span>
