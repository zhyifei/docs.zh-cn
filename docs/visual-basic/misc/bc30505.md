# <a name="methods-or-events-that-implement-interface-members-cannot-be-declared-39shared39"></a>方法或实现的接口成员不能声明的事件 &#39;共享 &#39;
已尝试声明为实现接口成员的 `Shared` 方法或事件。 不能将类中实现的方法和事件指定为 `Shared` 或 `Private`，非可继承的类除外。  
  
 **错误 ID：** BC30505  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   删除 `Shared` 关键字。
