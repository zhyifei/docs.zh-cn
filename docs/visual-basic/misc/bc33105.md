# <a name="if-operands-cannot-be-named-arguments"></a><span data-ttu-id="38b07-102">“If”操作数不能为命名的参数</span><span class="sxs-lookup"><span data-stu-id="38b07-102">'If' operands cannot be named arguments</span></span>
<span data-ttu-id="38b07-103">在 `If` 运算符的操作数中使用命名的参数无效。</span><span class="sxs-lookup"><span data-stu-id="38b07-103">Using named arguments in the operands of the `If` operator is not valid.</span></span> <span data-ttu-id="38b07-104">以下示例会导致此错误：</span><span class="sxs-lookup"><span data-stu-id="38b07-104">The following example causes this error:</span></span>  
  
```  
Dim i As Integer  
Dim result As String  
' Not valid.  
' result = (If(i > 0, TruePart:="positive", FalsePart:="not positive")  
```  
  
 <span data-ttu-id="38b07-105">这不同于 `IIf` 函数，该函数不允许使用命名参数，如以下代码中所示：</span><span class="sxs-lookup"><span data-stu-id="38b07-105">This differs from the `IIf` function, which does allow named arguments, as shown in the following code:</span></span>  
  
```  
' Valid.  
IIf(i > 0, TruePart:="positive", FalsePart:="not positive")  
```  
  
 <span data-ttu-id="38b07-106">**错误 ID:** BC33105</span><span class="sxs-lookup"><span data-stu-id="38b07-106">**Error ID:** BC33105</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="38b07-107">更正此错误</span><span class="sxs-lookup"><span data-stu-id="38b07-107">To correct this error</span></span>  
  
- <span data-ttu-id="38b07-108">从操作数删除名称分配，如下面的代码中所示。</span><span class="sxs-lookup"><span data-stu-id="38b07-108">Remove the name assignments from the operands, as shown in the following code.</span></span>  
  
    ```  
    result = If(i > 0, "positive", "not positive")  
    ```
