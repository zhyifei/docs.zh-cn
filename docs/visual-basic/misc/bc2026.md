# <a name="warning-number-39ltnumbergt39-for-the-option-39ltoptionnamegt39-is-either-not-configurable-or-not-valid"></a>警告编号 &#39;&lt;数&gt;&#39; 对于选项 &#39;&lt;选项名称&gt;&#39; 是无法配置要么无效
警告 ID 无效。 ID 必须是大于 0 的整数值。  
  
 **错误 ID：** BC2026  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   检查所提供的警告 ID，确保它是有效的。
