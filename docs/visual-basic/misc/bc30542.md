# <a name="division-by-zero-occurred-while-evaluating-this-expression"></a>计算此表达式时出现被零除的情况。
在计算此表达式时，你尝试用零除以一个数字。  
  
 **错误 ID:** BC30542  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
1. 检查常数和变量值，确定为何在表达式分母中出现了零
