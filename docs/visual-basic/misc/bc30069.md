# <a name="for-loop-control-variable-variablename-already-in-use-by-an-enclosing-for-loop"></a>For 循环控制变量\<变量名 >' 中已被封闭 For 循环使用
所有嵌套 `For` 循环必须使用唯一的控制变量。  
  
 **错误 ID:** BC30069  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
- 更改其中一个控制变量名。
