# <a name="data-types-of-the-type-parameters-in-extension-method-methodname-defined-in-typename-cannot-be-inferred-from-these-arguments-because-they-do-not-convert-to-the-same-type"></a><span data-ttu-id="693de-102">类型参数的扩展方法中的数据类型\<方法名称 > 定义不能从这些实参推断 typename 中，因为它们没有转换为同一类型</span><span class="sxs-lookup"><span data-stu-id="693de-102">Data type(s) of the type parameter(s) in extension method '\<methodname>' defined in 'typename' cannot be inferred from these arguments because they do not convert to the same type</span></span>
<span data-ttu-id="693de-103">类型参数的扩展方法中的数据类型\<方法名称 > 定义不能从这些实参推断 typename 中，因为它们没有转换为同一类型。</span><span class="sxs-lookup"><span data-stu-id="693de-103">Data type(s) of the type parameter(s) in extension method '\<methodname>' defined in 'typename' cannot be inferred from these arguments because they do not convert to the same type.</span></span> <span data-ttu-id="693de-104">显式指定数据类型可更正此错误。</span><span class="sxs-lookup"><span data-stu-id="693de-104">Specifying the data type(s) explicitly might correct this error.</span></span>  
  
 <span data-ttu-id="693de-105">在计算对泛型扩展方法的调用时，试图使用类型推断功能来确定类型形参的数据类型。</span><span class="sxs-lookup"><span data-stu-id="693de-105">An attempt has been made to use type inference to determine the data type (or types) of the type parameter (or parameters) when evaluating a call to a generic extension method.</span></span> <span data-ttu-id="693de-106">编译器找不到符合所有实参约束的数据类型。</span><span class="sxs-lookup"><span data-stu-id="693de-106">The compiler could not find a data type that meets the constraints of all the arguments.</span></span> <span data-ttu-id="693de-107">因此，编译器报告此错误。</span><span class="sxs-lookup"><span data-stu-id="693de-107">Therefore, it reported this error.</span></span>  
  
> [!NOTE]
>  <span data-ttu-id="693de-108">当无法指定实参时（例如，对于查询表达式中的查询运算符），显示的错误消息不包括第二个句子。</span><span class="sxs-lookup"><span data-stu-id="693de-108">When specifying arguments is not an option (for example, for query operators in query expressions), the error message appears without the second sentence.</span></span>  
  
 <span data-ttu-id="693de-109">下面的代码演示了此错误。</span><span class="sxs-lookup"><span data-stu-id="693de-109">The following code demonstrates the error.</span></span>  
  
```vb

Module Module3  
    Sub Main()  
  
        Dim c1 As New Class1  
  
        '' Not valid. Integer and Date do not convert to the same type.  
        'c1.targetMethod(19, #3/4/2007#)  
  
    End Sub  
  
    <System.Runtime.CompilerServices.Extension()> _  
    Sub targetMethod(Of T)(ByVal p0 As Class1, ByVal p1 As T, ByVal p2 As T)  
    End Sub  
  
    Class Class1  
    End Class  
  
End Module  
```

## <a name="to-correct-this-error"></a><span data-ttu-id="693de-111">更正此错误</span><span class="sxs-lookup"><span data-stu-id="693de-111">To correct this error</span></span>

