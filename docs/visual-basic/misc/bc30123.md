# <a name="end-statements-are-not-valid-in-the-immediate-window"></a>“End”语句在“即时”窗口中无效
`Stop` 和 `End` 语句暂停执行，且不允许在调试上下文中使用它们。  
  
 **错误 ID:** BC30123  
  
## <a name="to-correct-this-error"></a>更正此错误  
  
-   不要在“即时” `End``Stop` 窗口中发出 **或** 语句。
